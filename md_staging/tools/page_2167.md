## Content

### 3.21.3.1.3 Checkbox and Option buttons

#### Adding Checkbox to the Nodes

The `ShowCheckBoxes` property when set will display a checkbox for all the nodes. To apply checkboxes to individual nodes, set the `TreeNodeAdv.ShowCheckBox` property, only for the required nodes in the `NodeCollection Editor`.

The `InteractiveCheckBoxes` indicates whether the state of the parent node's checkbox is based on the checkstate of its child nodes' checkboxes. To set this for individual nodes set the `InteractiveCheckBox` property of the `TreeNodeAdv`.

| TreeViewAdv Property | Description                          |
|----------------------|---------------------------------------|
| ShowCheckBoxes      | Indicates if the checkbox will be shown for the nodes. |

## API Reference

This section focuses on:

- `ShowCheckBoxes`: A property indicating whether checkboxes should be displayed for all nodes.
- `InteractiveCheckBoxes`: A feature indicating whether the parent node's checkbox state depends on its child nodes' checkbox states.

## Visual Styles

### Overview
- Visual Styles determines the painting scheme to be used for drawing the control.

### Visual Styles
The Visual Styles supported are given below.

#### Office XP Style
- It specifies the standard colors and it can be enabled by setting the Visual Style to OfficeXP and ThemesEnabled property to `True`.

#### Office 2003 Style
- It specifies the Office 2003 look and it can be enabled by setting the Visual Style to Office2003 and ThemesEnabled property to `False`.

#### Office 2007 Style
- It specifies the new Office 2007 look and it can be enabled by setting the Visual Style to Office2007 and ThemesEnabled property to `False`.

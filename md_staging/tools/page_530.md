# Essential Tools for Windows Forms

## Overview
- Learn how to customize color groups using the Custom Groups property in the ColorPickerUIAdv control.
- Explore the use of the ColorUIAdvGroup Collection Editor to add custom user groups.

## Content

### 3.5.4.5.3.1.1 Custom Color Groups

Custom Color Groups can be added to ColorPickerUIAdv control using the **CustomGroups** property. This property invokes the **ColorUIAdvGroup Collection Editor** and lets you add custom user groups.

#### Figure: Custom Groups Overview
![Custom Groups Overview](https://example.com/image.png)

#### Figure: ColorUIAdvGroup Collection Editor
![ColorUIAdvGroup Collection Editor](https://example.com/editor.png)

#### Properties of the ColorUIAdvGroup Collection Editor
- **HeaderHeight**: 25
- **IsSubItemsVisible**: True
- **Items**: (Collection)
- **Name**: Custom User Colors
- **SubItemsDepth**: 1
- **Visible**: True

#### Custom User Colors Group
- **Custom User Colors** is added as a new group in the ColorUIAdvGroup Collection Editor.

#### Group Properties
- **CustomGroups**: (Collection)
- **RecentGroup**: Syncfusion.Windows.Forms.
- **StandardGroup**: Syncfusion.Windows.Forms.
- **ThemeGroup**: Syncfusion.Windows.Forms.

# Essential Tools for Windows Forms

The ToolStripTextBox item can be added to a ToolStripEx directly or through a panel.

## Overview
-ToolStripTextBox can be added directly to a ToolStripEx or via a panel.
-ToolStripTextBox supports AutoComplete functionality.
-Various properties control its appearance and behavior.

#### Properties Controlling Appearance and Behavior

The following properties control the appearance and behavior of the ToolStripTextBox item:

### Foreground Settings

| Property         | Description                                                                 |
|------------------|-----------------------------------------------------------------------------|
| BackColor        | Sets the back color for the textbox.                                      |
| BorderStyle      | Specifies the border style for the textbox. The options are:            |
|                  |   - FixedSingle,                                                          |
|                  |   - Fixed3D,                                                              |
|                  |   - None.                                                                 |
| Text             | Sets the Text for the ToolStripTextBox.                                   |
| Lines            | Lets you open a String Collection Editor, using which multiline text can be entered. |
| Font             | Sets the font style for the display text.                                |
| ForeColor        | Sets the fore color for the display text.                                |
| TextBox TextAlign | Specifies the alignment of the text in the item. The options are:      |
|                  |   - Left,                                                                 |
|                  |   - Right,                                                                |
|                  |   - Center.                                                               |

## Code Examples

### Example: Adding ToolStripTextBox to a ToolStripEx

```csharp
ToolStripEx toolStripEx = new ToolStripEx();
ToolStripTextBox toolStripTextBox = new ToolStripTextBox();
toolStripTextBox.Text = "Example Text";
toolStripTextBox.ForeColor = Color.Blue;
toolStripEx.Items.Add(toolStripTextBox);
```

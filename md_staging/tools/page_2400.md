## API Reference
- **Namespace**: Syncfusion.Windows.Forms.Localization
- **Class**: `LocalizationManager`
- **Methods**:
  - `SetCulture(String culture)`: Sets the culture for localization.
  - `ReloadResources()`: Reloads the localization resources for the current culture.
- **Properties**:
  - `CurrentCulture`: Gets or sets the current culture being used.

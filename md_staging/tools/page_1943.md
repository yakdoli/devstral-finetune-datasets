## Essential Tools for Windows Forms

### Overview
- This section describes how to set the orientation of a slider control to vertical.
- It explains the `Orientation` property and its impact on the layout of the slider.
- Demonstrates setting `SliderMin` and `SliderMax` properties to define the range of the slider.

### Content

#### Setting Orientation to Vertical

**Figure 1324: Horizontal**

b. When Orientation is set to vertical.

```csharp
rangeSlider.Orientation = Orientation.Vertical;
```

**Figure 1325: Vertical**

#### 3.14.2.10 Setting SliderMin and SliderMax

This position of the right and left thumb can be set using the `SliderMin` and `SliderMax` properties.

In the following example, the position of the right and left thumbs is set to `7, 2`.

```csharp
rangeSlider.SliderMax = 7;
rangeSlider.SliderMin = 2;
```

---

### API Reference

- **Properties**
  - `Orientation`: Sets the orientation of the slider to vertical or horizontal.
  - `SliderMax`: Specifies the maximum value of the slider.
  - `SliderMin`: Specifies the minimum value of the slider.

### Code Examples

The following code snippet demonstrates setting the orientation and range of a slider:

```csharp
rangeSlider.Orientation = Orientation.Vertical;
rangeSlider.SliderMax = 7;
rangeSlider.SliderMin = 2;
```

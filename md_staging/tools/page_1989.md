### FixedSingleBorderColor

The `FixedSingleBorderColor` property is used to set a color for the border of the TabPage in the TabControlAdv when the `BorderStyle` is set to `FixedSingle`.

#### TabControlAdv Property Details

| TabControlAdv Property        | Description                                                                 |
|-------------------------------|-----------------------------------------------------------------------------|
| `FixedSingleBorderColor`      | Gets / sets a color for the border of the TabPage in the TabControlAdv when the BorderStyle is set to `FixedSingle`. |

### Summary

The `FixedSingleBorderColor` property allows developers to customize the appearance of the TabPage by setting a specific border color when the `BorderStyle` is configured to `FixedSingle`. This property is particularly useful for enhancing the visual design of the TabControlAdv control.

## API Reference

### Namespace: Syncfusion.Windows.Forms.Tools

#### Properties
- **FixedSingleBorderColor**  
  Type: System.Drawing.Color  
  Description:  
  Gets or sets the color for the border of the TabPage when the BorderStyle is set to `FixedSingle`.  
  Default: Unspecified  
  Required: No

### Code Example (C#)
```csharp
using Syncfusion.Windows.Forms.Tools;

// Set the border color for TabPage when BorderStyle is FixedSingle
TabControlAdv tabControl = new TabControlAdv();
tabControl.FixedSingleBorderColor = Color.Blue;
```

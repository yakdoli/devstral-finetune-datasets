# Essential Tools for Windows Forms

## Overview
- Assigns the SuperAccelerator value to a RibbonControlAdv child element.
- Demonstrates how to set Office 2007 Style Key Tips using the form designer.

## Content

### Adding SuperAccelerator to the RibbonControlAdv

The SuperAccelerator key value for a RibbonControlAdv child element can be assigned from the form designer as shown in the following image:

#### Figure 1135: Adding SuperAccelerator to the RibbonControlAdv

![Properties window showing SuperAccelerator assignment](image)

The image shows the properties window for `RibbonControlAdv1`, indicating the following:
- The `SuperAccelerator` property is set to `superAccelerator1`.
- This assignment is made through the form designer.

## API Reference

### Properties
#### SuperAccelerator
```csharp
public string SuperAccelerator { get; set; }
```
- **Purpose**: Provides options to set Office 2007 Style Key Tips.
- **Default Value**: `(none)`

## Code Examples

### Assigning SuperAccelerator in Code
```csharp
// Assigning the SuperAccelerator programmatically
RibbonControlAdv ribbonControl = new RibbonControlAdv();
ribbonControl.SuperAccelerator = "superAccelerator1";
```

# Essential Tools for Windows Forms

## Overview
- Explains how to manage Stacked GroupBar Items and their Chevron property.
- Details properties related to the Navigation Pane in Stacked GroupBar mode.
- Includes ejemplo code in C# for setting Navigation Pane dimensions.

## Content

### Stacked GroupBar Items and Chevron

Stacked GroupBar Item automatically shows the **Chevron**, which can be made invisible by setting the **ShowChevron** property to 'False'.

![Figure 898: Chevron shown when GroupBar is in Stacked Mode](684844e6dabb45c995a8dd5f52471fa8.png)

*Figure 898: Chevron shown when GroupBar is in Stacked Mode*

**Note:** You should set **LargeImageMode** of **GroupBarItem** to 'True' to display the item images in the GroupBar's navigation pane.

### Navigation Pane

The following table lists the properties related to the Navigation Pane.

**GroupBar Property** | **Description**
--- | ---
NavigationPaneButtonWidth | Specifies the width of the GroupBar Items displayed in the navigation pane. This property will be available only when StackedMode property is set to 'True'.
NavigationPaneHeight | Specifies the height of the GroupBar Navigation pane. This property will be available only when StackedMode property is set to 'True'.

#### C# Code Example

```csharp
// StackedMode set to true.
this.groupBar1.NavigationPaneButtonWidth = 25;
this.groupBar1.NavigationPaneHeight = 35;
```

## API Reference

### Properties
- **NavigationPaneButtonWidth**: Sets the width of the GroupBar Items in the navigation pane.
- **NavigationPaneHeight**: Sets the height of the GroupBar Navigation pane.

## Code Examples

### C#

```csharp
// Example: Setting Navigation Pane properties.
this.groupBar1.StackedMode = true;
this.groupBar1.NavigationPaneButtonWidth = 25;
this.groupBar1.NavigationPaneHeight = 35;
```

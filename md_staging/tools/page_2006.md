## TabStyles

**TabControl** provides options to customize the TabStyle settings. Styles can be set through the **TabStyle** property.

### TabControlAdv Property - TabStyle

| Property        | Description                                                                                     |
|-----------------|-------------------------------------------------------------------------------------------------|
| **TabStyle**    | Specifies the look and feel of the TabControl. The options include:<br><ul><li>2D</li><li>3D</li><li>Workbook</li><li>InternetExplorer7Style</li><li>OneNoteStyle</li><li>VS2005DockingStyle</li><li>Office2007Style</li><li>VS2005Style</li><li>VS2005DockingStyleBeta</li><li>Office2003Style</li><li>VS2008Style</li></ul> |

### Code Examples

#### [C#]

```csharp
this.tabControlAdv1.TabStyle =
    typeof(Syncfusion.Windows.Forms.Tools.TabRendererWorkbookMode);
```

#### [VB.NET]

```vb
Me.tabControlAdv1.TabStyle =
    GetType(Syncfusion.Windows.Forms.Tools.TabRendererWorkbookMode)
```

## Overview

This page describes various properties used to customize the appearance of a Windows Forms control, including background colors, gradient styles, and background images.

## Content

### Control Appearance Properties

| Property       | Description                                                                                                                                                                                                                       |
|----------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|                | - Solid<br>- Pattern and<br>- Gradient.                                                                                                                          |
| BackColor      | Specifies the backcolor of the control.                                                                                                                          |
| ForeColor      | Specifies the forecolor for any text or graphics in the control.                                                                                               |
| GradientStyle  | Specifies the gradient style of the background.<br>- ForwardDiagonal,<br>- BackwardDiagonal,<br>- Horizontal,<br>- Vertical,<br>- PathRectangle and<br>- PathEllipse. |
| GradientColors | Specifies the gradient colors. The first entry in this list will be the same as the BackColor property, the last entry will be the same as the ForeColor property. |
| BackgroundImage| Specifies the background image used for the control.                                                                                                              |
| TransparentColor| Specifies the transparent color for the background.                                                                                                             |

### Code Examples

#### C#

```csharp
this.splashPanel1.BackgroundColor = new Syncfusion.Drawing.BrushInfo(Syncfusion.Drawing.GradientStyle.PathRectangle, System.Drawing.Color.AliceBlue, System.Drawing.Color.SteelBlue);
this.splashPanel1.BackgroundImage = ((System.Drawing.Image)(resources.GetObject("blue_hills")));
this.splashPanel1.TransparencyColor = System.Drawing.Color.White;
```

#### VB.NET

```vb
Me.splashPanel1.BackgroundColor = New
```

## API Reference

- **Namespace**: 
  - `Syncfusion.Drawing`: Contains classes for handling colors, brushes, and related drawing elements.
- **Class**: 
  - `BrushInfo`: Represents a gradient brush with specified styles and colors.
- **Properties**:
  - **GradientStyle**: Enumerates the available gradient styles such as `ForwardDiagonal`, `BackwardDiagonal`, `Horizontal`, `Vertical`, `PathRectangle`, and `PathEllipse`.
  - **GradientColors**: A list of colors used for the gradient effect.

# Essential Tools for Windows Forms

## Overview
- Describes the autocomplete features in Windows Forms.
- Explains the use of AutoCompleteCustomSource, AutoCompleteSource, and AutoCompleteMode properties.
- Details the text completion behavior and keyboard shortcuts for combo boxes.

## Content

### AutoComplete Customization

| Property            | Description                                                                 |
|---------------------|------------------------------------------------------------------------------|
| `AutoCompleteCustomSource` | Represents the custom source of string collection for the autocomplete feature,<br>when `AutoCompleteSource` property is set to `CustomSource`. |
| `AutoCompleteSource`  | Represents the source of strings used for autocomplete. The sources can be,<br>- FileSystem,<br>- AllSystemSources, (Default)<br>- AllUrl,<br>- CustomSource,<br>- FileSystemDirectories,<br>- HistoryList,<br>- ListItems,<br>- RecentlyUsedList and<br>- None. |
| `AutoCompleteMode`   | Indicates text completion behavior of the combo box. The modes are,<br>- Suggest - Displays the drop down list associated with the EditControl. This dropdown list is populated with one or more suggested completion strings,<br>- Append - Appends the reminder of the most likely candidate string to the existing character, highlighting the appended character, and<br>- SuggestAppend - Displays the drop down, also appends the highlighted string. |
| `ShortCut on Form1` | Specifies the Keyboard shortcut to be used at run time to access this combobox. |

## Code Examples

```csharp
// Example of setting AutoCompleteSource to CustomSource
comboBox1.AutoCompleteSource = AutoCompleteSource.CustomSource;
comboBox1.AutoCompleteMode = AutoCompleteMode.Suggest;
comboBox1.AutoCompleteCustomSource = new AutoCompleteStringCollection();
comboBox1.AutoCompleteCustomSource.Add("Example 1");
comboBox1.AutoCompleteCustomSource.Add("Example 2");

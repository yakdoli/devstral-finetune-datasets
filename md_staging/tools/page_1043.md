# Essential Tools for Windows Forms

## Overview
- This section describes the configuration of the FlowLayout control, focusing on the LayoutMode property.
- Learn how to set the layout mode to either horizontal or vertical for better control arrangement.
- Includes code examples in both C# and VB.NET for setting the LayoutMode property.

## Content

### Layout Mode

The layout mode dictates the core function of a FlowLayout, whether to layout the Child controls horizontally or vertically. This property will be in effect for both the scenarios.

#### FlowLayout Property Details

| FlowLayout Property | Description |
|----------------------|-------------|
| LayoutMode           | Specifies the layout mode of the Child controls. The default value is set to 'Horizontal'.<br>The options included are as follows:<br>**Horizontal** and<br>**Vertical**. |

#### Code Examples

- **C#**

```csharp
this.flowLayout1.LayoutMode =
    Syncfusion.Windows.Forms.Tools.FlowLayoutMode.Vertical;
```

- **VB.NET**

```vb
Me.flowLayout1.LayoutMode =
    Syncfusion.Windows.Forms.Tools.FlowLayoutMode.Vertical
```

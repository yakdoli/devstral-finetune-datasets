# Essential Tools for Windows Forms

## Overview

- Explains the `FullRowSelect` property of the `treeViewAdv`.
- Describes the `GetHeightOfRows` method in the `treeViewAdv`.
- Provides code examples for both C# and VB.NET to demonstrate the usage of these functionalities.

## Content

### `treeViewAdv Properties`

| `treeViewAdv Property` | Description |
|-------------------------|-------------|
| FullRowSelect | Specifies whether the whole row of a treeview needs to be selected on selecting a node of that row. |

### `treeViewAdv Methods`

| `treeViewAdv Method` | Description |
|-----------------------|-------------|
| GetHeightOfRows | To get the height of the rows of a tree from the start point to the end point. The parameters are: <br> (i) `start` - Represents the start point. <br> (ii) `end` - Represents the end point. |

## API Reference

### `FullRowSelect` Property

- **Description:** A boolean property that determines whether the entire row of a treeview should be selected when a node in that row is selected.

### `GetHeightOfRows` Method

- **Parameters:**
  - `start`: Integer representing the start point of the row selection.
  - `end`: Integer representing the end point of the row selection.
- **Returns:** An integer representing the height of the rows between the specified `start` and `end` points.

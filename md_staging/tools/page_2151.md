# Essential Tools for Windows Forms

## Overview
- This page provides instructions for using the `TreeViewAdv NodeCollection Editor` in Windows Forms.
- Instructions include adding, removing, and modifying tree nodes through both the UI and code.

## Content

### Collection Editor Property Grid

#### Figure 1432: Collection Editor Property Grid

This section explains how to interact with the `TreeViewAdv NodeCollection Editor`.

1. Click "Add Node" to add another sibling to the selected node.
2. Click "Add Child" to add a child node to the selected node.
3. Repeat steps 5 and 6 as required in the application.
4. Click "Remove" to delete a selected node.
5. To move a node to a different parent, just drag-and-drop that node over the parent or besides the desired sibling.
6. Click "OK" to save changes.

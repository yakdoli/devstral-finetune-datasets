## Overview
- This page explains how to create a GroupBar in a Windows Forms application using the Syncfusion library.
- It demonstrates the process of creating a GroupBar through the designer.
- The benefits of adding GroupBar items using design-time verbs are outlined.

## Content

### Creating GroupBar

#### Overview
This tutorial will show you how a GroupBar is created through the following ways.

##### Through Designer

The GroupBar's designer allows you to add new groups using its design-time verbs or the property browser. Adding a Child control to a group involves activating the particular group and dropping the control onto the GroupBar.

To create a GroupBar through designer:
- Drag-and-drop a GroupBar object from the controls toolbox onto your form.

#### Adding a GroupBar Item Using Design-Time Verbs

To add a GroupBar Item using design-time verbs, follow the procedure given below:

1. Drag-and-drop a GroupBar object from the controls toolbox onto your form.
   ![Figure 890: GroupBar in Toolbox](image.png)

2. To add a GroupBar Item using design-time verbs, right-click on the GroupBar and select the Add Group option. This adds a GroupBar Item to the GroupBar control.

## Content

### 3.22.1.1 Features

This section discusses the Features of Wizard control.

The Wizard control allows users to add/remove pages at design-time using verbs or the WizardControl Page Collection editor. Images, titles, and descriptions can also be easily added to enhance the user experience.

#### User-friendly design environment

- The Wizard control enables the user to add or remove pages at design-time using verbs or the WizardControl Page Collection editor.
- Images, titles, and descriptions can be easily added to the wizard control.

#### Navigation

- The Wizard control facilitates navigation through different pages at design time using verbs or by clicking on the Next and Back buttons.

#### Customizable user interface properties

- The Wizard control creates child controls representing the different portions of the wizard and exposes them in the designer for user customization.

## Code Examples

```csharp
using Syncfusion.WinForms.Tools;

// Create a new WizardControl
WizardControl wizard = new WizardControl();

// Add a new page
wizard.AddPage("Page 1", new Page1Control());

// Set page title
wizard.SetPageText(0, "Welcome Page");

// Handle PageChanged event
wizard.PageChanged += (sender, e) => {
    MessageBox.Show($"Current page: {e.PageIndex}");
};

// Show the wizard
wizard.ShowDialog();
```

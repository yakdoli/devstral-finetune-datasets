## Essential Tools for Windows Forms

### Custom Dictionary Editor Dialog

The Custom Dictionary Editor dialog (Figure 651) allows users to manage custom dictionary entries. It includes fields for the custom dictionary path, adding words, and editing or deleting entries from the dictionary. 

**Figure 651: Custom Dictionary Editor Dialog**

### SpellChecker Options Dialog

The SpellChecker Options dialog (Figure 652) provides settings to customize how the SpellChecker handles specific types of text, such as ignoring words in different cases, file names, Internet addresses, email addresses, HTML tags, and special characters.

**Figure 652: SpellChecker Options Dialog**

# Essential Tools for Windows Forms

## Content

### Displaying Ticks

Ticks are indicators placed at regular intervals on the slider for a range defined by the maximum and minimum scale. You can display ticks below the channel by setting the `ShowTicks` property to `True`.

#### Example: Setting ShowTicks to True

When the `ShowTicks` property is set to `True`, the ticks are displayed below the slider.

```csharp
rangeSlider.ShowTicks = True;
```

#### Example: Setting ShowTicks to False

When the `ShowTicks` property is set to `False`, the ticks are not displayed below the slider.

```csharp
rangeSlider.ShowTicks = False;
```

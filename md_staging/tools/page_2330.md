# Essential Tools for Windows Forms

![Properties window with WizardControl settings](wizardControl1.png)

## Overview
- Demonstrates accessing properties of a Wizard Page through the `SelectedWizardPage` property of the `WizardControl`.
- Explains how to hide the header section by setting the `FullPage` property to `true`.

## Content

### Figure 1501: Accessing Properties of a Wizard Page Through SelectedWizardPage Property of Wizard Control

The header section (GradientPanel and its child controls) can be hidden by setting the `FullPage` property to `true`. This makes the page occupy the entire space without the header.

| Property     | Description                                                                                                                                                                                                                                                                                                                                                              |
|--------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| FullPage     | Gets/sets the boolean value whether the Banner Panel should be shown for that page. Default value is false.                                                                                                                                                                                                                                                         |

### Code Examples

#### C#
```csharp
this.wizardControlPagel.FullPage = true;
```

#### VB.NET
```vb
Me.wizardControlPagel.FullPage = True
```

# Essential Tools for Windows Forms

## Overview
- Demonstrates setting the Title and Description for a Wizard Page in VB.NET.
- Expores accessing Wizard Page properties using the SelectedWizardPage property.
- Explains the role of LayoutName in identifying individual Wizard pages.

## Content

### Setting Title and Description for a Wizard Page

```vb
[VB.NET]

Me.wizardControlPage1.Title = "Registration Details"
Me.wizardControlPage1.Description = "Please enter your Details:"
```

### Accessing Wizard Pages

We can also access the properties of a Wizard Page using the `SelectedWizardPage` property of the `WizardControl` in the Designer.

#### Property Table

| Property                | Description                                                                 |
|-------------------------|-----------------------------------------------------------------------------|
| SelectedWizardPage      | Specifies the selected wizard page.                                        |
| WizardPage.LayoutName   | The individual Wizard page is identified using its LayoutName in the <br> `SelectedWizardPage` property. By default the LayoutName is set as Card1 for the first page added, Card2 for the next page and so on. |

## API Reference

### Properties

- **SelectedWizardPage**
  - Specifies the selected wizard page.
- **WizardPage.LayoutName**
  - Identifies the individual Wizard page using its LayoutName in the `SelectedWizardPage` property. By default, the LayoutName is set as Card1 for the first page added, Card2 for the next page, and so on.

# Essential Tools for Windows Forms

## Overview
- ButtonAdv extends the Windows Forms Button control with enhanced features.
- ButtonEdit provides a customized edit control with associated buttons.
- Calculator Control offers a complete calculator implementation.
- CheckBoxAdv enhances the basic check box control with advanced features.

## Content

### ButtonAdv
**Features**
- Easily change the button type to browse, calculator, etc.
- Supports PushButton behavior.
- Disables the dotted rectangle when focused using the KeepFocusRectangle property.
- Supports various BorderStyles.
- Compatible with XP, Office 2003, and Office 2007 Visual Styles.

### ButtonEdit
**Description**
It facilitates the creation of controls with an edit box and any number of associated buttons.

**Features**
- Buttons can be added and configured at design-time.
- Custom widths can be specified for buttons.
- Buttons can be aligned to the left or right of the text box.
- Events can be handled to customize functionality.
- Supports XP, Office 2003, and Office 2007 Visual Styles.

### Calculator Control
**Description**
Provides an easy way to implement a complete calculator in an application.

**Features**
- Implements all common arithmetic functions.
- Supports keyboard input.
- Can be displayed as a form or a popup.
- Available in Windows Standard and financial layouts.
- Complete designer support.
- Customizes font and color of buttons.
- Adjusts spacing between buttons using VerticalSpacing and HorizontalSpacing properties.
- Compatible with XP, Office 2003, and Office 2007 Visual Styles.

### CheckBoxAdv
**Description**
An advanced check box control capable of displaying gradients and images.

### Summary
These tools provide enhanced functionality and visual options, streamlining the development of user-friendly applications.

### Adding HighlightBorderColor property

#### Overview
The Essential tool now enhances with the **HighlightBorderColor property** to set the highlight color of the column header in MultiColumnTreeView.

#### Description
- **HighlightBorderColor property** allows developers to customize the highlight color of the column header in MultiColumnTreeView.
- This feature provides greater flexibility in tailoring the visual appearance of the control to fit specific requirements.

## API Reference

### Properties
- **HighlightBorderColor**:  
  Type: Color  
  Default: None  
  Description: Sets the highlight color of the column header in MultiColumnTreeView.

### Sample Code in C#

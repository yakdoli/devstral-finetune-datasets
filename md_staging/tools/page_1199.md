### 3.7.4.4 Popup Menu

A `PopupMenu` represents a context menu for `XPMenu`, that can be displayed over any control when the user right-clicks anywhere in an application. It must be associated with a `ParentBarltem` whose contents will be displayed in the popup menu. Similar to the `.NET` Context menus, a popup menu can be displayed. The `PopupMenusManager` class should be added to the designer to display the `PopUp` menu.

#### Figure 814: Popup Menu display on a TextBox Control

![Figure 814: Popup Menu display on a TextBox Control](image.png)

The image shows a `PopupMenu` displayed on a `TextBox` control, with standard editing options such as Undo, Redo, Cut, Copy, Paste, Find, Font Color, and Select All.

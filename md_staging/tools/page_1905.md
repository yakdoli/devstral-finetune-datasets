# Essential Tools for Windows Forms

## Content

### Overview
- Working with the Toolbox in Windows Forms.
- Setting properties for RibbonControls.
- Configuring RibbonStyle to match Office2010 theme.

### WinForms Setup

**Figure 1284: Toolbox**
The Toolbox panel contains various controls that can be added to a Windows Forms application. Key elements include:

- **PropertyEditor**
- **RadioButtonAdv**
- **RangeSlider**
- **RecordNavigationBar**
- **RecordNavigationControl**
- **RibbonControlAdv**
- **RibbonPanelMergeContainer**
- **ScheduleControl**
- **ScrollersFrame**
- **SparkLine**
- **SpellChecker**
- **SplashControl**
- **Server Explorer**

These tools provide advanced functionalities for designing and customizing Windows Forms applications.

#### Step-by-Step Configuration

1. **Access the Toolbox**: Locate and open the Toolbox as shown in Figure 1284.

2. **Select RibbonControlAdv**: From the Toolbox, select the `RibbonControlAdv` control.

3. **Configure Properties**: Use the Property Grid to set specific properties for the `RibbonControlAdv`.

**Figure 1285: RibbonStyle in Property Grid**
4. **Set RibbonStyle to Office2010**: In the Property grid, set the `RibbonStyle` property to `Office2010`.

#### Property Grid Details
The Property Grid for the `RibbonControlAdv` provides several configurable properties, including:

- **QuickPanelVisible**: `True`
- **RibbonStyle**: `Office2010`
- **RightToLeft**: `Office2007`
- **ScaleMenuButtonImage**: `Office2010`
- **ShowCaption**: `True`
- **ShowContextMenu**: `True`
- **ShowLauncher**: `True`
- **ShowMinimizeButton**: `False`
- **ShowPanelOnMouseHove**: `False`
- **ShowQuickItemsDropDownButton**: `True`
- **ShowQuickPanelBelowRibbon**: `False`
- **Size**: `155, 55`
- **SortTabItems**: `False`
- **SuperAccelerator**: `(none)`

The `RibbonStyle` property specifically controls the visual style of the ribbon, allowing it to match the Office2010 theme.

## API Reference

### RibbonControlAdv Properties

- **RibbonStyle**: Specifies the visual style of the ribbon.
  - **Type**: `string`
  - **Description**: Gets or sets the style of the ribbon.
  - **Default**: `Office2010`
  - **Required**: No
  - **Valid Values**: `Office2007`, `Office2010`

## Code Examples

### Example: Configuring RibbonStyle

```csharp
using Syncfusion.Windows.Forms.Tools;

// Instantiate RibbonControlAdv
RibbonControlAdv ribbonControl = new RibbonControlAdv();

// Configure RibbonStyle
ribbonControl.RibbonStyle = "Office2010";

// Add other configurations as needed
```

# Essential Tools for Windows Forms

## Overview
- Implementation of custom quick access toolbar with Office 2007 look and feel.
- Description of Office 2007 form features and compatibility.
- Highlight of color schemes and user interface elements.

## Content

### Customize Quick Access Toolbar

#### Figure: Office2007 Look and Feel for QAT Dialog Box
The figure illustrates a dialog box titled "Customize Quick Access Toolbar," designed with the Office 2007 look and feel. This dialog allows users to customize the toolbar by selecting commands from a list on the left side and adding them to the right side. The interface includes buttons for adding, removing, resetting, and confirming changes.

### 3.12.1.1 Office2007 Form

#### Description
The Office 2007 Form is now available in the Essential suite. It does not have any dependency on RibbonControlAdv, making it stand-alone and versatile. The form supports all three color schemes and features a help button, Right To left functionality, and an overall design similar to the normal form but with the distinct Office 2007 aesthetic.

#### Features
- **Color Schemes**: Supports all three color schemes available in Office 2007.
- **Help Button**: Includes a help button for user assistance.
- **Right-To-Left Support**: Provides support for Right-To-Left languages for enhanced functionality.
- **Office 2007 Look and Feel**: Maintains the visual consistency with Office 2007, ensuring a familiar user experience.

## API Reference

### Forms API
The Office 2007 Form functionalities can be accessed and customized through the Essential suite's provided libraries, ensuring seamless integration with existing applications.

## Code Examples

### C# Example
```csharp
using Syncfusion.Windows.Forms;
// Configure the Office 2007 form with custom settings.
Form form = new Office2007Form();
form.ColorScheme = Office2007Theme.Blue;
form.ShowHelpButton = true;
form.EnableRtl = true;
form.Show();
```

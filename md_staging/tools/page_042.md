# Essential Tools for Windows Forms

## Overview
- Provides detailed instructions on accessing both online and installed documentation for Windows Forms.
- Highlights the path to access the Class Reference in both online and installed documentation.
- Emphasizes the availability of PDF versions for easier offline access.

## Content

### Note
Click Download as PDF to access a PDF version of the documentation.

#### Installed Documentation
- **Path**: Dashboard -> Documentation -> Installed Documentation.

### Class Reference

#### Online
- **Website**: [http://help.syncfusion.com/resources](http://help.syncfusion.com/resources)
- **Navigation Steps**: Navigate to the Windows Forms User Guide. Select `Tools` in the second text box, and then click the Class Reference link found in the upper right section of the page.

#### Installed Documentation
- **Path**: Dashboard -> Documentation -> Installed Documentation.

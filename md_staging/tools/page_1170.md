# Essential Tools for Windows Forms

## Overview

- This page guides you through the process of grouping bar items in a Windows Forms application using Syncfusion WinForms.
- It covers both the designer interface and programmatic approaches to achieving this functionality.
- Includes steps to insert separators and organize bar items into groups.

## Content

### Designer Approach

1. **Select a Bar Item**
   - Select a bar item in the dropdown from which you want to start a group.
   - Right-click on the selected bar item.
   - Choose the "Begin A Group" option from the Context Menu.
   - This inserts a separator from the bar item selected and begins a group.

   **Figures:**

   **Figure 786: Enabling Begin a Group Option in the Designer**

   **Figure 787: Grouped Bar Items**

   **Explanation:**
   - The context menu allows you to start a group at the selected bar item.
   - A group is created, organizing the items as demonstrated in Figure 787.

### Programmatic Approach

- **Objective:** Programmatically draw a separator between BarItems in a bar.
- **Method:** Use the `BeginGroupAt` method, as shown below.

```csharp
// Example Usage
barManager.BeginGroupAt(barItem);
```

- Replace `barItem` with the specific bar item where you want the group to begin.
- This approach is useful when designing dynamically or through code-based configurations.

## API Reference

### Methods

- **`BeginGroupAt(barItem)`**
  - **Description:** Inserts a separator at the specified bar item and begins a new group.
  - **Parameters:**
    - `barItem`: The bar item at which the group is to begin.
  - **Returns:** None.

## Code Examples

```csharp
// Example: Programmatic Grouping of Bar Items
barManager.BeginGroupAt(copyBarItem);
barManager.BeginGroupAt(pasteBarItem);
```

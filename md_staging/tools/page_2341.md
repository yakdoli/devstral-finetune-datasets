## Overview
- Demonstrates customization of button styles using properties like ForeColor and BorderSize.
- Describes how to access properties of specific buttons within a WizardControl.
- Highlights properties of BackButton, CancelButton, FinishButton, HelpButton, and NextButton within the WizardControl.

### Accessing Button Properties in WizardControl

**Note:** You can access the properties of CancelButton, FinishButton, HelpButton, and NextButton using the following methods:
- `WizardControl.CancelButton`
- `WizardControl.FinishButton`
- `WizardControl.HelpButton`
- `WizardControl.NextButton`

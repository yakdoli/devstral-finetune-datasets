##### 3.22.1.4.1 Banner Settings
The controls inside the Banner Control include Banner Panel, Title, description, and a picture box. This section will walk you through the properties that customize these controls.

**Banner Panel**
A Banner Panel is a simple gradient panel which holds a Title label, a Description label, and a Picture box controls.

| Property   | Description                                                  |
|------------|--------------------------------------------------------------|
| Banner     | Gets or sets the picture box for the wizard                 |

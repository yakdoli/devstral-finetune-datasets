# Essential Tools for Windows Forms

## Overview
- Describes text direction properties like TextDirection.
- Specifies relative placement of text and image using TextImageRelation.
- Includes details about Image Settings such as Image and ImageAlign.

## Content

### Text Direction and TextImageRelation
| Property         | Description                                                                 |
|------------------|-----------------------------------------------------------------------------|
| TextDirection    | Specifies the direction of drawing the text. The directions are:<br>    <br>• Horizontal - Text is placed horizontally.<br>• Vertical90 - Text is placed vertically.<br>• Vertical270 - Text is placed vertically at 270 degrees. |
| TextImageRelation | Specifies the relative location of the image to the text on the item. The options are:<br>• Overlay - Image and text share the same space in the control.<br>• ImageAboveText - Image will be placed above the text.<br>• TextAboveImage - Text will be placed above the image.<br>• ImageBeforeText - Image will be placed before the text.<br>• TextBeforeImage - Text will be placed before the image. |

### Image Settings
| Property   | Description                                                                 |
|------------|-----------------------------------------------------------------------------|
| Image      | Sets the image for the item.                                               |
| ImageAlign | Specifies the alignment of the image. The options are:<br>• TopLeft,<br>• TopCenter,<br>• TopRight,<br>• MiddleLeft,<br>• MiddleCenter,<br>• MiddleRight,<br>• BottomLeft,<br>• BottomCenter and |
|            | BottomRight.                                                                 |

## API Reference (if applicable)
### Properties
- `TextDirection`: Specifies the direction of text placement (Horizontal, Vertical90, Vertical270).
- `TextImageRelation`: Determines how image and text are positioned relative to each other.
- `Image`: Sets the image for the item.
- `ImageAlign`: Specifies the alignment of the image (TopLeft, TopCenter, TopRight, MiddleLeft, MiddleCenter, MiddleRight, BottomLeft, BottomCenter, BottomRight).

### ENUMS
- `TextDirection`: Enumeration for text drawing direction.
- `TextImageRelation`: Enumeration for image and text placement options.
- `ImageAlign`: Enumeration for image alignment options.

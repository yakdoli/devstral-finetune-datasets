## API Reference (if applicable)

### Namespace: Syncfusion.Windows.Forms.Tools

- **Classes**
  - `GradientPanel`: Extends the functionality of panels with gradient support.
    - **Properties**: BackgroundGradientStyle, BorderStyle, ScrollBars
    - **Events**: Scroll, Resize
  - `GradientPanelExt`: Provides advanced features for gradient panels.
    - **Properties**: ExpandCollapseAnimation, ScrollAnimation
    - **Events**: OnExpand, OnCollapse
  - `SplitContainerAdv`: Advanced split container control.
    - **Properties**: Panel1, Panel2, SplitterDistance
    - **Events**: SplitterMoved

### Methods/Properties/Events/Enums

- **Examples**:
  - `GetGradientPanelBackground()`: Returns the current gradient background style.
  - `SetSplitterPosition()`: Sets the position of the splitter in a `SplitContainerAdv`.

## Code Examples

### C#
```csharp
GradientPanel gradientPanel = new GradientPanel();
gradientPanel.BorderStyle = BorderStyle.FixedSingle;

GradientPanelExt gradientPanelExt = new GradientPanelExt();
gradientPanelExt.Expand();

SplitContainerAdv splitContainer = new SplitContainerAdv();
splitContainer.SplitterDistance = 200;
splitContainer.SplitterMoved += OnSplitterMoved;

void OnSplitterMoved(object sender, EventArgs e)
{
    // Handle splitter move event
}
```

```vb

    ' Handle splitter move event

- Detailed description of various color properties for menu items and controls.

- **MenuComboboxButtonHighlightLightColor**: Gets or sets the light color for the highlighted ComboboxButton of the menu.

- **MenuComboboxButtonPushed2Color**: Color of the combobox button in the menu dropdown (Top half).
- **MenuComboboxButtonPushed3Color**: Color of the combobox button in the menu dropdown (Middle).

- **MenuItemArrowLightColor**: Arrow button light color of the submenu parent / dropdownbaritem.
- **MenuItemBorderColor**: Gets or sets the border color for the highlighted item of the menu.
- **MenuItemDarkColor**: Gets or sets the dark color for the highlighted item of the menu.

## Code Examples

```csharp
// Example: Setting menu colors programmatically
using Syncfusion.Windows.Forms.Tools;

// Create an instance of the menu
Syncfusion.Windows.Forms.Tools.ToolStripMenuItem menu = new Syncfusion.Windows.Forms.Tools.ToolStripMenuItem();

// Set the menu highlight dark color
menu.Colors.MenuCheckedColor = Color.Red;

// Set the menu arrow color
menu.Colors.MenuItemArrowDarkColor = Color.Black;

// Additional customization can be done similarly.
```

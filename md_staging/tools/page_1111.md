# Essential Tools for Windows Forms

## Overview
- Provides guidance on adding toolbars and populating bar items in Windows Forms.
- Instructions for creating a new toolbar and naming it within the Customize dialog.

## Content

### 3.7.3.1.2 Adding Toolbars and Populating the Bar Items

To add a toolbar and populate it with the bar items, follow the below steps:

- **Step 1: Create a new toolbar**
  - Go to the **Toolbars** tab in the **Customize** dialog.
  - Select **New** and specify a name (e.g., `MainMenu`) for the toolbar.

<figure>
 <img src="Figure 724: Adding a New Toolbar" alt="Figure 724: Adding a New Toolbar">
 <figcaption>Figure 724: Adding a New Toolbar</figcaption>
</figure>

- **Step 2: Naming the toolbar**
  - This will create a new bar component in the designer.
  - Name this component as `'MainMenu'`. This will also make a corresponding entry in the **Toolbars list**.

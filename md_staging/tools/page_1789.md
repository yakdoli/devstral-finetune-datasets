# Essential Tools for Windows Forms

## Overview
- Explores properties and settings for a ToolstripRadio Button control in Windows Forms.
- Describes alignment options, visibility, size adjustments, and group management for ToolstripRadio Buttons.
- Focuses on Tooltip settings and RTL (Right-to-Left) support.

## Content

### Properties of ToolstripRadio Button

| Property    | Description                                                                                                                                                                                                                                                                                                  |
|-------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **CheckAlign** | Gets or sets the horizontal and vertical alignment of the check mark on a ToolstripRadio Button item. The options are: <br> • TopLeft, <br> • TopCenter, <br> • TopRight, <br> • MiddleLeft, <br> • MiddleCenter, <br> • MiddleRight, <br> • BottomLeft, <br> • BottomCenter and <br> • BottomRight. |
| **Enabled** | Specifies whether the item is enabled.                                                                                                                                                                                                            |
| **Visible** | Specifies whether the item is visible.                                                                                                                                                                                                            |
| **Alignment** | Sets the alignment of the item within the Toolstrip. They can be set to beginning (Left) or end (Right) of the ToolstripEx control.                                                                                                            |
| **AutoSize** | Specifies whether the item should size itself based on its image and text.                                                                                                                                                                      |
| **GroupID** | Gets or Sets Group indicator which is used to create groups of ToolstripRadio Button controls on the same parent.                                                                                                                                  |

### Tooltip Settings

| Property     | Description                                                                                                                                                                                                 |
|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **AutoToolTip** | When set to true, will display the text set in the Text property as the item's tooltip. <br><br> When set to false, will display the text set in the TooltipText property.                                       |
| **TooltipText** | Sets the text for the tooltip when AutoToolTip is set to false.                                                                                                                                           |

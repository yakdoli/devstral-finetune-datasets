# Essential Tools for Windows Forms

## Overview
- Describes how to customize a BarItem's image at runtime.
- Explains the functionality of the ChageImage option in CustomizingPopupMenu.
- Illustrates the steps to select and apply an image to a barite.

## Content

### Image Icon Option of a Barite for CustomizingPopupMenu

ChageImage option is added in **CustomizingPopupMenu**. Barite's image can be changed using ChageImage option.

1. **Right click at the barite** during runtime customization.
   - Note: **CustomizingPopupMenu** appears.

   ![Figure 781: CustomizationPopupMenu](https://example.com/image)

2. Click ChageImage.
   - Note: **File Dialog Opens**.

3. You can select any image for the barite using this dialog.

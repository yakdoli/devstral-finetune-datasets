### Partial Menus
- **Description**: The `ParentBarItem` can hide some of its menu items and display them on clicking an expand button. This is achieved using the `UsePartialMenus` and `IsRecentlyUsedItem` properties. This feature allows the display of recently used items.

#### Feature Implementation
- The feature can be implemented in the `ParentBarItem.BeforePopup` event.
- Key properties to manage partial menus include:

| BarManager Property                    | Description                                                                                          |
|----------------------------------------|------------------------------------------------------------------------------------------------------|
| ExpandPartialMenusAfterDelay           | Enables automatic expansion of the partial menus into full menus after a delay.                    |
| UsePartialMenus                        | Enables or disables partial menus mode in submenus.                                                  |
| PartialMenusResetDelay                 | Specifies the delay in days after an item's recently used setting will be replaced.                 |

### Note
To know all the properties of `BarManager`, click [here](<link_to_property_list>).

## API Reference
- **Properties**
  - `ExpandPartialMenusAfterDelay`: Type – Boolean, Default – `false`.
  - `UsePartialMenus`: Type – Boolean, Default – `false`.
  - `PartialMenusResetDelay`: Type – Integer, Default – `7` days.

## Code Examples
- Sample C# code for handling partial menus in the `BeforePopup` event:

```csharp
private void ParentBarItem_BeforePopup(object sender, CancelEventArgs e)
{
    ParentBarItem.UsePartialMenus = true;
    ParentBarItem.IsRecentlyUsedItem = true;
}
```

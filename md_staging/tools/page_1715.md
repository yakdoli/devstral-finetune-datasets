# Essential Tools for Windows Forms

## Overview
- At run time when you press the Alt key, the menu button will display "F" key as shown in the image.
- The RibbonControlAdv now supports the SuperAccelerator functionality, allowing users to access its components through keyboard shortcuts similar to MS Office.
- SuperAccelerator can be wired to the RibbonControlAdv through the designer and through code.

## Content

### 3.12.1.2.3.2 Office-Like Super Accelerator Support

#### 3.12.1.2.3.2.1 Description

The RibbonControlAdv now supports the SuperAccelerator functionality. This allows the user to access the RibbonControlAdv and its components through the keyboard shortcuts as in MS Office.

The SuperAccelerator can be wired to the RibbonControlAdv through the designer and through code.

#### 3.12.1.2.3.2.2 Through Designer

To add the SuperAccelerator key to the RibbonControlAdv, the control must be wired with a SuperAccelerator component as shown in the following image:

![Figure 1134: SuperAccelerator set for MenuButton](./image1.png)

## API Reference (if applicable)
- **Namespace**: Syncfusion.Windows.Forms
- **Class**: RibbonControlAdv
- **Members**:
  - Properties:
    - `SuperAccelerator`: Gets or sets the SuperAccelerator for the RibbonControlAdv.

## Code Examples (multi-language supported)
```csharp
// Adding SuperAccelerator through code
RibbonControlAdv ribbon = new RibbonControlAdv();
SuperAccelerator accelerator = new SuperAccelerator();
ribbon.SuperAccelerator = accelerator;
```

## Essential Tools for Windows Forms

### Adding Bar Items through BarItem Collection Editor

This section demonstrates how to use the BarItem Collection Editor to add bar items within a Windows Forms application.

#### Overview
- Open the BarItem Collection Editor to manage bar items.
- Customize properties such as text, alignment, and behavior for each bar item.

#### Content

##### Figure: Adding bar items through BarItem Collection Editor

![BarItem Collection Editor](../images/baritem-collection-editor.png)

**Figure 812: Adding bar items through BarItem Collection Editor**

The BarItem Collection Editor allows you to add and configure various bar items such as ParentBarItem, DropDownBarItem, ComboBoxBarItem, ListBarItem, StaticBarItem, MdiListBarItem, ToolbarListBarItem, and TextBoxBarItem.

##### Notation

**Note:** This control is not normally used to create toolbars in the XP Menus. It is meant to be used within the form as a stand-alone control.

#### Code Examples

##### C#

```csharp
this.barItem10.Text = "XPMenus";
this.parentBarItem2.Text = "ParentBarItem";
```

##### VB.NET

No example is provided in VB.NET for this section.

#### Summary
- The BarItem Collection Editor provides tools to customize bar items.
- Properties like text, alignment, and behavior can be adjusted through this editor.
- Stand-alone controls within forms can be configured using this method.

# Essential Tools for Windows Forms

## Overview
1. Describes the events and properties related to the XPTaskBar box control.
2. Explains padding settings for the XPTaskBar control.
3. Provides details on customizing the spacing and layout within the XPTaskBar control.

## Content

### Events of XPTaskBar

| Event Name                  | Description                                                                 |
|-----------------------------|-----------------------------------------------------------------------------|
| BeforeAnimation             | This event gets called before the XPTaskBar Box expands or collapses.     |
| CollapsedStateChanged       | This event occurs after the XPTaskBar Box has been collapsed or expanded.  |
| ItemClick                  | This event occurs when a user clicks on an Item.                          |
| MinimumSizeChanged          | This event occurs when the MinimumSize property is changed.              |
| ProvideHeaderBackgroundBrush | This event occurs when the header portion of the XPTaskBar Box gets drawn. |
| ProvideItemsBackgroundBrush | This event occurs when the Items portion of the XPTaskBar Box gets drawn.   |

### 3.9.3.4.5 Padding

#### Padding Settings for XPTaskBar

The interior spacing of the XPTaskBar control can be specified by setting the **DockPadding** property to integer values.

The horizontal and vertical padding can be specified using the **HorizontalPadding** and **VerticalPadding** properties. The default value of both is 'Zero'.

#### XPTaskBar Properties

| XPTaskBar Property       | Description                                              |
|--------------------------|----------------------------------------------------------|
| DockPadding              | Specifies dock padding settings for all edges of the control. |
| HorizontalPadding        | Specifies horizontal spacing between the layout taskbar.    |
| VerticalPadding          | Specifies vertical spacing between the layout taskbar.      |

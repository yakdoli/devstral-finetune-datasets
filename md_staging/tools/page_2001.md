# Essential Tools for Windows Forms

## Overview
- Describes the customization of `TabStyle` in the designer for `TabControlAdv` control.
- Explains the alignment properties of tabs in the `TabControlAdv`.

## Content

### TabStyle in Designer
The `TabStyle` property of the `TabControlAdv` can be customized to modify the visual appearance of the tabs. In the designer, you can select different styles such as "OneNoteStyle," "3D," "Workbook," or "Notched," among others.

#### Figure 1362: Custom TabStyle in Designer
This figure illustrates the property window where the `TabStyle` is configured.

- [TabStyles](TabStyles)

### 3.17.7.1.1 TabAlignment
The alignment of the tabs in the `TabControlAdv` can be controlled using the following properties:

#### Alignment of the Tabs
The alignment of tabs can be set through the following properties:

| TabControlAdv Property | Description |
|-------------------------|-------------|
| Alignment               | Specifies the alignment of tab items with respect to the tab pages. The options include:<br>- Top<br>- Bottom<br>- Left<br>- Right |
| VerticalAlignment       | Specifies whether the tabs are aligned to the Top, Bottom, or based on the RightToLeft property when aligned vertically. |
| TabGap                 | Specifies the space between the tab items. |

## API Reference
The following properties are used to control the alignment and appearance of tabs in `TabControlAdv`:

- **Alignment**: Specifies the alignment of tab items relative to the tab pages.
- **VerticalAlignment**: Determines the vertical alignment of the tabs.
- **TabGap**: Sets the space between tab items.

## Code Examples
```csharp
// Example of setting TabStyle
tabControlAdv1.TabStyle = Syncfusion.Windows.Forms.Tools.TabControlAdv.Notched;

// Example of setting TabAlignment
tabControlAdv1.Alignment = TabAlignment.Top;
tabControlAdv1.VerticalAlignment = VerticalAlignment.Top;
tabControlAdv1.TabGap = 5;
```

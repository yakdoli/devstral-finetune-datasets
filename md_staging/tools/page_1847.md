# Essential Tools for Windows Forms

## Overview
- Describes how to add a StatusStripEx control to a form using the designer.
- Explains docking the StatusStripEx control to the bottom of the RibbonControlAdv.
- Includes steps for configuring properties related to docking.

## Content

### 3.12.2.3.1 Creating a StatusStripEx

#### Through Designer

The StatusStripEx can be added to the form by dragging a StatusStripEx control from the Toolbox. It can be docked to the bottom of the RibbonControlAdv.

## API Reference

- **StatusStripEx**: A control for displaying status information at the bottom of a form.
- **Dock**: Property used to specify the docking behavior of the control.
- **MinimumSize**: Defines the minimum size of the control.
- **Size**: Indicates the current size of the control.
- **Stretch**: Determines whether the control should stretch to fill the available space.
- **OfficeColorScheme**: Defines the color scheme for the control.
- **StatusControlsArea**: Specifies the area where status controls are placed.

## Code Examples

### Adding StatusStripEx in the Designer
```csharp
// Drag-and-drop StatusStripEx from the Toolbox to the form.
// Set the Dock property to Bottom to dock the control at the bottom of the form.
```

### Programmatically Adding StatusStripEx
```csharp
StatusStripEx statusStripEx1 = new StatusStripEx();
statusStripEx1.Dock = DockStyle.Bottom;
this.Controls.Add(statusStripEx1);
```

#### Note: Disabling Customizing Option
If you want to disable Customizing option for the users at run time, set **BarManager.EnableCustomizing** property to `false`. This will not provide the option to open the **Customize** dialog at run time.

##### 3.7.4.1.2.2.1 BarItem

A BarItem is a simple child bar item which can be dragged and dropped to a **ParentBarItem**. By selecting the Type as BarItem and giving the name, in the **Add New BarItem** dialog, we can create a new BarItem.

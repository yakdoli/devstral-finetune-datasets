### Banner Text Support

We can set banner text for the ComboBoxAutoComplete control. Refer **BannerTextProvider Component** topic for more details.

#### 3.5.1.2.3.2 Multi Columns

We can use multiples of columns in the ComboBoxAutoComplete control. In this case, we need to specify which column is to be used as the matching column using the **ComboBoxAutoComplete.AutoCompleteControl.Columns** properties. Adding multiple columns is discussed in the **Multiple Columns** topic in AutoCompleteControl UG.

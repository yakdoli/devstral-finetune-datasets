# Essential Tools for Windows Forms

## Overview
- The ComboBoxAutoComplete control provides autocompletion for text boxes and combo boxes.
- It is efficient when each combo box requires different data sources.
- The control includes its own AutoComplete feature for improved functionality.

## Content

### Features

The ComboBoxAutoComplete control fulfills the need for separate autocompletion for each combo box. It derives from the windows forms combo box control and holds its own AutoComplete control that provides autocompletion. Here are its features:

- **AutoComplete**: The ComboBoxAutoComplete control provides AutoComplete behavior.
- **Columns**: Multiple columns can be used.
- **Datasource**: An external datasource can be specified.

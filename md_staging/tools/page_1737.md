# Essential Tools for Windows Forms

## Overview
- This page discusses the style settings for `ToolStripEx` and how to set keyboard shortcuts for RibbonControl components.
- It highlights the `BorderStyle` and `LauncherStyle` properties of `ToolStripEx`.
- It also introduces the `SetShortcut` and `GetShortcut` methods for managing keyboard shortcuts in RibbonControl components.

## Content

### ToolstripEx Style Settings

| Property       | Description                                                                                                                                                          |
|-----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| BorderStyle     | Sets the border style for the `ToolStripEx`. The options are:<br> - None,<br> - StaticEdge,<br> - Etched (Default). |
| LauncherStyle   | Sets the style of the Launcher button. The options are:<br> - Office2007,<br> - Office12.                      |

**Note:** These caption and style settings can be overridden by the individual `ToolStripEx`'s caption and style settings.

### Keyboard Shortcut

**3.12.2.1.1.1.2 Keyboard Shortcut**

We can also set keyboard shortcut keys for RibbonControl components using the `SetShortcut` method. To get the keyboard shortcut for a particular component, use the `GetShortcut` method.

| Property       | Description                                                                                         |
|-----------------|-----------------------------------------------------------------------------------------------------|
| SetShortcut     | Sets shortcut key. The parameters are:<br> - Component - Component of the RibbonForm.<br> - Value - Represents the shortcut key for the component specified. |
| GetShortcut     | Gets shortcut key. The parameter is:<br> - Component - Component of the RibbonForm.                  |

# Essential Tools for Windows Forms

The `RenderMode` property of the control specifies the painting style applied to the control. The supported styles are:

| Property       | Description                                                                 |
|----------------|------------------------------------------------------------------------------|
| RenderMode     | Represents the painting style applied to the control. The different styles supported are: |
|                | - Professional<br>- System<br>- ManagerRenderMode.                                     |

## Setting the RenderMode

### C#

```csharp
this.contextMenuStripEx1.RenderMode = System.Windows.Forms.ToolStripItemRenderMode.Professional;
```

### VB.NET

```vb.net
Me.contextMenuStripEx1.RenderMode = System.Windows.Forms.ToolStripItemRenderMode.Professional
```

## Overview

- Provides details about the properties of `TreeColumnAdv` in Syncfusion WinForms.
- Explains how to specify column styles, including border styles, sorting, and image settings.
- Discusses the area background settings for columns.

## Content

### TreeColumnAdv Properties

| Property         | Description                                                                 |
|------------------|-----------------------------------------------------------------------------|
| BorderSingle     | Specifies the 2D border style for the columns when `BorderStyle` is set to `Fixed Single`. Options include Dotted, Dashed, Solid, Inset, and Outset. |
| Comparer         | Comparative value for sorting.                                             |
| SortOrder        | Specifies the sort order for the column.                                 |
| LeftImage        | Sets the left image for the column.                                       |
| LeftImageIndices | Specifies the left image index.                                           |
| RightImageIndices| Specifies the right image index.                                          |
| LeftImagePadding | Sets the padding of the left image.                                       |
| RightImagePadding| Sets the padding of the right image.                                      |
| RightImage       | Sets the right image for the column.                                      |

### Note

> **Note:** The `TreeColumnAdv1.Background` settings override the `MultiColumnTreeView.ColumnsHeaderBackground` property settings for individual column headers.

#### Painting the Column Area

| TreeColumnAdv Property | Description                                            |
|------------------------|--------------------------------------------------------|
| AreaBackground         | Gets / sets the background for the column area.       |

# Essential Tools for Windows Forms

## Overview
- This page explains the use of `StatusBarAdvPanel` control in Windows Forms and its theme properties.
- The themes properties allow customization of the control's appearance based on the theme settings.
- Demonstrates how to enable and disable theme support in `StatusBarAdvPanel`.

## Content

### 3.11.5.3.7 Themes

The `StatusBarAdvPanel` control can be provided with a themed look and feel using the properties given below.

#### Figure 1075: ToolTip displayed for StatusBarAdvPanel

![](statusbaradvpanel_tooltip.png)

The `StatusBarAdvPanel` control can be provided with a themed look and feel using the properties given below.

| StatusBarAdvPanel Property            | Description                                                                                                                       |
|---------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| ThemesEnabled                         | Indicates if the background color will be set to 'Transparent'. Indicated Settings: <br>- BorderSides = Right, <br>- BorderStyle = Fixed3D, <br>- Border3DStyle = Etched. |
| IgnoreThemeBackground                 | Indicates whether the control will ignore the theme's background color and draw the backcolor instead.                                    |

#### Code Examples

##### C#

```csharp
this.statusBarAdvPanel1.IgnoreThemeBackground = true;
this.statusBarAdvPanel1.ThemesEnabled = true;
```

##### VB.NET

```vb
Me.statusBarAdvPanel1.IgnoreThemeBackground = True
Me.statusBarAdvPanel1.ThemesEnabled = True
```

## API Reference

### Properties

- **ThemesEnabled**: Indicates whether the control uses the theme for its appearance.
- **IgnoreThemeBackground**: Indicates whether the control ignores the theme's background color.

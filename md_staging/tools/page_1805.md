## Essential Tools for Windows Forms

### Font Dialog Example

This section demonstrates how to display a font dialog box in Windows Forms applications.

#### Code Example in C#

```csharp
Form fontDialog = new Form2();
fontDialog.ShowDialog();
```

#### Code Example in VB.NET

```vb
Private Sub toolStripEx1_LauncherClick(ByVal sender As Object, ByVal e As EventArgs)
    Dim fontDialog As Form = New Form2()
    fontDialog.ShowDialog()
End Sub
```

### Content

The Font dialog box shown in the image allows users to adjust font properties, such as:
- **Font Family**: Selection of different font types.
- **Font Style**: Options like Regular, Italic, Bold, and Bold Italic.
- **Size**: Adjustable font size.
- **Font Color**: Selection of font color.
- **Effects**: Options like Strikethrough, Shadow, Small Caps, etc.
- **Preview**: Displays a live preview of the selected font settings.

### API Reference

The `Form2` class is used to create the Font dialog box. The `ShowDialog()` method is invoked to display the dialog box in a modal manner.

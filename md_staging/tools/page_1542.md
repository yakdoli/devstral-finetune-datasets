# Essential Tools for Windows Forms

## Overview
- Describes the setup and usage of the SplashPanel control in Windows Forms.
- Demonstrates how to configure and integrate the SplashPanel control into applications using the designer.

## Content

### 3.11.3.2.1 Through Designer

The SplashPanel control provides full support for the Windows Forms designer.

1. **Drag-and-drop the SplashPanel control from the toolbox onto the form.**

   ![SplashPanel in Toolbox](image_url_for_Figure_1042)
   
   *Figure 1042: SplashPanel in Toolbox*

2. **Set the properties for the SplashPanel control and also drag and drop any child controls you want to add to the panel. Set the TimerInterval property to specify the period of time the SplashPanel needs to be visible.**

3. **Specify the startup location of the SplashPanel using the DesktopAlignment property.**

4. **Launch the SplashPanel control by calling the ShowSplash() method.**

5. **You can cancel the SplashPanel by calling the HideSplash() method.**

   ![SplashPanel Sample](image_url_for_SplashPanel_Sample)

## API Reference

### Properties
- **TimerInterval:** Specifies the period of time the SplashPanel needs to be visible.
- **DesktopAlignment:** Specifies the startup location of the SplashPanel.

### Methods
- **ShowSplash():** Launches the SplashPanel control.
- **HideSplash():** Cancels the SplashPanel.

## Code Examples

### C#
```csharp
// Setting TimerInterval property
splashPanel1.TimerInterval = 5000; // 5 seconds

// Setting DesktopAlignment property
splashPanel1.DesktopAlignment = DesktopAlignment.Center;

// Launching the SplashPanel
splashPanel1.ShowSplash();

// Cancelling the SplashPanel
splashPanel1.HideSplash();
```

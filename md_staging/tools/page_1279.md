# Essential Tools for Windows Forms

## Border Properties

| Property | Description |
|----------|-------------|
| Border3DStyle | Specifies the 3D style for the border. |
| BorderColor | Sets the border color. |
| BorderSides | Specifies the sides which should have borders. |
| BorderSingle | Specifies the 2D style for the border when `BorderStyle` is set to `FixedSingle`. <br/> Options are, <br/> - Dotted, <br/> - Dashed, <br/> - Solid, <br/> - Inset and <br/> - Outset. |
| BorderStyle | Sets the border style, either `FixedSingle` or `Fixed3D`. |

### StyleSettings

StyleSettings for all the subitems can be specified using the `TreeNodeAdvSubItemStyleInfo` class. It is a default base style which can be accessed in the BaseStyles Collection Editor.

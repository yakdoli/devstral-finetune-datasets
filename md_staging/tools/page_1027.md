## Content

### Process for Finding Preferred Size

#### Finding Preferred Size

The following steps illustrate the process for determining the preferred size for a Child Control:

1. **Child Provides Dynamic Size Information?**
   - **Yes**: Proceed to **Query Child**.
   - **No**: Move to the next step.

2. **ProvideLayoutInformation Handler Available?**
   - **Yes**: Fire the **ProvideLayoutInformation** event.
     - If the **Handler provided Size Information?**, use that size.
     - Otherwise, move to the next step.
   - **No**: Move to the next step.

3. **PreferredSize explicitly specified?**
   - **Yes**: Use the explicitly specified **PreferredSize**.
   - **No**: Proceed to **Make PreferredSize = Current Size**.

4. **Make PreferredSize = Current Size**
   - If no other size can be determined, make the **PreferredSize** equal to the **Current Size** of the Child control.

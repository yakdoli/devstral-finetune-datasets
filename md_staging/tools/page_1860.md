# Essential Tools for Windows Forms

## Overview

- This page explains how to apply Office2007 style horizontal scroll bars with various color schemes in Syncfusion WinForms.
- Demonstrates the use of predefined color schemes such as Black, Blue, and Silver.
- Explains how to apply custom colors to the ScrollersFrame using the `ApplyManagedColors` method.

### Custom Colors

We can also apply custom colors to the ScrollersFrame by setting `OfficeColorScheme` to "Managed" and specifying the custom color through the `ApplyManagedColors` method as follows.

#### Code Examples

**[C#]**

```csharp
this.scrollersFrame1.OfficeColorScheme = Syncfusion.Windows.Forms.Office2007ColorScheme.Managed;
Office2007Colors.ApplyManagedColors(this, Color.LightSkyBlue);
```

**[VB.NET]**

```vb
Me.scrollersFrame1.OfficeColorScheme = Syncfusion.Windows.Forms.Office2007ColorScheme.Managed
Office2007Colors.ApplyManagedColors(Me, Color.LightSkyBlue)
```

## API Reference

### Namespace: Syncfusion.Windows.Forms

#### Class: ScrollersFrame
- **Property**: `OfficeColorScheme`
  - Type: `Office2007ColorScheme`
  - Description: Specifies the color scheme of the control.
  - Default: `None`
- **Method**: `ApplyManagedColors`
  - Parameters:
    - `control`: The control to which the colors are applied.
    - `color`: The custom color to apply.
  - Description: Applies a custom color to the specified control.

### 3.7.4.4.3 Grouping Items in a Popup Menu

This topic will guide on how to group the menu items by inserting separator(s), in a popup menu with and without `BarManager`.

**If the `ParentBarIem` associated with the popup menu is contained within a `BarManager`, dropdown the popup menu from the Popup Form, right click on an item and select `Begin A Group` from the context menu.**

#### Figure: Grouping Menu Items in a Popup Menu

![Grouping Menu Items in a Popup Menu](image_url)

1. Navigate to the `Customize` dialog via the `Customization` window.
2. Select the item under the `Commands` tab.
3. From the context menu, click on `Begin A Group`.
4. The separator will now appear in the popup menu.

### Summary

This section explains the process of organizing items within a popup menu by inserting separators, facilitating a more structured user interface.

## API Reference

- **Namespace**: Syncfusion.Windows.Forms
- **Class**: `PopupMenu`, `BarManager`
- **Methods**:
  - `Show()`: Displays the popup menu.
  - `InsertSeparator()`: Adds a separator in the popup menu.

### Code Examples

```csharp
// Programmatically showing a popup menu
PopupMenu popupMenu = new PopupMenu();
popupMenu.Items.Add(new MenuItem("Undo"));
popupMenu.Items.Add(new MenuItem("Redo"));
popupMenu.Items.Add(new Separator()); // Adding a separator
popupMenu.Items.Add(new MenuItem("Cut"));
popupMenu.Show(Cursor.Position, new MenuFlags());
```

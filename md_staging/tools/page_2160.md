# Essential Tools for Windows Forms

## Overview
- The page discusses how to customize image sizes in the `TreeViewAdv` control.
- Explains the concept of expand and collapse images in `TreeNodes`.
- Details the implementation using `NodeStateImageList` and `DefaultExpandImageIndex` along with `DefaultCollapseImageIndex`.

## Content

### Customizing Image Sizes in `TreeViewAdv`

```vb.net
[VB.NET]
Me.leftImageList.ImageSize = New System.Drawing.Size(16, 16)
Me.rightImageList.ImageSize = New System.Drawing.Size(16, 16)
Me.stateImageList.ImageSize = New System.Drawing.Size(15, 15)
```

### Expand and Collapse Image

When child nodes are added to a node, automatically the expand / collapse (+/-) images are set by default, to the parent node, which indicates whether the nodes are opened or closed. These default images can be replaced with custom images using `NodeStateImageList` property.

- **Key Properties:**
  - **DefaultExpandImageIndex:** Specifies the image for the expanded nodes.
  - **DefaultCollapseImageIndex:** Specifies the image for the collapsed nodes.

## API Reference

### Properties
- **DefaultExpandImageIndex:** Gets or sets the default value for expanding the image index.
- **DefaultCollapseImageIndex:** Gets or sets the default value for collapsing the image index.

### Example Usage

```vb.net
Dim treeNode As TreeNodeAdv = New TreeNodeAdv("Node1")
treeNode.DefaultExpandImageIndex = 1
treeNode.DefaultCollapseImageIndex = 2
```  

### Summary
This documentation provides guidance on customizing images in `TreeViewAdv`, which is particularly useful for enhancing the user experience by providing visually distinct symbols for expand and collapse functionalities. The use of custom `NodeStateImageList` allows developers to Create a visually consistent and intuitive interface.

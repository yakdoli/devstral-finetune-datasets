# Essential Tools for Windows Forms

## Overview
- Describes the GroupBar and its support for integrated scrolling in GroupView client controls.
- Details the properties for GroupBar Items when in StackedMode.

## Content

### GroupBar with Integrated Scrolling

The GroupBar provides scrolling support for GroupView client controls. Here's how to enable integrated scrolling:

```csharp
this.groupBar1.IntegratedScrolling = true;
```

```vbnet
Me.groupBar1.IntegratedScrolling = True
```

### StackedGroupBar

#### 3.9.1.4.1.2 StackedGroupBar

The following properties are available for GroupBar Items when the GroupBar is in Stacked Mode. Enable Stacked Mode by setting the `StackedMode` property to `True`.

| GroupBarItem Property | Description |
|------------------------|-------------|
| InNavigationPane       | Specifies the value which determines whether the GroupBar Item should be added to the GroupBar's navigation pane or not. |
| NavigationPanelIcon    | The icon representing the GroupBar's item in the navigation pane. |
| NavigationPanelImage   | Gets/sets the image representing the item in the navigation pane. |

---

## API Reference

### GroupBar Properties

- **IntegratedScrolling**  
  Enables or disables integrated scrolling for GroupView client controls.

### GroupBarItem Properties (Stacked Mode)

- **InNavigationPane**  
  Determines whether the GroupBarItem should appear in the navigation pane.
  
- **NavigationPanelIcon**  
  Sets the icon for the item in the navigation pane.

- **NavigationPanelImage**  
  Sets the image for the item in the navigation pane.

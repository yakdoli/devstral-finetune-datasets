# Essential Tools for Windows Forms

## Overview
- Demonstrates how to access and modify the Font and ForeColor properties of a Label control in Windows Forms, both through the designer and programmatically.
- Utilizes the `wizardControl1` within the `Syncfusion.Windows.Forms.Tools.WizardControl` namespace.
- Provides examples in both C# and VB.NET to set the font style and foreground color for labels.

## Content

### Accessing Font and ForeColor Properties
The following image illustrates how to access the Font and ForeColor properties of a Label through the designer. The properties such as Font, ForeColor, BackColor, and others are visible and can be modified directly.

![Figure 1496: Accessing Font and ForeColor Properties for Page Title Through Designer](Properties-WizardControl.png)

#### Code Examples

##### C#
```csharp
// Setting Font Style for the Label
this.label1.Font = new System.Drawing.Font("Verdana", 9F, System.Drawing.FontStyle.Bold);
this.label1.ForeColor = System.Drawing.Color.DarkBlue;
```

##### VB.NET
```vb
' Setting Font Style for the Label
Me.label1.Font = New System.Drawing.Font("Verdana", 9F, System.Drawing.FontStyle.Bold)
this.label1.ForeColor = System.Drawing.Color.DarkBlue
```

### Additional Notes
- The example shown uses a Label control (`label1`), but the approach can be applied to any control that supports font and foreground color properties.
- The `FontStyle.Bold` specifies a bold style, and `Color.DarkBlue` sets the foreground color to dark blue.

## API Reference

### Properties
- **Font**: Sets the font for the control. The example uses the `System.Drawing.Font` class with parameters: font name, size, and style.
- **ForeColor**: Sets the foreground color of the control.

### Samples
The provided C# and VB.NET code snippets effectively demonstrate how to set the font and foreground color programmatically.

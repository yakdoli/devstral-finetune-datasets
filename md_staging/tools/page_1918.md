## Content

### 3.12.3.2.3 How to set SuperToolTip for MenuStrip, Toolstrip, or ToolstripEx and RibbonTabControl?

Few Controls like MenuStrip, Toolstrip or ToolstripEx and RibbonTabControl already have the feature of showing Tooltip.

To set the SuperToolTip for such controls, set their `ShowItemToolTips` property to false and then configure the SuperToolTip for those controls.

### 3.12.3.3 Super Accelerator

#### Overview

The following topic is discussed in this section:

#### 3.12.3.3.1 How to get or set an accelerator key programmatically?

To get or set the accelerator key, call `GetAccelerator` and `SetAccelerator` methods respectively.

#### Example: Getting and Setting Accelerator Keys in Code

```csharp
this.superAccelerator1.GetAccelerator(this.toolstripTabItem1);
this.superAccelerator1.SetAccelerator(this.toolstripTabItem1, "C");
```

---

## API Reference

### Methods

- `GetAccelerator(this.toolstripTabItem1):` Retrieves the accelerator key associated with the specified toolstrip tab item.
- `SetAccelerator(this.toolstripTabItem1, "C"):` Sets the accelerator key for the specified toolstrip tab item.

## Code Examples

The following code example demonstrates how to programmatically manage accelerator keys:

```csharp
// Retrieve the current accelerator key
char currentAccelerator = this.superAccelerator1.GetAccelerator(this.toolstripTabItem1);

// Set a new accelerator key
this.superAccelerator1.SetAccelerator(this.toolstripTabItem1, "C");
```

### StandardStyle Settings

The below properties let you customize the standard style settings.

| TreeNodeAdvSubItemStyleInfo Property | Description |
|---------------------------------------|-------------|
| Background                           | Sets the background for the node. |
| CheckColor                           | Indicates the check color. |
| CheckState                           | Indicates the Check state of the node. |
| Enabled                              | Specifies if the node is enabled. |
| EnabledButtons                       | Specifies if the buttons are enabled for the node. |
| Font                                 | Sets the font for the node text. |

## API Reference (if applicable)

### Overview of Properties
The following properties can be used to customize the appearance and behavior of nodes in the TreeNodeAdv control:

- **Background**: Sets the background color of the node.
- **CheckColor**: Specifies the color of the check mark.
- **CheckState**: Indicates the state of the check mark (Checked, Unchecked, Indeterminate).
- **Enabled**: Specifies whether the node is enabled or disabled.
- **EnabledButtons**: Specifies whether additional buttons associated with the node are enabled.
- **Font**: Sets the font properties for the text within the node.
- **Height**: Sets the height of the node.
- **HelpText**: Sets the tooltip (help text) for the node.
- **IntermediateCheckColor**: Specifies the color for the intermediate state of a check mark.

## Code Examples (multi-language supported)

### Example: Configuring TreeNodeAdvSubItemStyleInfo

```csharp
using Syncfusion.Windows.Forms.Tools;

// Example of configuring background and font
TreeNodeAdvNode node = new TreeNodeAdvNode("Sample Node");
node.StyleInfo.Background = Color.LightBlue;
node.StyleInfo.Font = new Font("Arial", 10, FontStyle.Bold);

// Example of enabling tooltips
node.StyleInfo.HelpText = "This is a sample tooltip for the node.";
```

# Essential Tools for Windows Forms

## Overview
- Techniques for setting the border width and applying themes to the `TabControlAdv`.
- Code snippets for configuring border properties and enabling themes in `C#` and `VB.NET`.

## Content

### Setting Border Width

Figure 1374: Default border for the `TabControlAdv`

We can set the width for the border using the `BorderWidth` property. Default value is 5.

```csharp
this.tabControlAdv1.BorderWidth = 10;
```

```vb.net
Me.tabControlAdv1.BorderWidth = 10
```

**Figure 1375: Border Width of `TabControlAdv` is set to 10**

![Border Width of TabControlAdv is set to 10](alt: Visual representation of TabControlAdv with BorderWidth set to 10)

**Note:** These border properties are especially implemented for "VS2008Style" `TabStyles`.

### Applying Themes

#### 3.17.9 Applying Themes

The `TabControlAdv` can be themed by enabling the `ThemesEnabled` property.

**Note:** On setting this property, all the tabpages will be themed. This can be disabled or enabled for individual pages using the `TabPageAdv.ThemesEnabled` property.

```csharp
this.tabControlAdv1.ThemesEnabled = true;
```

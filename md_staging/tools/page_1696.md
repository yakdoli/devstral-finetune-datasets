## Essential Tools for Windows Forms

### Customizing the Top Left Edge

The `TopLeftRadius` property gets/sets the curved radius of the top left edge of the form. Default is `8`.

```csharp
this.TopLeftRadius = 20;
```

```vbnet
Me.TopLeftRadius = 20
```

# Essential Tools for Windows Forms

## Overview
- Discusses key properties and code examples related to configuring the gradient style for a control's background in Syncfusion Winforms.
- Focuses on the use of gradient styles, colors, and patterns to enhance the visual appearance of Windows Forms controls.

## Content

### Control Properties

The following table outlines the essential properties used to configure the gradient styles and patterns for a control:

| Property           | Description                                                                 |
|--------------------|-----------------------------------------------------------------------------|
| ForeColor          | Specifies the foreground color for any text or graphics in the control.    |
| PatternStyle       | Specifies the pattern style of the control.                               |
| GradientBackground | Indicates whether the background will be drawn with the gradient.          |
| GradientStyle      | Specifies the gradient style of the background.                           |
|                    | - ForwardDiagonal                                                         |
|                    | - BackwardDiagonal                                                        |
|                    | - Horizontal                                                              |
|                    | - Vertical                                                                |
|                    | - PathRectangle                                                           |
|                    | - PathEllipse                                                             |
| VerticalGradient   | Indicates whether the gradient is vertical.                               |
| GradientColors     | Specifies the gradient colors.                                           |
|                    | The first entry in this list will be the same as the BackColor property,  |
|                    | and the last entry will be the same as the ForeColor property.           |

### Code Examples

#### C#

```csharp
this.statusBarAdv1.BackColor = new Syncfusion.Drawing.BrushInfo(Syncfusion.Drawing.GradientStyle.PathRectangle, System.Drawing.Color.NavajoWhite, System.Drawing.Color.IndianRed);
```

#### VB.NET

```vb
Me.statusBarAdv1.BackColor = New Syncfusion.Drawing.BrushInfo(Syncfusion.Drawing.GradientStyle.PathRectangle, System.Drawing.Color.NavajoWhite, System.Drawing.Color.IndianRed)
```

## API Reference

### Properties

- **ForeColor**: Specifies the foreground color for any text or graphics in the control.
- **PatternStyle**: Specifies the pattern style of the control.
- **GradientBackground**: Indicates whether the background will be drawn with the gradient.
- **GradientStyle**: Specifies the gradient style of the background.
- **VerticalGradient**: Indicates whether the gradient is vertical.
- **GradientColors**: Specifies the gradient colors.

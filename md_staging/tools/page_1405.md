# Essential Tools for Windows Forms

## Overview

1. Illustrates the `GroupViewItemHighlighted` event using a sample GroupView window.
2. Describes the `GroupViewItemRenamed` event, which occurs when a GroupView item is renamed via an inplace edit operation.
3. lists the properties associated with the `GroupItemRenamedEventArgs` event arguments.

#### GroupViewItemRenamed Event

- **Description:** It occurs when a GroupView Item has been renamed by an inplace edit operation.
- **Event Handler Arguments:** The event handler receives an argument of type `GroupItemRenamedEventArgs`. The properties associated with this event are as follows:

| Members             | Description                                                                 |
|---------------------|-----------------------------------------------------------------------------|
| **BackgroundBrush** | Gets / sets the brush that will be used to draw the specified bounds.      |
| **Bounds**          | Returns the bounds for which a brush is requested.                         |
| **Item**            | Returns the index of the GroupBar Item being drawn.                        |

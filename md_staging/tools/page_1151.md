# Essential Tools for Windows Forms

## Overview
- Describes the PaintStyle property for displaying MenuItems in different styles (TextOnly, ImageOnly, or Image and Text).
- Explains the various options available in the PaintStyle property and their behavior.

## Content

### Paint Style

The MenuItem can be displayed as a TextOnly, ImageOnly, or Image and Text using the PaintStyle property.

#### Figure 767: PaintStyle property Options

![PaintStyle property Options](Properties_dropdownBarItem1_Syncfusion.Windows.Forms.T_PaintStyle_options)

#### BarItem Property Description

| BarItem Property        | Description                                                                                                                                                                                                                                                                                   |
|-------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| PaintStyle             | Indicates the painting style of the bar item.                                                                                                                       |
|                         | - **Default**: Displays the style of its parents. In case of MainMenu, only text will be drawn; in other toolbars, only image; and in a Dropdown menu, both image and text will be drawn.                                   |
|                         | - **TextOnly**: Displays only text.                                                                                                                                                                                             |
|                         | - **TextOnlyInMenus**: Image will be ignored when the BarItem is in a drop-down menu.                                                                                                                                         |
|                         | - **ImageAndText**: Both Image and Text will be drawn.                                                                                                                                                                        |

## API Reference

### Properties

- **PaintStyle**
  - **Type**: Enum
  - **Description**: Indicates the painting style of the bar item.
  - **Options**:
    - **Default**: Inherits the style of its parents.
    - **TextOnly**: Displays only text.
    - **TextOnlyInMenus**: Displays text only in menus, ignoring images.
    - **ImageAndText**: Displays both image and text.

# Essential Tools for Windows Forms

## Overview

- Lists properties related to border customization in Windows Forms.
- Explains border styles, sides, and colors.
- Defines the default border value as 'Sunken'.

## Content

### BorderStyle Properties

| Property         | Description                                                                                                                                                  |
|------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| BorderStyle       | Determines the style of the 2D border. The options include: <br> RaisedOuter, <br> SunkenOuter, <br> RaisedInner, <br> SunkenInner, <br> Raised, <br> Etched, <br> Bump, <br> Sunken, <br> Adjust, and <br> Flat. <br> The default value is set to 'Sunken'. |
| BorderColor       | Specifies the color of the 2D border.                                                                                                                      |
| BorderSides      | Indicates the border sides of the panel. The options included are as follows: <br> Left, <br> Top, <br> Right, <br> Bottom, <br> Middle, and <br> All.                                                       |
| BorderStyle      | Indicates whether the edit control should have a border. The options included are given below: <br> FixedSingle, <br> Fixed3D, and <br> None.                                                             |

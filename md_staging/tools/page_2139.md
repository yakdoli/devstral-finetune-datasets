### Layout Group Editor

The Layout Group Editor is a feature that allows users to configure layout groups in their Windows Forms applications. The following screenshot illustrates the editor interface.

#### Figure 1426: Getting ImageStreamer Items in Property Window
![Layout Group Editor Screenshot](https://example.com/screenshot.png)

The image shows the `LayoutGroup Collection Editor` window with the following sections:

1. **Members List**
   - Lists the current layout groups (`layoutGroup1` and `layoutGroup2`).
   - Provides options to move, add, or remove layout groups using the buttons below the list.

2. **Layout Group Properties**
   - Displays the properties for the selected layout group (`layoutGroup1`).
   - Key properties include:
     - **AutoScrollMargin**: `0, 0`
     - **AutoScrollMinSize**: `0, 0`
     - **AutoSize**: `False`
     - **AutoSizeMode**: `GrowOnly`
     - **Dock**: `None`
     - **Location**: `100, 125`
     - **Margin**: `3, 3, 3, 3`
     - **MaximumSize**: `0, 0`
     - **MinimumSize**: `0, 0`
     - **Padding**: `0, 0, 0, 0`
     - **Size**: `260, 418`
     - **Items**: (Collection)
       - Shows the items associated with the layout group.
       - **SetItem**: `True`

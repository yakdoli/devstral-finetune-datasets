## Overview
- Demonstrates how to remove items from a drop-down list.
- Explains the concept of setting the button mode to either normal or toggle in a split button.

## Content

### Removing Item from Drop-Down List
You can also remove the added items if required. The following code illustrates how to remove items from a drop-down list:

#### C#
```csharp
this.splitButton1.DropDownItems.RemoveAt(1);
```

#### VB
```vb
Me.splitButton1.DropDownItems.RemoveAt(1);
```

### 3.16.2.3 Button Mode
This feature enables you to set the button in normal or toggle mode.

- **Normal Mode** - Execute normal button command
- **Toggle Mode** - Execute toggle mode click event

#### Setting Button Mode
You can set the button mode using the `ButtonMode` property.

The following code illustrates how to set the button in normal mode:

```csharp
this.splitButton1.ButtonMode = Syncfusion.Windows.Forms.Tools.ButtonMode.Normal;
```

## API Reference
- **ButtonMode**: A property that controls whether the button operates in normal or toggle mode.
- **DropDownItems**: A collection of items in the drop-down list that can be manipulated programmatically.

### Code Examples
- **Removing an Item**: Demonstrates removing an item at a specific index from the `DropDownItems` collection.
- **Setting Button Mode**: Shows how to configure the `ButtonMode` property to either `Normal` or `Toggle` mode.

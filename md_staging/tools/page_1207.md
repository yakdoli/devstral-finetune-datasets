## 3.7.4.4 Visual Styles for the Popup Menu

Visual Style for a popupmenu can be controlled by using `PopupMenu.ParentBarItem.Style` property.

```csharp
this.popupMenu1.ParentBarItem.Style =
    Syncfusion.Windows.Forms.VisualStyle.Office2007Outlook;
```

```vb
Me.popupMenu1.ParentBarItem.Style =
    Syncfusion.Windows.Forms.VisualStyle.Office2007Outlook
```

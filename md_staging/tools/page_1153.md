# Essential Tools for Windows Forms

## Overview
- This section explains how to set up and manage shortcut keys and shortcut text for bar items in Windows Forms.
- It provides a visual reference for the property grid and demonstrates how to configure shortcuts programmatically in both `C#` and `VB.NET`.

## Content
### Configuring Shortcuts for Bar Items

#### Property Grid

The figure below shows a property grid displaying the `Shortcut` and `ShortcutText` options for a `barItem`.

**Figure 770**: Property Grid Displaying Shortcut with options and ShortcutText

**Code Examples**

```csharp
// C#
this.barItem4.Shortcut = System.Windows.Forms.Shortcut.CtrlX;
this.dropDownBarItem2.ShortcutText = "Cut";
```

```vb
' VB.NET
Me.barItem4.Shortcut = System.Windows.Forms.Shortcut.CtrlX
this.dropDownBarItem2.ShortcutText = "Cut"
```

## Essential Tools for Windows Forms

### Overview
- Explains how to assign a SuperAccelerator to a RibbonControlAdv in both the form designer and through code.
- Demonstrates setting accelerator keys for components and child items.

### Content

#### Assigning SuperAccelerator in the form designer

The following screenshot shows how to assign a SuperAccelerator using the form designer.

**Figure 1136: Assigning SuperAccelerator in the form designer**

#### Through Code
Using the following code, the SuperAccelerator can be assigned to the RibbonControlAdv. Once SuperAccelerator is assigned, its child items can be assigned with SuperAccelerator key values.

---

### Code Sample

```csharp
SuperAccelerator superAccelerator1 = new SuperAccelerator();
this.ribbonControlAdv1.SuperAccelerator = this.superAccelerator1;
```

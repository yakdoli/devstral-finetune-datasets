### Summary

- XPTaskPane can be added and configured within a Windows Forms application.
- Pages can be added through either the **Smart Tag** or the **Context Menu** of the **Header**.

## Code Examples

This section uses C# code to demonstrate adding a new page to the XPTaskPane programmatically:

```csharp
public void AddPageToTaskPane()
{
    // Assuming taskPane is a reference to the XPTaskPane control
    XPPage page = new XPPage();
    page.HeaderText = "New Page";
    taskPane.Pages.Add(page);
}
```

## API Reference

### Methods

- **AddPage()**: Adds a new page to the TaskPane.
- **RemovePage()**: Removes a specified page from the TaskPane.

### Parameters

| Name       | Type       | Description                  |
|------------|------------|------------------------------|
| page       | XPPage     | The new page to be added.    |
| pageNumber | int        | The index of the page to remove. |

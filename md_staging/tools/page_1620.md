# Essential Tools for Windows Forms

## Overview
- The format of the data in the panel can be set using the StatusBarAdvPanel property.
- The PanelType property indicates the type of the panel.
- PanelType is used to display predefined text representing key states, date, time, or culture information.

## Content

The format of the data in the panel can be set using the property given below.

### Table: StatusBarAdvPanel Property and Description

| StatusBarAdvPanel Property | Description |
|----------------------------|-------------|
| PanelType                  | Indicates the type of the panel. |

The `PanelType` property can be used to display predefined text representing key states, date/time, or culture information.

### Note
Users can also specify their own text to be displayed in the control by setting the `PanelType` property to `Custom`. The text to be displayed is set using the `Text` property of the `StatusBarAdvPanel`.

#### Code Examples

[C#]

```csharp
this.statusBarAdvPanel1.PanelType = 
    Syncfusion.Windows.Forms.Tools.StatusBarAdvPanelType.NumLockState;
```

[VB.NET]

```vb
Me.statusBarAdvPanel1.PanelType = 
    Syncfusion.Windows.Forms.Tools.StatusBarAdvPanelType.NumLockState
```

### Example of Panel Types

- **NumLock OFF**: NumLockState
- **NumLock ON**: Custom
- **CapsLock OFF**: CapsLockState
- **Insert**: InsertKeyState
- **ScrollLock OFF**: ScrollLockState
- **English (United States)**: CurrentCulture
- **Wednesday, February 27, 2008**: LongDate
- **2/27/2008**: ShortDate
- **3:34:37 AM**: LongTime
- **3:35 AM**: ShortTime

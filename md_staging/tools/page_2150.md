# Essential Tools for Windows Forms

## Overview
- Explore tools to add nodes in a design-time environment.
- Modify node properties using the property grid.
- Create a custom label for the node by editing its text property.

## Content

### Design-Time Actions

#### Step-by-Step Instructions
1. Open the Windows Forms application in the editor.
2. Navigate to the design-time environment.
3. Locate the context menu or design-time tools.
4. Select the appropriate option to access node-related actions.

##### Figure 1431: "Add Node" Design-Time Verb
![](ToolsWinformsAddNodeDesignTimeVerb.png)

5. Click "Add Node". This will add a new top-level node.
6. The node's properties will be displayed in the property grid to the right. Specify a custom label for the node by changing its `text` property as shown in the below image.

### Customizing Node Properties
- After adding a node, the property grid on the right will display editable properties.
- To modify the node's label, locate and edit the `text` property in the property grid.

#### Example
- After clicking "Add Node," change the `text` property to your desired label.

## API Reference
- `NodesEditor`
  - Methods and properties related to node editing.
  - `AddNode()`: Adds a new top-level node.
  - `AddChildNode()`: Adds a child node under a parent node.

## Code Examples
```csharp
// Example: Adding a node and customizing its label
using Syncfusion.Windows.Forms.Tools;

public void AddCustomNode()
{
    // Add a new top-level node
    var node = NodesEditor.AddNode();

    // Customize the node's label
    node.Text = "Custom Node Label";
}
```

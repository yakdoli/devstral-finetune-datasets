### Reordering XPTaskPage

#### 3.22.2.4.1.1 Reordering XPTaskPage

When the end user adds a page to the XPTaskPane control, the order of the page is decided as the page is added. They can be reordered using any one of the below methods in the designer.

- Through XPTaskPage Collection Editor.

## API Reference

- **XPTaskPage**: A control used to display and manage different task pages.
- **XPTaskPage.FontStyle**: Property to customize the font style of the XPTaskPage control.
- **XPTaskPage.Collection Editor**: Feature to reorder pages within the XPTaskPane control.

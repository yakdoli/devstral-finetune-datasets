## Overview

- Describes how to edit and customize Quick Access Toolbar items.
- Explores design-time and runtime customization options for toolbar controls.
- Provides sample code for programmatically adding items to the Quick Access Toolbar.

## Content

### Quick Access Toolbar Customization

**Figure 1198: Editing Quick Access Toolbar Items**

The provided figure illustrates the dialog for customizing the Quick Access Toolbar. This dialog allows you to choose commands from available options and add or remove them from the toolbar.

#### Notes on Customization

- **Design-Time Customization**:
  - **Note**: You can also add items to the Quick Access Toolbar by enabling the `UseInQuickAccessMenu On RibbonControl1` extended property for any control in the designer.
  
- **Runtime Customization**:
  - **Note**: Ribbon provides an option to edit the items at run time also. See [**Runtime Customization**](https://docs.syncfusion.com/windowsforms/latest/ribbon/runtime-customization) for more details.

# Tools for Windows Forms

## Overview
- Configures text display properties for a ToolStripSplitButton.
- Sets font style, forecolor, text alignment, text direction, and text-image relationship.

## Content

### Text
- **Description**: Sets the Text for the ToolStripSplitButton.
- **Display Condition**: This text will be displayed only if the `DisplayStyle` is `Text` or `ImageAndText`.

### Font
- **Description**: Sets the font style for the display text.

### ForeColor
- **Description**: Sets the fore color for the display text.

### TextAlign
- **Description**: Specifies the alignment of the text in the item.
- **Options**:
  - TopLeft
  - TopCenter
  - TopRight
  - MiddleLeft
  - MiddleCenter
  - MiddleRight
  - BottomLeft
  - BottomCenter
  - BottomRight

### TextDirection
- **Description**: Specifies the direction of drawing the text.
- **Options**:
  - Horizontal: Text is placed horizontally.
  - Vertical90: Text is placed vertically.
  - Vertical270: Text is placed vertically at 270 degrees.

### TextImageRelation
- **Description**: Specifies the relative location of the image to the text on the item.
- **Options**:
  - Overlay: Image and text share the same space in the control.
  - ImageAboveText: Image will be placed above the text.
  - TextAboveImage: Text will be placed above the image.
  - ImageBeforeText: Image will be placed before the text.
  - TextBeforeImage: Text will be placed before the image.

### API Reference

#### Properties
- **Text**: string
  - Sets the text for the ToolStripSplitButton.
  - Only displayed if `DisplayStyle` is `Text` or `ImageAndText`.

- **Font**: Font
  - Sets the font style for the display text.

- **ForeColor**: Color
  - Sets the foreground color for the display text.

- **TextAlign**: ContentAlignment
  - Specifies the alignment of text in the control.

- **TextDirection**: TextRenderingDirection
  - Specifies the direction of text drawing.

- **TextImageRelation**: TextImageRelation
  - Specifies the relative placement of the image to the text.

### Code Examples

```csharp
// Example of configuring a ToolStripSplitButton
ToolStripSplitButton button = new ToolStripSplitButton();
button.Text = "Save";
button.Font = new Font("Arial", 10, FontStyle.Bold);
button.ForeColor = Color.Blue;
button.TextAlign = ContentAlignment.MiddleLeft;
button.TextDirection = TextRenderingDirection.Horizontal;
button.TextImageRelation = TextImageRelation.Overlay;
```

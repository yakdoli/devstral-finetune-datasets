# Essential Tools for Windows Forms

## Overview
- Describes the StatusBarAdv control with a focus on its Sizing Grip feature.
- Explains how to enable the visibility of the sizing grip using the `SizingGrip` property.
- Demonstrates the implementation in C# and VB.NET.
- Introduces the AutoSize settings and behavior settings of StatusBarAdv.

## Content

### Sizing Grip

A sizing grip can be displayed for the StatusBarAdv control using the property given below.

| StatusBarAdv Property | Description                  |
|-----------------------|------------------------------|
| SizingGrip           | Indicates if the sizing grip is visible. |

**[C#]**
```csharp
this.statusBarAdv1.SizingGrip = true;
```

**[VB.NET]**
```vb
Me.statusBarAdv1.SizingGrip = True
```

The `SizingGrip` property, when set to `True`, will display a grip at the bottom right of the control as shown in the below image.

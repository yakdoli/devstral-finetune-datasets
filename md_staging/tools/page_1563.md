## ToolTip

### Overview
- Shows how ToolTip can be displayed for the SplashPanel control when the mouse hovers over it.
- Explains the usage of the `toolTip1` property to define the ToolTip.
- Provides both design-time and runtime examples in C# and VB.NET.

### Content

#### ToolTip Display for SplashPanel

ToolTip can be displayed for the SplashPanel when the mouse is moved over the control. This can be done using the following property.

| SplashPanel Property | Description                                                                                               |
|----------------------|-----------------------------------------------------------------------------------------------------------|
| ToolTip on toolTip1 | Determines the ToolTip shown when the mouse hovers over the control.<br><br>The ToolTip on toolTip1 property is an extender property, i.e., it will appear in the Property grid only when you add a tooltip control onto the form. |

#### Setting ToolTip Through Code

ToolTip can also be set through the code provided below.

##### C#
```csharp
this.toolTip1.SetToolTip(this.splashPanel1, "Splash Panel Tooltip");
```

##### VB.NET
```vb
Me.toolTip1.SetToolTip(this.splashPanel1, "Splash Panel Tooltip")
```

# Essential Tools for Windows Forms

## Overview
- Images for individual nodes can be specified in `treeNodeAdv.ExpandImageIndex` / `treeNodeAdv.CollapseImageIndex` properties. Setting these properties will override the expand / collapse image settings that are applied for the control.

## Content

### TreeViewAdv Properties and Descriptions
The following table lists the properties of the TreeViewAdv control and their descriptions:

| TreeViewAdv Property         | Description                                                                 |
|------------------------------|-----------------------------------------------------------------------------|
| Expanded                     | Indicates if the node is expanded.                                         |
| DefaultCollapseImageIndex    | It is the default imageindex when a tree node is collapsed.              |
| DefaultExpandImageIndex      | It is the default imageindex when a tree node is expanded.               |
| NodeStateImageList           | Indicates the imagelist with images that are displayed instead of expand / collapse buttons. |

### TreeNodeAdv Properties for Node-Level Settings
These properties can be set at the node level using the following properties:

| TreeNodeAdv Property         | Description                                                                |
|------------------------------|----------------------------------------------------------------------------|
| CollapseImageIndex           | It is the imageindex for the collapse button.                             |
| ExpandImageIndex             | It is the imageindex for the expand button.                               |

## API Reference

### Properties
- **Expanded**: Indicates if the node is expanded.
- **DefaultCollapseImageIndex**: Default imageindex when a tree node is collapsed.
- **DefaultExpandImageIndex**: Default imageindex when a tree node is expanded.
- **NodeStateImageList**: Indicates the imagelist with images that are displayed instead of expand / collapse buttons.
- **CollapseImageIndex**: Imageindex for the collapse button.
- **ExpandImageIndex**: Imageindex for the expand button.

## Code Examples
```csharp
// Example of setting expand/collapse image indices for a TreeNodeAdv
treeNodeAdv.DefaultExpandImageIndex = 1;
treeNodeAdv.DefaultCollapseImageIndex = 2;

// Setting node-specific image indices
TreeNodeAdv node0 = treeNodeAdv.Nodes.Add("Node0");
node0.CollapseImageIndex = 3;
node0.ExpandImageIndex = 4;
```

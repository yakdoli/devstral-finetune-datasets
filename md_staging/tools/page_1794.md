# Essential Tools for Windows Forms

## Overview
- Demonstrates the functionality of `ToolStripEx` in `Office12Mode` and `ManagerRenderMode`.
- Explains the `OfficeColorScheme` property and its impact on the control's appearance.
- Provides examples in both C# and VB.NET for setting the `OfficeColorScheme`.

## Content

### Office2007 Mode

**Disabling the Office12Mode property will automatically give the Ribbon control, Office2007 look and feel.**

#### Figure 1185: `ToolStripEx` in Office12Mode and ManagerRenderMode
![](image.png)

#### OfficeColorScheme Property
Property | Description
--- | ---
`OfficeColorScheme` | Sets the office color schemes for the control. Blue, Black, and Silver schemes can be applied.

**Note:** This setting will overwrite the `Panel.OfficeColorScheme` property.

### Code Examples

#### C#
```csharp
this.toolStripEx1.OfficeColorScheme = 
    Syncfusion.Windows.Forms.Tools.ToolStripEx.ColorScheme.Silver;
```

#### VB.NET
```vb
Me.toolStripEx1.OfficeColorScheme = 
    Syncfusion.Windows.Forms.Tools.ToolStripEx.ColorScheme.Silver
```

## Interactive Features

### Summary
- The `OfficeColorScheme` property allows users to customize the appearance of the toolstrip control to match specific office color schemes.
- The examples provided show how to implement this feature in both C# and VB.NET.

## API Reference

### Properties
- **`OfficeColorScheme`**  
  Type: `ToolStripEx.ColorScheme`  
  Description: Sets the color scheme for the toolstrip control.  
  Values: `Blue`, `Black`, `Silver`

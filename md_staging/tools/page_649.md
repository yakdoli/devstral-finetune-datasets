# Essential Tools for Windows Forms

## Overview
- Describes how to set the border style for the GradientPanel control.
- Highlights properties that configure 2D and 3D borders for the GradientPanel.
- Explains options for `BorderStyle`, `Border3DStyle`, `BorderSingle`, and `BorderColor`.

## Content

The GradientPanel can have 2D and 3D borders. The properties which set the border style are as follows.

| GradientPanel Property | Description |
|-------------------------|-------------|
| `BorderStyle`          | Sets the 2D or 3D border for the GradientPanel. The options are, <br> FixedSingle and <br> Fixed3D. |
| `Border3DStyle`       | Sets the style of the 3D border. The options are, <br> RaisedOuter, <br> RaisedInner, <br> SunkenOuter, <br> SunkenInner, <br> Raised, <br> Etched, <br> Bump, <br> Sunken, <br> Adjust and <br> Flat. |
| `BorderSingle`        | Indicates the 2D border style. The options are, <br> Solid, <br> Dotted, <br> Dashed, <br> Inset, <br> Outset and <br> None. |

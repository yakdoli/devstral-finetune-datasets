# Essential Tools for Windows Forms

## Overview
- Explores key controls and tools used in Windows Forms development.
- Focuses on the ToolstripEx control with various properties for displaying text, images, alignment, and visibility.
- Details tooltip settings and RTL (right-to-left) support for enhanced user interface handling.

## Content

### Properties of ToolstripEx Items

| Property       | Description                                                                                           |
|----------------|-------------------------------------------------------------------------------------------------------|
|                | • Text - Displays only text, <br> • Image - Displays only image, <br> • ImageAndText - Displays image and text. |
| Enabled        | Specifies whether the item is enabled.                                                                |
| Visible        | Specifies whether the item is visible.                                                               |
| Alignment      | Sets the alignment of the item within the ToolstripEx. They can be set to beginning (Left) or end (Right) of the ToolstripEx control. |
| AutoSize       | Specifies whether the item should size itself based on its image and text. |

### ToolTip Settings

The ToolstripLabel can show tooltips during runtime, using the below properties.

| Property       | Description                                                                                           |
|----------------|-------------------------------------------------------------------------------------------------------|
| AutoToolTip    | When set to true, will display the text set in the Text property as the item's tooltip. <br> When set to false, will display the text set in the ToolTipText property. |
| ToolTipText    | Sets the text for the tooltip when AutoToolTip is set to false.                                    |

### RTL Support

| Property                  | Description                                                                                                  |
|---------------------------|--------------------------------------------------------------------------------------------------------------|
| RightToLeft               | Indicates whether the item should draw right to left for RTL languages.                                     |
| RightToLeftAutoMirrorImage | Specifies whether the image should mirror when RightToLeft is enabled for the item.                        |

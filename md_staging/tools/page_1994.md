# Essential Tools for Windows Forms

## Overview
- TabPrimitivesHost provides additional features like adding Images, ToolTips, and enabling the Visible property for each TabPrimitive.
- The TabPrimitives feature allows for enhanced customization of tab navigation in Windows Forms applications.

## Content

Apart from doing the defined task of Navigation, TabPrimitivesHost comes with options for adding Images, ToolTips and enabling the **Visible** property for each TabPrimitive.

![Figure 1355: TabPrimitives Features](https://i.imgur.com/a/9No.png)

* **Figure 1355:** TabPrimitives Features

**Note:** *You can set the other properties for adding Images and ToolTips for the TabPrimitives using the TabPrimitives Collection Editor.*

- **ToolTips** feature is available for TabPrimitives.

### 3.17.5.1 Creating TabPrimitives

#### Creating TabPrimitives Through Designer

To create Tab Primitives through the designer, follow the steps given below:

1. After adding a **TabControlAdv** with a set of **TabPages** in it, select the **TabPrimitivesHost.TabPrimitives** property in the Properties window.

## API Reference (if applicable)
- **Property:** TabPrimitivesHost.TabPrimitives
  - Type: `Syncfusion.Windows.Forms.Tools.TabPrimitives`
  - Description: Enables customization of TabPrimitives, allowing the addition of Images, ToolTips, and enabling the Visible property for each TabPrimitive.
  - Default: `null`

- For more information on the **TabControlAdv** control, refer to the documentation on Tab Controls.

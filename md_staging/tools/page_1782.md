# Essential Tools for Windows Forms

## Overview
- Describes key properties and settings for text-based controls.
- Focuses on `HideSelection`, `MaxLength`, `ReadOnly`, and `ShortCutsEnabled` properties for textboxes.
- Enumerates autocomplete features including `AutoCompleteCustomSource`, `AutoCompleteSource`, and `AutoCompleteMode` for combo boxes.

## Content

### General Properties
| Property          | Description                                                                 |
|-------------------|-----------------------------------------------------------------------------|
| HideSelection     | Indicates whether the selection should be hidden when the control loses focus. |
| MaxLength         | Maximum number of characters that can be entered into the control.            |
| ReadOnly          | Indicates whether the text in the textbox is read-only.                      |
| ShortCutsEnabled  | Specifies whether the keyboard shortcut can be specified for this textbox item. |
| ShortCut on Form1 | Specifies the Keyboard shortcut to be used at run time to access this combobox. |

### AutoComplete Settings
| Property                  | Description                                                                                                                                           |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| AutoCompleteCustomSource  | Represents the custom source of string collection for the autocomplete feature, when `AutoCompleteSource` property is set to `CustomSource`.          |
| AutoCompleteSource        | Represents the source of strings used for autocompletion. The sources can be, <br><ul><li>FileSystem,</li><li>AllSystemSources, (Default)</li><li>AllUrl,</li><li>CustomSource,</li><li>FileSystemDirectories,</li><li>HistoryList,</li><li>ListItems,</li><li>RecentlyUsedList,</li><li>None.</li></ul> |
| AutoCompleteMode          | Indicates text completion behavior of the combo box. The modes are, <br><ul><li>Suggest - Displays the drop-down list associated with the `EditControl`. This dropdown list is populated with one or more items.</li></ul> |

## API Reference
- Properties such as `HideSelection`, `MaxLength`, `ReadOnly`, `ShortCutsEnabled`, `AutoCompleteCustomSource`, `AutoCompleteSource`, and `AutoCompleteMode` are documented to illustrate their functionality and usage in WinForms.

## Code Examples
```csharp
// Example setting AutoCompleteMode in C#
comboBox1.AutoCompleteMode = AutoCompleteMode.Suggest;
```

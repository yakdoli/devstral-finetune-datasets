# Essential Tools for Windows Forms

## Overview
- Attach Office 2007 style scrollbars to a TreeViewControl.
- Guide on attaching scrollbars to a control in a Windows form.

## Content

### Figure 1236: Office2007Style ScrollBars Attached to TreeViewControl

![Office2007Style ScrollBars Attached to TreeViewControl](image.png)

#### Attaching Scrollbar to a Control

To the Windows form, add a control, which should be attached with the Office2007Style scrollbars. Select the control in the ScrollersFrame.AttachedTo property.

---

### API Reference

- **Property**: ScrollersFrame.AttachedTo
  - Attach scrollbars to a specified control.

### Code Examples

#### C# Example

```csharp
// Assuming the control is named 'myControl'
ScrollersFrame scrollFrame = new ScrollersFrame();
scrollFrame.AttachedTo = myControl;
```

## Essential Tools for Windows Forms

| NodeStatImageList | Indicates the imagelist with images that<br/>are displayed instead of expand /<br/>collapse button. |
|--------------------|---------------------------------------------------------------------------------------------------------------- |
| OpenImgIndex      | It is the StatImageList index value of<br/>the image that is displayed, when a tree<br/>node is expanded.     |
| StateImageList    | This indicates the imagelist that holds<br/>the images to be drawn based on the<br/>state of the Node.       |

**Note:** *The above properties can also be set for individual nodes.*

### TreeNodeAdv Property

| TreeNodeAdv Property        | Description                                                                                                              |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------|
| ClosedImageIndex            | It is the imageindex in StatImageList where the node is collapsed.                                                        |
| NoChildrenImgIndex          | It is the imageindex indicating the image in the StatImageList where the node has no children.                            |
| OpenImgIndex                | It is the imageindex in StatImageList where the node is expanded.                                                        |
| LeftStatImagePadding         | It is the space provided between, the LeftStatImage of the node and node.                                               |
| RightStatImagePadding      | It is the space provided between, the RightStatImage of the node and node.                                               |


## Customizing the Image Size

The ImageSize property let you enhance the image size for a TreeNodeAdv. By default, the TreeViewAdv control displays the image size depending on the size of the image that is set in the image list.

**C#**
```csharp
this.leftImageList.ImageSize = new System.Drawing.Size(16, 16);
this.rightImageList.ImageSize = new System.Drawing.Size(16, 16);
this.stateImageList.ImageSize = new System.Drawing.Size(16, 16);
```

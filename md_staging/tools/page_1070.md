# Essential Tools for Windows Forms

## Overview
- Discusses methods for managing components in a layout list.
- Explains how to add or remove controls programmatically using `SetParticipateInLayout`.
- Highlights the ability to rearrange controls in a `GridLayout` at design time.

## Content

### Methods and Description

| Methods                     | Description                                                                                                                                                   |
|-----------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| GetParticipateInLayout      | Indicates whether the component is in the layout list.                                                                                                       |
| SetParticipateInLayout      | Adds or removes the specified control from the layout list.                                                                                                   |

#### Adding or Removing Controls from the GridLayout List
The following code demonstrates how to add or remove a control from the `GridLayout` list programmatically.

```csharp
this.gridLayout1.SetParticipateInLayout(this.button1, false);
```

```vb
Me.gridLayout1.SetParticipateInLayout(Me.button1, False)
```

# Essential Tools for Windows Forms

## Overview
- Demonstrates the use of the `SegmentWidth` property in `ProgressBarAdv`.
- Explains how to set the `SegmentWidth` property in both C# and VB.NET.
- Provides information on color settings for the foreground of `ProgressBarAdv`.

## Content

### `ProgressBarAdv` Property: SegmentWidth

#### Description
Specifies the width of the segments.

| ProgressBarAdv Property | Description |
|-------------------------|-------------|
| SegmentWidth            | Specifies the width of the segments. |

#### Code Examples

**[C#]**
```csharp
this.progressBarAdv1.SegmentWidth = 20;
```

**[VB.NET]**
```vb
Me.progressBarAdv1.SegmentWidth = 20
```

### 3.11.1.3.3.2 Foreground Color Settings

This section illustrates the color settings that can be applied to the foreground of the `ProgressBarAdv`.

#### Font and Fore Color Settings

The font color and the fore color can be set using the properties given below.

**Font Color and Fore Color Properties**

| ProgressBarAdv Property | Description |
|-------------------------|-------------|
| ForeColor               | Specifies the color used to draw the foreground in segment mode and constant mode. |

### Cross References

- **See Also:**
  - [Background Settings](#background-settings)

### API Reference

- **Properties:**
  - `SegmentWidth`: Sets the width of the segments.
  - `ForeColor`: Specifies the color used to draw the foreground.

### Code Examples

#### Setting `SegmentWidth` and `ForeColor`

**[C#]**
```csharp
this.progressBarAdv1.SegmentWidth = 20;
this.progressBarAdv1.ForeColor = Color.Black;
```

**[VB.NET]**
```vb
 Me.progressBarAdv1.SegmentWidth = 20
 Me.progressBarAdv1.ForeColor = Color.Black
```

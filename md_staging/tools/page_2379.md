# Essential Tools for Windows Forms

## Overview
- Describes the process of reordering pages using the XP_TASK Page Collection Editor.
- Highlights how to move pages to the beginning or end of the collection.

## Content

### XP_TASK Page Collection Editor

#### Reordering Pages

The XP_TASK Page Collection Editor allows users to manage and reorder the pages in a collection. The interface includes:

- **Members** section:
  - Lists the pages in the collection with their respective indices (e.g., `xpTaskPage1`, `xpTaskPage2`, `xpTaskPage4`, `xpTaskPage3`).
  - Provides options to add or remove pages from the collection.

- **Appearance Properties** section:
  - Displays properties such as `BackColor`, `BorderColor`, and `BorderStyle`, with values like `231, 243, 255` and `99, 146, 206`.

Figure 1536 illustrates the reordering process:

**Figure 1536: Reordering Pages Using XP_TASKPage Collection Editor**

To reorder the pages:
1. Select a page in the designer.
2. Choose the 'Bring To Front' or 'Send To Back' verb.
   - 'Bring To Front' moves the page to the beginning of the collection.
   - 'Send To Back' moves the page to the end of the collection.

- **Buttons**:
  - **OK**: Applies the changes and closes the editor.
  - **Cancel**: Cancels any changes and closes the editor.

### Steps to Reorder Pages

Select a page in the designer and choose the 'Bring To Front' or 'Send To Back' verb, which will move the page to the beginning of the collection or to the end of the collection, respectively.

## Terminology and References

- **XP_TASKPage Collection Editor**: A tool for managing and reordering pages within a Windows Forms application.
- **Bring To Front**: A verb to move a page to the beginning of the collection.
- **Send To Back**: A verb to move a page to the end of the collection.

# Essential Tools for Windows Forms

## Overview
- Explains how to configure the appearance of a radial slider control using different styles.
- Demonstrates the default slider style with an outer circle.
- Introduces the Frame style to customize the slider's background.

## Content

### Default Slider with Outer Circle

![Default Slider with outer circle](image_link_placeholder)
*Figure 1301: Default Slider with outer circle*

This figure illustrates a radial slider with an outer circle as its default style.

#### Code sample:

```csharp
this.radialSlider1.SliderStyle = Syncfusion.Windows.Forms.Tools.SliderStyles.Default;
```

```vb
Me.radialSlider1.SliderStyle = Syncfusion.Windows.Forms.Tools.SliderStyles.Default
```

## API Reference

The `SliderStyles` class provides methods to set different styles for a slider control. One such style defined here is `Default`, as seen in the code examples provided.

## Code Examples

The examples above demonstrate setting the `SliderStyle` property to the `Default` value for a `radialSlider1` control in both C# and VB.NET.

### Essential Controls for Windows Forms

- **The Page Title**  
  This is a label control that displays the title of the current wizard page. It is selectable in the designer and can be accessed directly or through the `WizardControl.Title` property in code.

- **The Description label**  
  This label control, a child of the Banner Panel, shows the title of the current wizard page. It is selectable in the designer and can be accessed directly or through the `WizardControl.Description` property in code.

- **The Banner control**  
  This picture box control, a child of the Banner Panel, allows you to display a picture in the top-right corner. The picture remains constant across all pages. It is selectable in the designer and can be accessed directly or through the `WizardControl.Banner` property in code.

- **The WizardContainer**  
  This is the control within which the selected WizardControl Page will be displayed. It is selectable in the designer and can be accessed directly or through the `WizardControl.WizardPages` property in code.

- **The Browsing Buttons**  
  These buttons allow you to navigate through the different wizard pages.

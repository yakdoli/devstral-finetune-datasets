# Essential Tools for Windows Forms

## Overview
- Explains how to configure the appearance of the `PageTitle` and `Description` text in a `Wizard` control.
- Demonstration of editing font style and foreground color for improved UI customization.
- Focus on `Font` and `ForeColor` properties for description text modification.

#### Description Text

The appearance of the description text for a wizard control can be edited using the description label properties.

| Property   | Description                                                 |
|------------|-------------------------------------------------------------|
| Font       | Sets the font style for the Page description in the Wizard Control. |
| ForeColor  | Sets the forecolor for the Page description in the Wizard Control. |

### API Reference
- **Namespace**: `Syncfusion.Windows.Forms.Tools`
- **Class**: `WizardControl`
- **Properties**:
  - `Font`: Type `System.Drawing.Font`. Used to set the font style for description text.
  - `ForeColor`: Type `System.Drawing.Color`. Used to set the foreground color for description text.

### Code Examples
```csharp
// Setting Font for the Page description
wizardControl.DescriptionLabel.Font = new Font("Verdana", 9, FontStyle.Bold);

// Setting ForeColor for the Page description
wizardControl.DescriptionLabel.ForeColor = Color.DarkBlue;
```

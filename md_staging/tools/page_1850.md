# Essential Tools for Windows Forms

## Overview
- The page explains the Smart Tag Options for the StatusStripEx control in Syncfusion Windows Forms.
- Details include the Tasks window displayed upon clicking the Smart Tag, along with various options to add ToolStripStatus Items.
- Describes different categories of items such as Dock, StatusControl Items, and Notifications Items.

## Content

### Smart Tag Options
**3.12.2.3.2 Smart Tag Options**

#### Overview
- Clicking the Smart Tag of the StatusStripEx displays the Tasks window shown below. This window allows you to add ToolStripStatus Items.

#### Tasks Window
- **Dock** - Provides docking options for the StatusStripEx control.
- **StatusControl Items**
  - Add StatusLabel - Adds a status label item.
  - Add ProgressBar - Adds a ProgressBar item.
  - Add DropDownButton - Adds a dropdownbutton item.
  - Add SplitButton - Adds a split button item.
  - Add PanelItem - Adds a Panel item.
  - Add TrackBar Item - Adds a TrackBar item.
- **Notifications Items**
  - Add StatusStripButton - Adds a Button item.

### Key Features
- **Dock**
  - Provides docking options for the StatusStripEx control.
- **StatusControl Items**
  - Items such as StatusLabel, ProgressBar, DropDownButton, SplitButton, PanelItem, and TrackBarItem can be added.
- **Notifications Items**
  - Allows adding a Button item through the StatusStripButton option.

### API Reference
- **Properties**
  - **Dock** - Configures the docking options for the StatusStripEx control.
- **Methods**
  - **Add** - Methods for adding different types of control items to the StatusStripEx.

### Code Examples
- Example showcasing how to use the Smart Tag Options for StatusStripEx:
  ```csharp
  // Example of adding a StatusLabel
  StatusStripEx statusStripEx = new StatusStripEx();
  statusStripEx.Dock = DockStyle.Bottom;
  statusStripEx.Items.Add(new ToolStripStatusLabel("Status: Ready"));
  ```

## Overview

- Close buttons displayed for all TabItems in the TabControlAdv.  
- Option to restrict close buttons to Active TabPages only.  
- Offer for custom setup using TabPrimitives and properties like `ShowTabCloseButton` and `ShowCloseButtonForActiveTabOnly`.

## Content

### TabControlAdv Close Buttons

#### Displaying Close Buttons for All TabItems

Close Button on all the `TabItems` can be made visible by setting the **ShowTabCloseButton** property to `True`.

#### C#
```csharp
this.tabControlAdv1.ShowTabCloseButton = true;
```

#### VB
```vb
Me.tabControlAdv1.ShowTabCloseButton = True
```

**Figure 1377:** Close buttons displayed for all the Pages in the TabControlAdv

#### Restricting Close Buttons to Active TabPages

Also, the Tab Close Button can be restricted to be shown only for Active TabPages. This can be done by setting the **ShowCloseButtonForActiveTabOnly** property to `True`.

#### C#
```csharp
this.tabControlAdv1.ShowCloseButtonForActiveTabOnly = true;
```

#### VB
```vb
Me.tabControlAdv1.ShowCloseButtonForActiveTabOnly = True
```

**Figure 1378:** Close button displayed only for the Active Page in the TabControlAdv

![Close button displayed only for the Active Page in the TabControlAdv](image.png)

### Note: Close Button Configuration

**Close Button can be set for the whole TabControlAdv by using TabPrimitives.**

#### Additional Information
- Close buttons are configurable via the `ShowTabCloseButton` and `ShowCloseButtonForActiveTabOnly` properties.
- The custom `TabPrimitives` class can be utilized for advanced functionality and styling.

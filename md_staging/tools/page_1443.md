# Essential Tools for Windows Forms

## Overview

- This section explains how to integrate child controls inside the XPTaskBar Boxes using the Panel control.
- Describes the PreferredChildPanelHeight property and its usage in setting the height of child panels.
- Explains the concept of ToolTips for TaskBar items in the XPTaskBar Box and how to assign tooltips to items.

## Content

### 3.9.3.4.2.5 Integrating Child Controls to the XPTaskBar Box

To host multiple controls inside the XPTaskBar Boxes, we prefer the **Panel** control. We can set the panel's height using the **PreferredChildPanelHeight** property.

| XPTaskBar Box Property             | Description                                        |
|-------------------------------------|----------------------------------------------------|
| PreferredChildPanelHeight          | It sets the height of the panel hosted inside the XPTaskBar Boxes. |

#### Code Examples

##### [C#]

```csharp
this.xpTaskBarBox1.PreferredChildPanelHeight = 35;
```

##### [VB.NET]

```vb
Me.xpTaskBarBox1.PreferredChildPanelHeight = 35
```

### 3.9.3.4.2.6 XPTaskBar- ToolTips

ToolTips can be provided for the TaskBar Items of the XPTaskBar Box. The interesting part is that tooltips can also be assigned for the disabled TaskBar items.

The **ToolTipText** property of the XPTaskBar control can be used to set the text of the tooltip, while the tooltip can be displayed using the **ShowToolTip** property.

| XPTaskBar Box Property | Description                       |
|-------------------------|-----------------------------------|
| ToolTipText            | Gets / sets the text of the tooltip. |

## API Reference

### Properties

- **PreferredChildPanelHeight**
  - **Type**: integer
  - **Description**: Sets the height of the panel hosted inside the XPTaskBar Boxes.

- **ToolTipText**
  - **Type**: string
  - **Description**: Gets or sets the text of the tooltip.

## Code Examples (continued)

The code examples for setting the ToolTipText and displaying the tooltip are as follows:

```csharp
this.xpTaskBarBox1.ToolTipText = "This is a tooltip.";
this.xpTaskBarBox1.ShowToolTip(); // Displays the tooltip
```

## Essential Tools for Windows Forms

### Selecting Cards using Design Time Verbs

You can also browse through the different cards using the methods given below.

```csharp
Me.cardLayout1.SelectedCard = "Card1"
```

![Figure 669: Selecting Cards using Design Time Verbs](./image.png)

**Figure 669: Selecting Cards using Design Time Verbs**

### Methods for Browsing Cards

| Methods     | Description                             |
|-------------|-----------------------------------------|
| **First**   | Shows the first card.                  |
| **Next**    | Show the next card in the list.        |
| **Previous**| Show the previous card in the list.    |
| **Last**    | Show the last card in the list.        |

### Note:

The SmartTag feature (available only in Visual Studio 2005) can also be used to browse through the cards of the CardLayout.

### Through Code

Drag and drop the ComboBox and the Previous and Next Buttons for viewing the selected card. Use the `Previous()` and `Next()` methods of the CardLayout to see the CardLayout in action inside the Previous and Next Button Clicks.

### API Reference

Namespace: `Syncfusion.Windows.Forms.Tools`

#### Methods

- **Previous():** Navigates to the previous card in the list.
- **Next():** Navigates to the next card in the list.

#### Properties

- **SelectedCard:** Gets or sets the currently selected card.

#### Events

- **SelectedCardChanged:** Triggered when the selected card changes.

### Code Examples

#### C#
```csharp
// Example usage of Previous() and Next() methods
private void btnPrevious_Click(object sender, EventArgs e)
{
    cardLayout1.Previous();
}

private void btnNext_Click(object sender, EventArgs e)
{
    cardLayout1.Next();
}
```

## Summary

- **Design Time Verbs:** Allows selecting different cards in the CardLayout designer.
- **Methods for Navigation:** `First`, `Next`, `Previous`, and `Last` for browsing through the cards.
- **SmartTag Feature:** Available in VS 2005 for additional card browsing.
- **Programming Commands:** Use `Previous()` and `Next()` methods for runtime navigation.

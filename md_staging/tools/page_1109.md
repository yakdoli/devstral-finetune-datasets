## Content

### Adding Bar Items to a Category

1. **Selecting the Category:**
   - To add bar items into this category, **select the category** and click the **Modify...** button located in the top right corner of the **Commands** tab.
   - After clicking **Modify...**, select **Add** to invoke the **Add New BarItem** dialog.

2. **Setting Up BarItem Properties:**
   - In the **Add New BarItem** dialog, **select the type of BarItem** you would like to add from the list.
   - Name the BarItem using the **BarItem's text property**.
   - For more information on the different types of BarItems, refer to the **Bar Items** topic.

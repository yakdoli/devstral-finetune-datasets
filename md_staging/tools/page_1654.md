# Essential Tools for Windows Forms

## Content

### Overview

This section explains how to add and customize the Clock control in a Windows Forms application using both Visual Studio and code.

#### 3.11.6.1 Adding Clock to an Application

##### Adding Through Visual Studio

Follow the steps below to add the Clock control to an application through Visual Studio:

1. Create a new Windows Form application in Visual Studio.
2. Drag Clock from the Toolbox tab to the designer.

![Toolbox with Clock control selected](https://i.imgur.com/undefined.png)

3. Clock control is added.
4. Customize the properties of Clock in the Properties Window.

##### Adding through Code

Follow the steps below to add the Clock control to an application through code:

1. Include the Tools Windows namespace as given in the following code:

   ```csharp
   using Syncfusion.Windows.Forms.Tools;
   ```

   ```vb
   Imports Syncfusion.Windows.Forms.Tools
   ```

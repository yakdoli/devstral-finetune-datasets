# Essential Tools for Windows Forms

## Setting Minimum and Maximum values

### Overview
- **Minimum and Maximum Bounds**: Specifies the bounds of the RangeSlider control's channel extension.
- **Example**: Demonstrates setting the maximum bound to 10.

### Content

#### 3.14.2.11 Setting Minimum and Maximum values
These properties specify the Minimum and Maximum bounds of the RangeSlider control till which the channel extends.

In the following example, the maximum bound is set to 10.

```csharp
// C#
rangeSlider.Maximum = 20;
rangeSlider.Minimum = 0;
```

#### 3.14.2.12 Events
- **Scroll**: Occurs when the mouse moves any of the thumb over the channel.

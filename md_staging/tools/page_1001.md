# Essential Tools for Windows Forms

## Overview
- Offers customizable options including repeated words, special symbols, words with mixed casing, words with numbers, words in upper case, and custom dictionary support with a dictionary editor.
- All options are customizable at runtime.

## Content

### Adding SpellChecker to an Application

#### Procedure
- Drag and drop the `SpellChecker` from the tool box to the form.
- The `SpellChecker` will be added to the component tray of the form it is hosted in.

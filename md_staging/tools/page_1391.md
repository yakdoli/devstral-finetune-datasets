## Overview

- Explains the use of methods to reset various highlight properties to their default values.

### Methods to Reset Highlight Properties

The following table lists the methods related to resetting the above properties:

| Methods                     | Description                             |
|-----------------------------|-----------------------------------------|
| ResetSelectedHighlightItemColor | Resets the SelectedHighlightItemColor property to its default value. |
| ResetSelectedHighlightTextColor | Resets the SelectedHighlightTextColor property to its default value. |
| ResetSelectedItemColor      | Resets the SelectedItemColor property to its default value. |
| ResetSelectedTextColor      | Resets the SelectedTextColor property to its default value. |
| ResetSelectingItemColor     | Resets the SelectingItemColor property to its default value. |

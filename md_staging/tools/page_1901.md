# Essential Tools for Windows Forms

## Overview
- Describes the usage of the `SuperAccelerator` property on `ToolStripButton` items in a Windows Forms application.
- Demonstrates configuring and displaying accelerator strings for enhanced usability.

## Content

### Setting Up SuperAccelerator

#### Property Configuration

The `SuperAccelerator` property is used to set a shortcut string for a `ToolStripButton` item. Here’s how to configure it:

1. Open the Properties window for the `ToolStripButton` item in your Designer.
2. Locate the `SuperAccelerator` property in the grid.
3. Assign a string value to the `SuperAccelerator` property.

**Figure 1280: SuperAccelerator Extended Property on ToolStripButton Item**

This figure shows the Properties window with the `SuperAccelerator` property highlighted, where the string value is being set.

#### Illustrating SuperAccelerator

**Figure 1281: SuperAccelerator Illustrated**

This figure demonstrates how the configured `SuperAccelerator` string appears on the `ToolStripButton` at runtime. When the ALT key is pressed, the accelerator strings are displayed.

#### Runtime Acceleration

To activate the accelerator functionality at runtime:

1. Ensure the `SuperAccelerator` property is set for the desired items.
2. Press the ALT key. This will trigger the display of all configured accelerator strings below the corresponding items.

## API Reference

### SuperAccelerator Property

- **Type**: String
- **Default**: None
- **Description**: Specifies a shortcut string for a `ToolStripButton` item. This string can be used in conjunction with the ALT key to provide quick access to the item’s functionality.

## Code Examples

### Example: Setting SuperAccelerator

```csharp
// Assuming saveToolStripButton is your environment's ToolstripButton
saveToolStripButton.SuperAccelerator = "S";
```

### Example: Handling the ALT Key

```csharp
private void Form1_KeyDown(object sender, KeyEventArgs e)
{
    if (e.Alt)
    {
        // Logic to display accelerator strings
    }
}
```

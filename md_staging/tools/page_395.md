# Essential Tools for Windows Forms

## Overview
- Lists and describes essential tools and operations available for Windows Forms.
- Focuses on calculator functionalities such as basic arithmetic, square root calculations, memory recall, and percentage operations.
- Includes shortcuts like CTRL+R and CTRL+M for accessing memory functions.

## Content

### Calculator Operations

The following table outlines the functionality of various calculator operations in Windows Forms.

| **Symbol/Key** | **Description**                                                                                         | **Shortcut** |
|----------------|---------------------------------------------------------------------------------------------------------|--------------|
| 9             | Puts this number in the calculator display.                                                             | 9            |
| /             | Divides.                                                                                               | /            |
| sqrt          | Calculates the square root of the displayed number.                                                    | @            |
| MR            | Recalls the number stored in memory. The number remains in memory.                                    | CTRL+R       |
| 4             | Puts this number in the calculator display.                                                             | 4            |
| 5             | Puts this number in the calculator display.                                                             | 5            |
| 6             | Puts this number in the calculator display.                                                             | 6            |
| *             | Multiplies.                                                                                           | *            |
| %             | Displays the result of multiplication as a percentage. Enter on number, click, enter the second number, and then click %. For example, \( 50 * 25\% \) will display 12.5.<br><br>You can also perform operations with percentages. Enter one number, click the operator (\(+\), \(-\), \(*\), or \(/\)), enter the second number, click \(\%\), and then click \(=\).<br><br>**Example:**<br>\( 50 + 25\% \) (of 50) = 62.5.                                                                 | %            |
| MS            | Stores the displayed number in memory.                                                                 | CTRL+M       |

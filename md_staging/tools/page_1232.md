# Essential Tools for Windows Forms

## Overview
- This section discusses how to enable various Office 2007 visual styles and themes for Windows Forms applications.
- Enables Office 2007 and Outlook visual styles by setting the `Style` property.
- Contains examples in C# and VB.NET for configuring the styles.

## Content

### Visual Styles

#### Office 2007 Look
- **Enabled by setting Style to Office2007.**

![Figure 840: VS 2005 Visual Style](#)

- **Figure 841: Office 2007 Visual Style**

#### Office 2007 Outlook
- **Enabled by setting Style to Office2007Outlook.**

![Figure 842: Office 2007 Outlook Visual Style](#)

#### Code Examples
- **C#**
```csharp
this.mainFrameBarManager1.Style = VisualStyle.Office2003;
```
- **VB.NET**
```vb
Me.mainFrameBarManager1.Style = VisualStyle.Office2003
```

### Office 2007 Themes
- You can specify the color schemes for Office 2007 visual styles.
- Available themes include blue, silver, and black.

#### Note:
The property `ThemesEnabled` must be set to `true`, and the property `Style` must be set to either `Office2007` or `Office2007Outlook`.

## API Reference
- **Namespace**: Syncfusion.Windows.Forms.Tools
- **Class**: MainFrameBarManager
- **Properties**:
  - `Style`: To set the visual style (VisualStyle.Office2007 or VisualStyle.Office2007Outlook).
  - `ThemesEnabled`: To enable themes (boolean).

## Code Examples
The examples demonstrate how to set the `Style` property for the `MainFrameBarManager`.

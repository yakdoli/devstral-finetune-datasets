## Overview

- This section details various primitives used in Syncfusion WinForms to display and interact with images, checkboxes, option buttons, and custom controls. It also touches on how TreeNodeAdv custom images can be overlaid on default images in specific modes.

## Content

### Primitives for Node Display

- **LeftImagePrimitive**: Used to display an image to the left of the node.
- **RightImagePrimitive**: Used to display an image to the right of the node.
- **StateImagePrimitive**: Displays the state image of a node depending on its expanded or collapsed state.
- **CheckBoxPrimitive**: Displays a checkbox for the nodes. When clicked, the node will be checked.
- **OptionButtonPrimitive**: Displays an Option button for the nodes. When clicked, the corresponding node will be selected.
- **CustomControlPrimitives**: Displays a custom control for the nodes, allowing users to use the functionality of custom control primitives added to the nodes.

## Code Examples

### Example: Overlaying Images

```csharp
// Example code for overlaying custom images on default node images
using Syncfusion.Windows.Forms.Tools;

// Assuming a TreeNodeAdv instance
TreeNodeAdv node = new TreeNodeAdv("Sample Node");

// Set up the images
node.LeftImageKey = "DefaultLeftImage";
node.RightImageKey = "DefaultRightImage";
node.StateImageKey = "DefaultStateImage";

// Overlay custom images
node.OverlayImageKey = "CustomOverlayImage";
```

# Essential Tools for Windows Forms

## Overview
- This section demonstrates the use of pages options using a smart tag in Windows Forms.
- It highlights features like adding, removing, and navigating between pages in a wizard control.
- The functionality allows for context menu usage for page management.

## Content

### Using Context Menu
This section explains how to access and utilize the context menu associated with the wizard control for managing pages.

---

## API Reference

### WizardControl Tasks
The wizard control provides several methods for page management, such as:
- `AddPage()`: Adds a new page to the wizard.
- `RemovePage()`: Removes a specified page from the wizard.
- `PreviousPage()`: Navigates to the previous page.
- `NextPage()`: Navigates to the next page.

These methods are accessible via both the smart tag and context menu options.

---

## Code Examples

### C#
```csharp
// Adding a new page to the wizard control
wizardControl.AddPage();

// Removing a page from the wizard control
wizardControl.RemovePage();

// Navigating to the previous page
wizardControl.PreviousPage();

// Navigating to the next page
wizardControl.NextPage();
```

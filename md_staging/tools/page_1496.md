# Essential Tools for Windows Forms

## Foreground Styles

The following image shows the various *Foreground Styles* available for the ProgressBarAdv control:

![Figure 1017: Foreground Styles](https://example.com/figure_1017.png)

The **Waiting Gradient Style** of the ProgressBarAdv consists of the following properties that can be used to change the appearance and behavior of the style.

### ProgressBarAdv Property and Description

The table below lists the properties and their descriptions for customizing the **Waiting Gradient Style**:

| ProgressBarAdv Property         | Description                                                                                                                                                   |
|----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WaitingGradientEnabled**      | Determines if the waiting gradient is enabled. The ProgressStyle property should be set to 'WaitingGradient'.                                                   |
| **WaitingGradientInterval**     | Determines the interval of the waiting gradient.                                                                                                              |
| **WaitingGradientWidth**        | Determines the width of the waiting gradient.                                                                                                                 |
| **CustomWaitingRender**         | Indicates whether the waiting gradient will be replaced by another custom waiting render which is defaulted to segments. This property when set to 'True' will display |

## API Reference

### Properties

#### WaitingGradientEnabled

- **Description**: Determines if the waiting gradient is enabled.
- **Dependent Property**: The ProgressStyle property should be set to 'WaitingGradient'.

#### WaitingGradientInterval

- **Description**: Determines the interval of the waiting gradient.

#### WaitingGradientWidth

- **Description**: Determines the width of the waiting gradient.

#### CustomWaitingRender

- **Description**: Indicates whether the waiting gradient will be replaced by another custom waiting render which is defaulted to segments. This property when set to 'True' will display the custom render.

## Code Examples

You can customize the Waiting Gradient Style using the following C# code example:

```csharp
using Syncfusion.Windows.Forms.Tools.Controls;

// Set up ProgressBarAdv
ProgressBarAdv progressBar = new ProgressBarAdv();
progressBar.Location = new Point(50, 50);
progressBar.Size = new Size(200, 20);
progressBar.ProgressStyle = ProgressStyle.WaitingGradient;

// Configure Waiting Gradient properties
progressBar.WaitingGradientEnabled = true;
progressBar.WaitingGradientInterval = 500;
progressBar.WaitingGradientWidth = 10;
progressBar.CustomWaitingRender = true;
```

#### 3.5.2.1.4.1 How to draw or hide focus rectangle for the ButtonAdv control

**Description:**
ButtonControl shows some special features which the user interacts with the control. Those properties are discussed in this section.

| Properties             | Description                                                                 |
|------------------------|-----------------------------------------------------------------------------|
| KeepFocusRectangle     | Specifies whether rectangle will be drawn around the control when it is focused at runtime. |

##### C# Code Example
```csharp
this.buttonAdv1.KeepFocusRectangle = true;
```

##### VB.NET Code Example
```vb
Me.buttonAdv1.KeepFocusRectangle = True
```

# Essential Tools for Windows Forms

## Overview
- Describes the ToolstripRadioButton control and its properties.
- Explains how to configure the appearance and behavior of ToolstripRadioButton.

## Content

### 3.12.2.1.1.2.1.1.20 Radio Button

The ToolstripRadioButton can be added to a ToolstripEx directly or through a panel.

![Figure 1181: ToolstripRadioButton](./radio-button-figure.png)

#### The below properties controls the appearance and behavior of the ToolstripRadioButton item.

#### Foreground Settings

| Property    | Description                                                                                      |
|-------------|--------------------------------------------------------------------------------------------------|
| Font        | Sets the font style for the display text.                                                        |
| Text        | Sets the Text for the ToolstripRadioButton item.                                                 |
| TextAlign   | Specifies the alignment of the text in the item. The options are:                                 |
|             | - TopLeft,                                                                                       |
|             | - TopCenter,                                                                                     |
|             | - TopRight,                                                                                      |
|             | - MiddleLeft,                                                                                    |
|             | - MiddleCenter,                                                                                  |
|             | - MiddleRight,                                                                                   |
|             | - BottomLeft,                                                                                    |
|             | - BottomCenter,                                                                                  |
|             | - BottomRight.                                                                                   |

#### Style Settings

| Property    | Description                                                                                      |
|-------------|--------------------------------------------------------------------------------------------------|
| Checked     | Indicates whether button is checked when the application loads.                                  |

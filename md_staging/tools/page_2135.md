# Essential Tools for Windows Forms

## Overview
- Explains how to programmatically collapse a split pane using the `Collapsed` property.
- Introduces the `TileLayout` control and its use cases in showcasing items.

---

### 3.19.5.1 How to collapse the splitter pane programmatically

This can be done programmatically using the `Collapsed` property.

#### C#
```csharp
this.tabSplitterContainer1.Collapsed = true;
```

#### VB.NET
```vbnet
Me.tabSplitterContainer1.Collapsed = True
```

### 3.20 TileLayout Control for Windows Forms

Essential Tools for Windows Forms enables you to use the `TileLayout` control in an application. This feature has a similar look to that of the Windows 8 Tiles.

#### Use Case Scenarios
This feature can be used in any kind of application that is used to showcase items.

#### Properties

**Table 22: TileLayout Property Table**

| Property          | Description                          | Data Type |
|-------------------|--------------------------------------|-----------|
| SetParentFormFlat | Sets parents form as flat.          | Bool      |
| ShowGroupTitle    | Specifies the group title show.     | Bool      |

---

## API Reference

### Properties
- **SetParentFormFlat**: bool
- **ShowGroupTitle**: bool

---

## Code Examples

#### Example in C#
```csharp
this.tileLayout1.SetParentFormFlat = true;
this.tileLayout1.ShowGroupTitle = true;
```

#### Example in VB.NET
```vbnet
Me.tileLayout1.SetParentFormFlat = True
Me.tileLayout1.ShowGroupTitle = True
```

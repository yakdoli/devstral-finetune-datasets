## Overview
- The `DesktopAlignment` property of the `SplashPanel` is set to `Custom` and the `ShowSplash` method is invoked with the pointer position as a parameter.
- The `StatusBarAdv` is introduced as an advanced StatusBar control with enhanced features.

## Content

### Setting DesktopAlignment and Calling ShowSplash

Set the `DesktopAlignment` property of the `SplashPanel` to `Custom`, and call the `ShowSplash` method by passing the pointer position as the parameter as follows.

#### C# Example
```csharp
Point pt = Point.Empty;
if ( SplashPanel1.DesktopAlignment == SplashAlignment.Custom)
{
    pt = Control.MousePosition;
    SplashPanel1.ShowSplash(pt, this, true);
}
```

#### VB.NET Example
```vb.net
Private pt As Point = Point.Empty
If SplashPanel1.DesktopAlignment = SplashAlignment.Custom Then
    pt = Control.MousePosition
    SplashPanel1.ShowSplash(pt, Me, True)
End If
```

### 3.11.4 StatusBarAdv

#### Overview
The `StatusBarAdv` is an advanced StatusBar control that can display StatusBarPanels with enhanced backgrounds and appearances. The `StatusBarAdv` control can be dragged and dropped from the toolbox onto a form during design time. Users can then drag and drop controls onto the `StatusBarAdv` control.

## API Reference

### Methods

- **ShowSplash(pt, form, keepfocus)**
  - **Parameters:**
    - `pt`: `Point` - The pointer position.
    - `form`: `Form` - The form to display the splash screen on.
    - `keepfocus`: `bool` - A boolean indicating whether to keep focus on the form.

### Properties

- **DesktopAlignment**
  - Type: `SplashAlignment`
  - Description: Specifies the alignment of the SplashPanel within the display area.
  - Default: `SplashAlignment.Custom`

# Essential Tools for Windows Forms

## Overview
- This section demonstrates the usage of XPToolBar components in Windows Forms applications.
- Details the process of adding items to the XPToolBar using verbs.
- Explains hosting a Panel control within the XPToolBarBox.

## Content

### XPToolBarBox: Adding Items
#### Figure 970: Adding XPTaskBar Items using Verbs
![Figure 970: Adding XPTaskBar Items using Verbs](image)

This figure illustrates the context menu for the XPToolBarBox control, which includes an "Add Item" option. This allows users to add new items to the XPToolBar at design time.

### XPToolBar Structure
#### Figure 971: XPToolBar with a single TaskBar Box and TaskBar Item
![Figure 971: XPToolBar with a single TaskBar Box and TaskBar Item](image)

This figure shows a basic XPToolBar with one XPToolBarBox ("xpTaskBarBox1") and a single taskbar item ("XPBarItem0"). The structure demonstrates the minimum configuration required to create a visible XPToolBar.

### Hosting Panel Controls
**Note:** The XPToolBar Boxes can also host a Panel control within it. During design time, users can simply drag and drop the panel on the box. In code, users can do this by adding the necessary logic to host the Panel control.

# Essential Tools for Windows Forms

## Overview
- This section explains various tab styles available in the TabControlAdv control.
- Demonstrates applying different tab styles such as Workbook, InternetExplorer7Style, OneNoteStyle, and various VS2005/Office 2007/2003 styles.
- Details on customizing color schemes for the TabControlAdv.

## Content

### Tab Styles

Each style has a unique visual appearance, as demonstrated by the images below. These styles can be selected to customize the look and feel of the TabControlAdv control in your Windows Forms application.

#### Figure 1365: Tab Styles

The following styles are available:
- **Workbook**
- **2D/3D**
- **InternetExplorer7Style**
- **OneNoteStyle**
- **VS2005DockingStyle**
- **Office2007Style**
- **VS2005Style**
- **VS2005DockingStyleBeta**
- **Office2003Style**
- **VS2008Style**

### Applying Border Styles
- Note: We can apply border styles when TabStyle is set to **VS2008Style**.  
  Refer to the **Border for TabControlAdv** topic to know more.

#### Custom Color Schemes
Custom colors can also be applied to the TabControlAdv. Use the following code snippet:

### WinForms-specific conventions
Use the provided code snippet below for customizing the TabControlAdv with custom colors:

```csharp
using Syncfusion.Windows.Forms.Tools;

// Example of applying custom colors to TabControlAdv
tabControlAdv1.SelectedTabCloseButtonFill = Color.Blue;
tabControlAdv1.SelectedTabCloseButtonHoverFill = Color.DarkBlue;
tabControlAdv1.CloseButtonHoverImage = Image.FromFile("close_hover.png");
// Additional customizations can be applied as needed
```

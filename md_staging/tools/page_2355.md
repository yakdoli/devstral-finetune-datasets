# Essential Tools for Windows Forms

## Overview
- Explains the usage of BannerPanel with `Fixed3D` border style set to "Sunken."
- Provides details on properties for setting 2D border styles in the Banner Control when `GradientPanel.BorderStyle` is set to "FixedSingle."
- Outlines the properties that control the border settings for a Wizard control page.

## Content

### BannerPanel with Fixed3D BorderStyle = "Sunken"

![Figure 1521: BannerPanel with Fixed3D BorderStyle = "Sunken"](images/banner_panel_sunken.gif)

You can use the below properties to set 2D border style for the Banner Control when `GradientPanel.BorderStyle` property is set to "FixedSingle."

#### Properties for 2D Border Style

| Banner Panel Property | Description |
|------------------------|-------------|
| `BorderColor`         | Sets the border color for the Banner panel. |
| `BorderSides`         | Specifies the sides of the control which should have border. |
| `BorderSingle`        | Specifies the 2D Border style for the panel when `BorderStyle` property is set to `FixedSingle`. |

### Wizard Page Border

The below properties control the border settings for a Wizard control page.

#### Properties for Wizard Page Border

| Wizard Page Property  | Description                                                                                                                                                   |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `BorderStyle`         | Specifies the border style for Wizard page.<br>The available styles are `Fixed3D` and `FixedSingle`. |

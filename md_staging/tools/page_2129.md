# Essential Tools for Windows Forms

## Overview
- This page explains the components and properties of the Splitter, a control in Syncfusion WinForms for organizing and managing multiple pages.

## Content

### Components in the Splitter

![Components in the Splitter](#)

**Figure 1418: Components in the Splitter**

- **Primary Page** - We can add primary pages using the `PrimaryPages` property. It invokes the TabSplitterPage Collection Editor and lets you add primary pages.
- **Secondary Page** - We can add secondary pages using the `SecondaryPages` property. It invokes the TabSplitterPage Collection Editor and lets you add secondary pages.
- **Swap Button** - This button lets you swap between the primary and secondary pages.
- **Vertical Orientation** - Sets the orientation of the splitter to be vertical.
- **Horizontal Orientation** - Sets the orientation of the splitter to be horizontal.
- **Expand / Collapse Button** - Clicking this button can collapse the secondary pane and place it at the bottom of the control. Clicking this button again will expand the pane and place it in its original position.

### 3.19.3.1.1 SplitterPage

The properties which customize the Splitter page are as follows.

#### TabSplitterContainer Property

| TabSplitterContainer Property | Description |
|------------------------------|-------------|
| BorderStyle                  | Sets border for the splitter page. |
| BackgroundImage              | Sets the background image for the control. |
| BackgroundImageLayout        | Sets the background image layout for the control. |
| Image                        | Lets you set image icons for the tabs. |

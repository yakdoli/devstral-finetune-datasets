## API Reference

### Methods/Properties/Events Related to Wizard Control

- **BackColor**: Sets the background color for the Wizard control.
- **BackgroundImage**: Sets the background image for the Wizard control.
- **BackgroundImageLayout**: Sets the layout for the background image.

## Summary

The Wizard Control in System.Windows.Forms enables customization of its appearance through several properties such as **BackColor**, **BackgroundImage**, and **BackgroundImageLayout**. This allows developers to align the control's visual style with their application's overall theme. The **Banner Panel Background** provides additional options for customizing the banner panel within the Wizard control. The Note applies a standard of overriding default settings based on Wizard Container background settings, ensuring a consistent look.

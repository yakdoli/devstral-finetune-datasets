# Essential Tools for Windows Forms

## Overview
- Describes the text settings available for GroupView in Windows Forms.
- Focuses on text alignment options, highlighting, and text offset adjustments for GroupView.

## Content

### 3.9.2.4.2.1 Text Settings

This section describes the text alignment options available for GroupView.

#### Text Highlighting

The GroupView control provides highlighting of text when the mouse is over the GroupView Item. This can be activated by setting the **HighlightText** property to 'True'.

##### Example Code

**C#**
```csharp
this.groupView1.HighlightText = true;
```

**VB.NET**
```vb
Me.groupView1.HighlightText = True
```

##### Figure: Text Highlighting in GroupView

![GroupView Control with Highlighted Item](https://i.imgur.com/example_figure_png)

**Figure 937: "My Computer" Item is Highlighted in the GroupView Control**

#### Text Offset

The following properties are used to set the text offset for the GroupView Items.

##### Properties

| GroupView Property     | Description                                |
|------------------------|--------------------------------------------|
| HighlightTextOffset    | Sets the text offset for the highlighted GroupView Item. |

## API Reference

- **HighlightText**  
  - Type: Boolean  
  - Description: Enables or disables text highlighting when the mouse is over the GroupView Item.  

- **HighlightTextOffset**  
  - Type: Int32  
  - Description: Sets the offset for the text highlighting effect.  

## Code Examples

### C# Example for Highlighting

```csharp
this.groupView1.HighlightText = true;
this.groupView1.HighlightTextOffset = 5;
```

### VB.NET Example for Highlighting

```vb
Me.groupView1.HighlightText = True
Me.groupView1.HighlightTextOffset = 5
```

### Designer Support

#### Overview
- This section explains the designer property that enables users to set the color schemes for the Ribbon2013 Style in the design view.

#### WinForms Designer Property

**Designer property**
- The **Office2013ColorScheme Designer property** allows users to set the color schemes for the Ribbon2013 Style. This property is particularly useful in the design phase, empowering users to customize the visual appearance of their Ribbon interface within the design environment.

#### Table: Properties and Descriptions
| Property                | Description                                                                 |
|-------------------------|-----------------------------------------------------------------------------|
| Office2013ColorScheme   | Enables the user to set the color schemes for the Ribbon2013 Style in the design view. |

## API Reference

### Properties
- **Office2013ColorScheme**:
  - **Description**: Allows customization of the Ribbon2013 Style in design mode.
  - **Type**: String or Enum representing different color schemes.

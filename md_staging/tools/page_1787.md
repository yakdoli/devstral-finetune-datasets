# Essential Tools for Windows Forms

## Overview
- Describes essential properties and functionalities of Windows Forms tools.
- Provides details on ToolStrips, ToolTips, and Right-to-Left (RTL) support.

## Content

### Check Box Properties
| Property       | Description                                                                 |
|----------------|-----------------------------------------------------------------------------|
|                | • Checked, <br> • Unchecked and <br> • Indeterminate.                      |
| ThreeState    | Indicates whether the check box can display all the three check states. i.e., Checked, Unchecked and Indeterminate. |
| Enabled       | Specifies whether the item is enabled.                                      |
| Visible       | Specifies whether the item is visible.                                      |
| Alignment     | Sets the alignment of the item within the ToolStriEx. They can be set to beginning (Left) or end (Right) of the ToolStripEx control. |
| AutoSize      | Specifies whether the item should size itself based on its image and text. |

### ToolTip Settings
| Property      | Description                                                                 |
|---------------|-----------------------------------------------------------------------------|
| AutoToolTip   | When set to true, will display the text set in the Text property as the item's tooltip. <br> When set to false, will display the text set in the ToolTipText property. |
| ToolTipText   | Sets the text for the tooltip when AutoToolTip is set to false.                     |

### RTL Support
| Property      | Description                                                                 |
|---------------|-----------------------------------------------------------------------------|
| RightToLeft   | Indicates whether the item should draw right to left for RTL languages.              |

## Code Examples

```csharp
// Example of enabling AutoToolTip
toolStripItem.AutoToolTip = true;
toolStripItem.Text = "This text will appear as tooltips when AutoToolTip is true.";
```

```csharp
// Example of setting ToolTipText
toolStripItem.AutoToolTip = false;
toolStripItem.ToolTipText = "Custom tooltip text when AutoToolTip is false.";
```

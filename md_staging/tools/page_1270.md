# Essential Tools for Windows Forms

## Overview
- Dragging and dropping the MultiColumnTreeView control onto the form.
- Adding nodes using the Nodes Collection Editor.
- Adding columns and subitems in the MultiColumnTreeView control.

## Content

### WinForms-specific conventions

#### MultiColumnTreeView in the Toolbox

![](https://via.placeholder.com/400x200.png?text=Figure+856%3A+MultiColumnTreeView+in+the+Toolbox)

1. **Drag and drop the MultiColumnTreeView control on to the form.**
2. **Open the Nodes Collection Editor, add required nodes using Add nodes and Add Child buttons.**
3. **You can add columns using Columns Editor by clicking add columns button.**
4. **The Subitems can be added through Subitems Collection Editor available in the Nodes Collection Editor. You can add any number of columns using this collection. Refer Adding Multiple Columns and Subitems.**
5. **Appearance and behavior related aspects can be controlled by setting the appropriate properties through the property grid of the MultiColumnTreeView control.**
6. **A simple MultiColumnTreeView with three columns is displayed below.**

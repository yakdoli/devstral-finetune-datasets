# Essential Tools for Windows Forms

## Overview
- Utilizes SuperAccelerator key assignment to ToolStrips for enhanced keyboard navigation.
- Demonstrates assigning accelerator keys to controls for easy user interaction.
- Showcases SuperAccelerator key visibility in RibbonControlAdv.

## Content

### Overview of SuperAccelerator Key Assignment

With the following code sample, the SuperAccelerator key value can be assigned to a `ToolstripTabItem` (named `Home` in the sample). By specifying controls and accelerator keys in arguments, the SuperAccelerator key will be assigned to the appropriate controls and enable users to select particular controls by pressing the assigned SuperAccelerator keys values on the keyboard.

#### Code Sample

[C#]

```csharp
this.superAccelerator1.SetAccelerator(this.Home, "A");
```

## API Reference

### Namespace: Syncfusion.Windows.Forms.Tools

#### Methods
- `SetAccelerator(Control control, string key)`
  - Assigns a SuperAccelerator key to the specified control.

#### Parameters
- **control**: The control to which the SuperAccelerator key is assigned.
- **key**: The accelerator key to be assigned.

### Exceptions
- Throws an exception if the key is already mapped or if the specified control is invalid.

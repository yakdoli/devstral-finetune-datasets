# Essential Tools for Windows Forms

![Figure 1381: TabControlAdv with RotateTabsWhenRTL set to False and True](image.png)

## Overview
- Demonstrates the functionality of the `RotateTabsWhenRTL` property in TabControlAdv.
- Provides examples in C# and VB.NET for enabling horizontal tab alignment in right-to-left layouts.
- Explains the `RotateTextWhenVertical` property for rotating text on tab items aligned left or right.

## Content

### TabControlAdv: Rotating Tabs in Right-to-Left Layouts

When `RotateTabsWhenRTL` is enabled, the tabs in a `TabControlAdv` are displayed horizontally to enhance readability in right-to-left languages.

#### Code Examples

**C#:**
```csharp
this.tabControlAdv1.RightToLeft = System.Windows.Forms.RightToLeft.Yes;
this.tabControlAdv1.RotateTabsWhenRTL = true;
```

**VB.NET:**
```vb
Me.tabControlAdv1.RightToLeft = System.Windows.Forms.RightToLeft.Yes
Me.tabControlAdv1.RotateTabsWhenRTL = True
```

#### Important Note

> Only the following styles support the `RotateTabsWhenRTL` property: VS2005Style, OneNoteStyle, and DockingWhidbeyStyle.

### 3.17.9.2.3 Rotating Text

The `RotateTextWhenVertical` property is used to rotate the text of tab items that are aligned left or right on the page, improving user readability.

#### Property Details

| TabControlAdv Property             | Description                                                                 |
|------------------------------------|-----------------------------------------------------------------------------|
| `RotateTextWhenVertical`           | Specifies to rotate the text along with the tab to draw the text horizontally. |

## API Reference

### TabControlAdv Properties

- **`RotateTabsWhenRTL`**  
  Type: `bool`  
  Determines whether tabs are rotated in a right-to-left layout.  
  Default: `false`

- **`RotateTextWhenVertical`**  
  Type: `bool`  
  Rotates the text of tab items aligned left or right.  
  Default: `false`

## Code Examples

### Example: Using RotateTextWhenVertical

**C#:**
```csharp
this.tabControlAdv1.TabItems["Page1"].RotateTextWhenVertical = true;
```

**VB.NET:**
```vb
Me.tabControlAdv1.TabItems("Page1").RotateTextWhenVertical = True
```

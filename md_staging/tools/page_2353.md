# Essential Tools for Windows Forms

## Border Styles

### Wizard Control

The various border styles for a Wizard control are as follows.

- None
- FixedSingle
- Fixed3D

[C#]

```csharp
this.wizardControl1.BorderStyle = 
System.Windows.Forms.BorderStyle.FixedSingle;
```

[VB.NET]

```vb
Me.wizardControl1.BorderStyle = 
System.Windows.Forms.BorderStyle.FixedSingle
```

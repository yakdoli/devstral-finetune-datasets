# Essential Tools for Windows Forms

## Overview
- Describes the process of creating a `ComboBoxBase` control in Windows Forms.
- Explains how to drag and drop the `ComboBoxBase` control using the designer.
- Provides steps to configure the `ComboBoxBase` with additional data for the popup.

## Content

### Creating ComboBoxBase

`ComboBoxBase` can be created easily through the designer by just dragging and dropping the `ComboBoxBase` control from the Toolbox.

#### Figure 364: ComboBoxBase in Toolbox
![ComboBoxBase in Toolbox](https://i.imgur.com/example_image1.png)

The `ComboBoxBase` control is available in the `Syncfusion` section of the Toolbox.

#### Figure 365: ComboBoxBase in Designer
![ComboBoxBase in Designer](https://i.imgur.com/example_image2.png)

Once added to the form, the `ComboBoxBase` appears as a dropdown control in the designer view.

To add data for the popup, add a `listbox` control to the form and select it in the `ListControl` property.

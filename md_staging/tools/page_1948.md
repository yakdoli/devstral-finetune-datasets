# Essential Tools for Windows Forms

## Overview
- A calculator-style form demonstrating the use of `SkinManager` for theming.
- Code snippets to apply the Office2007Blue theme using `SkinManager` in C# and VB.

## Content

### Applying Skin Manager to the Entire Form
To add Skin Manager to the entire form, specify the form as the root control. The following code illustrates this:

#### Calculator UI
The form features a graphical user interface with calculator buttons for mathematical operations.

#### Code Examples

##### C#
```csharp
SkinManager.SetVisualStyle(this, VisualTheme.Office2007Blue);
```

##### VB
```vb
SkinManager.SetVisualStyle(Me, VisualTheme.Office2007Blue)
```

### Explanation
The `SkinManager` is used to apply a visual theme to the entire form, in this case, the "Office2007Blue" theme. This enhances the appearance of the application making it visually consistent with Microsoft Office 2007 standards.

### Essential Tools for Windows Forms

#### Overview
- Demonstrates properties related to Context Menu Strip.
- Explains how to configure the display of check margins and image margins.

#### Content

**Note:**  
The check functionality can be enabled using the `Checked` property, and the check state can be provided using the `CheckedState` property available for individual menu items, through the items collection editor.

##### RunTime Features

**Menu Item Tooltips**

The menu items can display tooltips while the mouse is hovering over them by enabling the `ShowItemToolTips` property.

- **Property Configuration:**

  ```csharp
  this.contextMenuStripEx1.ShowItemToolTips = true;
  ```

#### API Reference

- **Property:** `ShowItemToolTips`
  - Type: `bool`
  - Description: Enables or disables tooltips for menu items while hovering.

#### Code Examples

- **C# Example:**
  
  ```csharp
  this.contextMenuStripEx1.ShowItemToolTips = true;
  ```

# Essential Tools for Windows Forms

## Overview
- Demonstrates how to use the TabControlAdv control in Syncfusion Winforms.
- Focuses on the TabPrimitives property and its editor to customize tab items.

## Content

### Using TabControlAdv Properties

#### TabPrimitives Property

The TabPrimitives property allows customization of tab items in the TabControlAdv control. Here's how to use it:

1. Open the Properties Grid for the TabControlAdv control.
2. Locate the "TabPrimitives" property, as shown in Figure 1356.

**Figure 1356: TabPrimitives property in the Properties Grid**

#### Adding a TabPrimitive

3. A TabPrimitives Collection Editor will be opened.
4. Click the Add option in the Editor to add a TabPrimitive.

**Figure 1357: TabPrimitive Collection Editor**

### TabPrimitive Collection Editor

The TabPrimitive Collection Editor allows you to define and customize individual tab items (TabPrimitives). Here's a breakdown of the interface elements:

- **Members:** This section lists all TabPrimitives currently defined.
- **TabPrimitive0 properties:** This section displays properties for the selected TabPrimitive.
  - **Enabled:** Determines the tab's visibility and activity.
  - **Image:** Allows setting an image for the tab.
  - **Indent:** Sets the indentation level for the tab.
  - **Name:** The internal name of the TabPrimitive.
  - **TabPrimitiveType:** Specifies the type of tab (Custom, Standard, etc.).
  - **ToolTip:** Text displayed when hovering over the tab.
  - **TransparentImage:** Determines if the tab's image should have transparent areas.
  - **Visible:** Controls whether the tab is visible.

**Figure 1357: TabPrimitive Collection Editor**

```csharp
// Example of adding a TabPrimitive programmatically
TabPrimitive newTab = new TabPrimitive();
newTab.Name = "NewTab";
newTab.Text = "Custom Tab";
tabControlAdv1.TabPrimitives.Add(newTab);
```

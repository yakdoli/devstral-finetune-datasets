# Essential Tools for Windows Forms

## Content

### Image Spacing

We can provide spacing between the highlighted edge of a GroupView Item and the image by setting the `ImageSpacing` property to integer values.

#### HighlightImage property must be set to 'True'.

**C#**
```csharp
this.groupView1.ImageSpacing = 7;
```

**VB.NET**
```vb.net
Me.groupView1.ImageSpacing = 7
```

![GroupView with ImageSpacing = "7"](https://i.imgur.com/example_image.png)
**Figure 954: GroupView with ImageSpacing = "7"**

## API Reference

### Methods

- **ResetSelectingImageOffset**  
  Resets the `SelectingImageOffset` property to its default value.

- **ResetSelectedHighlightImageOffset**  
  Resets the `SelectedHighlightImageOffset` property to its default value.

- **ResetHighlightImageOffset**  
  Resets the `HighlightImageOffset` property to its default value.

## Code Examples

The following examples demonstrate how to set the `ImageSpacing` property for a `GroupView`.

```csharp

```

**VB.NET**
```vb.net

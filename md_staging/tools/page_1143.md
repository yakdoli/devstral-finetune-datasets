# Essential Tools for Windows Forms

## Overview
- Describes the behavior and common properties of `ListBarItem`, a component supporting bar item functionalities.
- Introduces `MDIListBarItem`, a specialized `ListBarItem` for multi-document interface (MDI) child windows.
- Explains adding `MDIListBarItem` using the designer and setting its size.

## Content

### Behavior Settings

`ListBarItem` supports all the behavior properties of `BarItem`.

### Other Common properties

- For setting images, see [Image Settings](#).
- For editing the text and setting text alignments, see [BarItem Text](#).
- For setting shortcuts for the bar items, see [Keyboard shortcuts](#).
- For changing the Paint Style, see the [PaintStyle](#) topic.
- Appearance of the Text can be customized. See [Foreground Settings](#) for details.

- 3.7.4.1.2.2.6 `MDIListBarItem`

An `MDIListBarItem` is a `ListBarItem` that expands itself to show a list of MDIChild windows in the form, when placed in a ParentBarItem.

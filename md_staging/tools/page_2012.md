## Essential Tools for Windows Forms

### Overview
- Demonstrates how to manage images and their placement in `TabControlAdv`.
- Explains properties to set images outside the TabBounds.
- Provides details on setting the background image for TabPages.

## Content

### Images outside the TabBounds
`TabControlAdv` can have images set outside the TabBounds.

```csharp
Me.tabControlAdv1.ImageIndex = 0
```

#### Figure 1369: TabControlAdv with Images set outside the TabBounds
![Figure 1369: TabControlAdv with Images set outside the TabBounds](https://example.com/image_link)

**To move the image outside the TabBounds, set the following properties:**

| TabControlAdv Property        | Description                                                                 |
|-------------------------------|-----------------------------------------------------------------------------|
| `ImageOffset`                 | Adjusts Y-axis position of the image.                                    |
| `AdjustTopGap`                | Adjusts the gap between the tabcontroladv's top and tabs.                |
| `LevelTextAndImage`           | Indicates whether the text and image should be in the same level.       |
| `ItemSize`                    | Sets the size of the tabs.                                                |

### Background Image for TabPages
The below properties set the background image for the pages.

| TabPageAdv Property    | Description                                               |
|------------------------|-----------------------------------------------------------|
| `BackgroundImage`      | Specifies the background image for the tabpage.          |

## Essential Tools for Windows Forms

A **ParentBarItem** represents a submenu (drop-down menu) which can display one or more child BarItems on drop-down. A **ParentBarItem** can be placed inside a toolbar and it can contain another **ParentBarItem** as its child bar item. **ParentBarItem** comes with properties to control its appearance and behavior.

![](https://example.com/image.png "Adding ParentBarItem by selecting Type as ParentBarItem through Designer using Customize Dialog Box")

### Behavior Settings

The following properties control the behavior of the **ParentBarItem**.

| ParentBarItem Property   | Description                                                                                          |
|--------------------------|------------------------------------------------------------------------------------------------------|
| Checked                  | Draws the bar item with a checked appearance.                                                       |
| Enabled                  | Enables or disables the bar item. Default value is true.                                           |
| IsRecentlyUsedItem       | Indicates whether this item will appear in its parent's partial menus list.                         |
| Visible                  | Sets the visibility of the bar item.                                                                 |
| MergeOrder               | Relative Position of the bar item when it is merged with another.                                   |
| MergeType                | Sets behavior of the bar item when its bar is merged with another. The options are, <br> - **Add**: Adds to the existing menu items. (Default) <br> - **Replace**: Replaces an existing menu items at the same position in a merged menu. |

## API Reference (if applicable)

- **Checked**: Property to set the check mark appearance of the **ParentBarItem**.
- **Enabled**: Property to enable or disable the **ParentBarItem**.
- **IsRecentlyUsedItem**: Property to indicate whether the item should appear in its parent's partial menus list.
- **Visible**: Property to control the visibility of the **ParentBarItem**.
- **MergeOrder**: Property to set the relative position of the **ParentBarItem** when it is merged with another.
- **MergeType**: Property to determine the behavior when the bar is merged with another. Options include:
  - **Add**: Adds to the existing menu items (default).
  - **Replace**: Replaces existing menu items at the same position in a merged menu.

## Code Examples

Example of setting properties for **ParentBarItem**:

```csharp
// Create ParentBarItem
ParentBarItem parentBarItem = new ParentBarItem();
parentBarItem.Checked = true;
parentBarItem.Enabled = true;
parentBarItem.IsRecentlyUsedItem = true;
parentBarItem.Visible = true;
parentBarItem.MergeOrder = 1;
parentBarItem.MergeType = MergeType.Add;

// Add to a Toolbar
ToolBar toolbar = new ToolBar();
toolbar.Items.Add(parentBarItem);
```

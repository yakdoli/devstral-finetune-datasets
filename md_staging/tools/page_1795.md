# Essential Tools for Windows Forms

## Content

### AutoSizing of ToolStripEx

By enabling the `AutoSize` property of `ToolStripEx`, the toolbar width will be resized automatically while adding controls to the toolbar in the designer.

#### Toolstrip Width Adjustments

| Property  | Description                                                                                     |
|-----------|-------------------------------------------------------------------------------------------------|
| AutoSize  | Setting this to `true` will automatically resize the toolbar as controls are added.            |

#### Code Examples

```csharp
this.toolStripEx1.AutoSize = true;
```

```vb.net
Me.toolStripEx1.AutoSize = True
```

The `ToolStripLayoutStyle` property is used to specify the alignment for the `ToolStripEx` items. The following table lists the possible alignments.

| Constraint | Description                                                                                   |
|------------|-------------------------------------------------------------------------------------------------|
| Flow       | Specifies that items flow horizontally or vertically as necessary. <br> Note: The Alignment property is ignored under this Layout |

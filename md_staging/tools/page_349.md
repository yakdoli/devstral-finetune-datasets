# Essential Tools for Windows Forms

## Overview
- BackgroundImage is used for the control's background and can be laid out using properties.
- Images can be added to ButtonAdv in two ways, either using the Image property or additional alignment properties.

## Content

### BackgroundImage
BackgroundImage is the image used as the Background for the control, which is set using the `BackgroundImage` property. This BackgroundImage can be laid in various manner with the `BackgroundImageLayout` property.

![Figure 160: Background Image for ButtonAdv](https://i.imgur.com/ignore/syncfusion-image-for-ex.png)

### Image
#### Image that will be displayed on the control.

![Figure 161: Image for ButtonAdv](https://i.imgur.com/ignore/syncfusion-image-for-ex.png)

#### Adding Images to ButtonAdv
The Images can be added to the ButtonAdv control in two ways. Either `Image` property can be used, or the following properties:

| **Properties** | **Description** |
|----------------|-----------------|
| `ImageList`    | Sets the imagelist used for this control. |
| `ImageAlign`   | Sets the alignment of the image inside the control. The options are, <br> TopLeft, <br> TopCenter, <br> TopRight, <br> MiddleLeft, <br> MiddleCenter, <br> MiddleRight. |

- Preserves control properties and design parameters as described.

### API Reference
- Properties such as `BackgroundImage` and `Image` can be used to enhance the control's visual presentation.

# Essential Tools for Windows Forms

## Overview
- Describes techniques for setting images on controls in a CardLayout.
- Covers sizing and layout options for child controls within the CardLayout.

## Content

### Code Examples

#### Setting an Image for a Control

##### C#
```csharp
this.label1.Image =
    ((System.Drawing.Bitmap)(resources.GetObject("label1.Image")));
```

##### VB.NET
```vb
Me.label1.Image = DirectCast((resources.GetObject("label1.Image")),
                               System.Drawing.Bitmap)
```

### Size

The preferred size and minimum size of the Child controls can be set using the `PreferredSize` and `MinimumSize` extended properties of the Child controls that are added to the CardLayout. Refer to [Child Control Settings](#child-control-settings) to know more about this topic.

### Layout Mode

The CardLayout provides two modes to layout the Child controls. The mode can be set using the properties given below.

#### Table: CardLayout Properties

| CardLayout Property | Description                          |
|---------------------|--------------------------------------|
| [unclear]          | [unclear]                            |

### Child Control Settings
- **PreferredSize**: Sets the preferred size for the child control.
- **MinimumSize**: Sets the minimum size constraint for the child control.

### Layout Properties
- **Layout Mode**: Specifies the mode used to arrange child controls.

# Essential Tools for Windows Forms

## Overview

- Highlights the DockLabel property in the Property Grid of a docked control.
- Demonstrates setting the Caption for a docked control.
- Explains how to align the CaptionLabel to the left.
- Notes on setting DockLabelAlignment using the Task Window.
- Discusses showing images in captions using the ShowCaptionImages property.

### Custom CaptionLabel Alignment

- **Custom CaptionLabel aligned to left:**
  - The caption label can be aligned to the left using the DockLabelAlignment property.

#### Note

**Note:** DockLabelAlignment can also be set easily using `Task Window`.

### Image for the Caption

- The captions can also hold images, which can be enabled using the `ShowCaptionImages` property.

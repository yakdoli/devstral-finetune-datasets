### 3.7.4.1.2.2.5 ListBarItem

A `ListBarItem` is a `BarItem` which, when added to a `ParentBarItem`, will expand itself into an ordered list of `BarItems` represented by the string list in its `ChildCaptions` property.

#### Figure: Adding ListBarItem in the designer using Customize Dialog Box

![Figure 757: Adding ListBarItem in the designer using Customize Dialog Box](https://i.imgur.com/figure.jpg)

**Figure 757: Adding ListBarItem in the designer using Customize Dialog Box**

#### Adding Items for the ListBarItems

### Conclusion

The `ListBarItem` provides a mechanism to expand into an ordered list, offering flexibility in UI design for Windows Forms applications.

## API Reference

- **Property**: `ChildCaptions`
  - **Description**: Represents the list of child bar items in a `ListBarItem`.

## Code Examples

#### C# .Net
```csharp
ListBarItem item = new ListBarItem();
item.ChildCaptions.Add("Item 1");
item.ChildCaptions.Add("Item 2");
toolStrip1.Items.Add(item);
```

#### VB.Net
```vb
Dim item As New ListBarItem()
item.ChildCaptions.Add("Item 1")
item.ChildCaptions.Add("Item 2")
toolStrip1.Items.Add(item)
```

# Essential Tools for Windows Forms

## Overview
- Lists the various bar items used in Windows Forms.
- Explains how to fill the XP Toolbar with items from the BarManager.
- Demonstrates dragging and dropping items from the Customize dialog.

## Content

### Bar Items
- BarItem
- ParentBarItem
- DropDownBarItem
- ComboBoxBarItem
- StaticBarItem
- TextBoxBarItem
- ListBarItem
- MdiListBarItem
- ToolbarListBarItem

### Filling the XP Toolbar with Items from the BarManager

You can drag-and-drop items from the **Customize** dialog of the **BarManager** into the **XP Toolbar**, in the same way you filled the menus and toolbars.

# Essential Tools for Windows Forms

## Overview
- Complete persistence support is now available for TabControlAdv using the PersistTabState property.
- Close button can be added for each TabPage like IE7 to close the tabpage.
- TabState has been saved at a specific location.
- Features include setting tooltips, adding VS2008 TabStyle, Border settings, and handling TabPage Closed and Closing events.

## Content

### TabControlAdv

#### TabControlAdv in Toolbox

- **Description**: TabControlAdv is a control in the Tabs Package that manages a set of TabPages and provides options to customize the TabPages according to the needs of the user.

#### Features and Enhancements
- **Persistence Support**: Complete persistence support is now available for the TabControlAdv using the PersistTabState property. TabState has been saved at the location:
  ```
  C:\Documents and Settings\Username\Local Settings\Application Data\IsolatedStorage.
  ```
- **Close Button**: A close button can be added for each TabPage, similar to IE7 functionality. This is illustrated in Figure 1339.

---

![Close button added for Tab Page](https://raw.githubusercontent.com/Syncfusion-Docs/tools/master/images/close_button_added_for_tab_page.png)

*Figure 1339: Close button added for Tab Page*

- **Tooltips**: You can set tooltips for every TabPrimitive.
- **VS2008 TabStyle**: Added VS2008 TabStyle for TabControlAdv.
- **Border Settings**: Added Border settings for the control, implemented especially for VS2008 TabStyle.
- **Events**: TabPage Closed and Closing events are added.

#### Sample Usage
A sample that illustrates the features of TabControlAdv is available in the following installation location:

```
..\<My Documents>\Syncfusion\EssentialStudio\<Version Number>\Windows\Tools.Windows\Samples\2.0\Tabs Package\TabControlAdvDemo
```

# Essential Tools for Windows Forms

The ribbon panel supports all the three office color schemes: Blue, black, and silver schemes.

## Overview
- The ribbon panel supports the Blue, black, and silver office color schemes.
- Examples in both C# and VB.NET are provided to set the color scheme for the ribbon panel.
- Visual illustrations showcasing the different color schemes (Silver, Blue, and Black) are included.

## Content

### Setting Office Color Scheme for RibbonPanel

#### C#
```csharp
this.toolStripTabItem2.Panel.OfficeColorScheme = 
Syncfusion.Windows.Forms.Tools.ToolStripEx.ColorScheme.Blue;
```

#### VB.NET
```vb
Me.toolStripTabItem2.Panel.OfficeColorScheme = 
Syncfusion.Windows.Forms.Tools.ToolStripEx.ColorScheme.Blue
```

## API Reference

### Members
- **Syncfusion.Windows.Forms.Tools.ToolStripEx.ColorScheme** is used to set the color scheme.
- **OfficeColorScheme** is the property of the panel to configure the desired scheme.

### Parameters
- **ColorScheme**: The OfficeColorScheme enumeration, providing options for Blue, Silver, and Black.

The provided code snippets in C# and VB.NET demonstrate how to apply the blue color scheme to the ribbon panel.

# Essential Tools for Windows Forms

## Overview
- Explains the use of TabControlAdv with background images and image settings for TabItem.
- Describes the ImageAlignmentR property and its options.
- Provides code snippets for setting image alignment.

## Content

### TabControl with Background Image Set for the TabPage and Image Set for the TabItem

![TabControl with Background Image Set for the TabPage and Image Set for the TabItem](attachment:Figure-1368.png)

**Figure 1368: TabControl with Background Image Set for the TabPage and Image Set for the TabItem**

**ImageAlignmentR** of the TabControlAdv allows you to set the tab's text and image. By default, the image appears to the left and the text appears to the right. This setting can be changed using one of the below given options.

| **TabControlAdv Property** | **Description** |
|----------------------------|-----------------|
| ImageAlignmentR           | Specifies the alignment of the image relative to the text. The options include: |
|                            | - LeftOfText,    |
|                            | - RightOfText,   |
|                            | - AboveText,     |
|                            | - BelowText,     |
|                            | - Overlap.       |
| ImageIndex                | Gets / sets the image for the tab item.                           |

### Code snippets showing the Image Settings

#### [C#]

```csharp
this.tabControlAdv1.ImageAlignmentR = Syncfusion.Windows.Forms.Tools.RelativeImageAlignment.RightOfText;
this.tabPageAdv1.ImageIndex = 0;
```

#### [VB]

```vb
Me.tabControlAdv1.ImageAlignmentR = Syncfusion.Windows.Forms.Tools.RelativeImageAlignment.RightOfText
```

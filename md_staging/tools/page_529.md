### Custom Groups
- **CustomGroups**: Allows addition of custom ColorGroups apart from the default groups.

### Layout Properties
- **Anchor**: Top, Left
- **Dock**: None
- **Location**: 12, 12
- **MaximumSize**: 0, 0

### Known Sections of Color Groups

- **Header**: Name of the color group.
- **Header Text**: Text displayed in the header.
- **Group Color Items**: Set of colors within a group.
- **Color Items or Sub Items**: Individual colors within the group.

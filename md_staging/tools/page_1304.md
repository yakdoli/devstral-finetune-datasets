## Overview

- Describes various properties and functionalities for nodes in a Windows Forms environment.
- Covers settings for visibility, style, and image indexing for different states of a node.
- Includes options for managing node states, sorting, and image padding.

## Content

### Node Properties

| Property Name             | Description                                                                 |
|---------------------------|-----------------------------------------------------------------------------|
| BaseStyle                 | Specifies the base style of the node.                                     |
| EnsureDefaultOptionedChild| Specifies if at least one child of the parent node should be optioned at all times. |
| InteractiveCheckbox       | Sets the visibility of the InteractiveCheckbox for the node.              |
| ClosedImageIndex          | Indicates the image index for closed node.                                 |
| CollapseImageIndex        | Indicates the image index in the NodeStateImageList, when the node is collapsed. |
| ExpandImageIndex          | Indicates the image index in the NodeStateImageList, when the node is Expanded. |
| LeftImageIndices          | Specifies the ImageIndex for the left image.                               |
| LeftImagePadding          | Padding for the left image.                                               |
| LeftStateImagePadding     | Padding for the left state image.                                         |
| NoChildrenImageIndex      | Indicates the image index in the StaticImageList where node has no children. |
| OpenImageIndex            | Indicates the image index for the open node.                               |
| RightImageIndices         | Specifies the image index for the right image.                             |
| RightImagePadding         | Padding for the right image.                                              |
| RightStateImagePadding    | Padding for the right state image.                                        |
| ComparerOptions           | Comparer options during sorting.                                          |
| Comparer                  | IComparer object that compares two items.                                |

## API Reference

### Properties

- **BaseStyle**: Specifies the base style of the node.
- **EnsureDefaultOptionedChild**: Ensures that at least one child of the parent node is optioned at all times.
- **InteractiveCheckbox**: Sets the visibility of the InteractiveCheckbox for the node.
- **ClosedImageIndex**: Indicates the image index for closed nodes.
- **CollapseImageIndex**: Indicates the image index in the NodeStateImageList when the node is collapsed.
- **ExpandImageIndex**: Indicates the image index in the NodeStateImageList when the node is expanded.
- **LeftImageIndices**: Specifies the ImageIndex for the left image.
- **LeftImagePadding**: Padding for the left image.
- **LeftStateImagePadding**: Padding for the left state image.
- **NoChildrenImageIndex**: Indicates the image index in the StaticImageList for nodes with no children.
- **OpenImageIndex**: Indicates the image index for the open node.
- **RightImageIndices**: Specifies the image index for the right image.
- **RightImagePadding**: Padding for the right image.
- **RightStateImagePadding**: Padding for the right state image.
- **ComparerOptions**: Options for sorting using a comparator.
- **Comparer**: An IComparer object used for comparing two items.

# Essential Tools for Windows Forms

## Overview
- Property settings for configuring the behavior and appearance of buttons in Windows Forms.
- Event handling for interactions such as button state changes and drop-down item selections.

## Content

### 3.16.1.3.1 Properties

#### Table 19: Properties Table

| Property        | Description                                                                 | Type    | Data Type                     | Reference links              |
|-----------------|-----------------------------------------------------------------------------|---------|-------------------------------|------------------------------|
| ButtonMode      | Specifies the mode of the button as normal or toggle.                     | Normal  | Enum                          | Button Mode                 |
| IsButtonChecked | Specifies the state of the button. This property can be set only when the ButtonMode is set to toggle. | Normal  | Boolean                       | NA                           |
| DropDownItems   | Sets the item to be added in the drop-down list.                          | Normal  | ToolStripItemCollection      | Adding Items to Drop-Down   |

### 3.16.1.3.2 Events

#### Table 20: Events Table

| Event             | Description                                                                              | Arguments                 | Type | Reference links |
|-------------------|------------------------------------------------------------------------------------------|---------------------------|------|-----------------|
| DropDownItemChecked | Will be triggered when drop-down item is clicked.                            | ToolStripItemClickedEventArgs | NA   | NA              |
| Checked            | Will be triggered when the button is checked. This event can be triggered only when the button is in toggle mode. | EventArgs                  | NA   | NA              |
| UnChecked          | Will be triggered when the button is unchecked. This event can be triggered only when the button is in toggle mode. | EventArgs                  | NA   | NA              |

## Code Examples

```csharp
// Example of setting ButtonMode to toggle
myButton.ButtonMode = ButtonMode.Toggle;

// Example of handling Check event
myButton.CheckedChanged += (sender, e) => 
{
    // Handle checked state change
};

// Example of adding items to DropDown
var item = new ToolStripMenuItem("Menu Item");
myButton.DropDownItems.Add(item);
```

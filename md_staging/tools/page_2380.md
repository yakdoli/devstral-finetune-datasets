## Overview

### Reordering Pages Using 'Bring To Front' and 'Send To Back' Options

![Figure 1537: Reordering Pages Using 'Bring To Front' and 'Send To Back' Options](images/figure_1537.png)

The image above illustrates the context menu options available for reordering pages in a Windows Forms application. By right-clicking on a document or control, options such as 'Bring to Front' and 'Send to Back' allow developers to adjust the z-order of the elements.

### Going to Next Page or Previous Page

- **Right Click a page in the designer and choose the 'Previous Page' or 'Next Page' verb** which will show you the page, which is before the current page or the page which is after the current page.
- These options can also be accessed through smart tag and property grid commands.

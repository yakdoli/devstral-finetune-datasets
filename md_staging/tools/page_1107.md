# Essential Tools for Windows Forms

## Overview
- Customizing menu options and layout using the Designer in Windows Forms.
- Accessing the Properties Grid to modify commands and add new categories.

## Content

### Figure 718: Customize option in Designer on Right-Clicking the Mouse
The first figure illustrates the process of customizing options in the Designer by right-clicking the mouse. It shows a context menu that includes options such as:
- Customize...
- Activate Menus
- Add Detached CommandBar
- Add Detached ControlBar
- Cut
- Copy
- Paste
- Delete
- Properties

### Figure 719: Customize Command in the Property Grid
The second figure demonstrates how to use the Properties Grid to customize commands. It includes:
- **mainFrameBarManager1**:
  - **Misc**:
    - AfterClone
    - BarControlBindingChange
    - FloatingFormClosed
    - ProvidePersisteceID
    - ResetBarItemClicked
    - UserChangedBarVisibility
  - **Property Changed**:
    - FontChanged
- Additional options like "Customize...", "Activate Menus", "Add Detached CommandBar", and "Add Detached ControlBar" are available.
- The **SelectedItemChanged** event is described, which occurs when a new BarItem is selected or unselected by the user.

### Steps for Customization
- **In the customization dialog**:
  - Go to the **Commands** tab.

- **To add new categories**:
  - Right-click on the **Categories** area and select **Add**.

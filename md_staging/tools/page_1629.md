# Essential Tools for Windows Forms

## Overview

- Demonstrates different HALign options for text alignment.
- Explains how to set tooltips for StatusBarAdvPanel using properties.
- Includes code examples in C# and VB.NET.

## Content

### HAlign Options

Figure 1074 illustrates the HALign options available for text alignment:

- **Left**: Text is aligned to the left side.
- **Right**: Text is aligned to the right side.
- **Center**: Text is centered.
- **Justify**: Text is justified across the width.

### ToolTip

#### ToolTip for StatusBarAdvPanel

##### Property Usage

ToolTip can be set for the StatusBarAdvPanel using the property provided below.

| StatusBarAdvPanel Property | Description |
|-----------------------------|-------------|
| ToolTip                    | Indicates the tooltip of the panel. <br> The text to be displayed in the tooltip can be directly given in the property window. |

##### Code Implementation

This can be done through code as follows:

- **C#**
  ```csharp
  this.statusBarAdvPanel1.ToolTip = "Tooltip for StatusBarAdvPanel";
  ```

- **VB.NET**
  ```vb
  Me.statusBarAdvPanel1.ToolTip = "Tooltip for StatusBarAdvPanel"
  ```

### 3.21.3.7 LoadOnDemand

#### Overview
- The `LoadOnDemand` feature enables a delay in loading a node in a Tree, ensuring the user initiates a node before it expands.

#### Implementation
By setting the `LoadOnDemand` property to `true`, the plus (+) and minus (-) symbols of all nodes will be visible initially. Handling the `BeforeExpand` event allows subnodes to be added dynamically to the respective nodes. Based on whether children are added, the tree will display or hide the plus or minus symbols.

#### TreeViewAdv Properties

| TreeViewAdv Properties         | Description                                                                                                                                                 |
|---------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **LoadOnDemand**                | Specifies if the tree should follow the `LoadOnDemand` paradigm.                                                                                          |
| **AddSeparatorAtEnd**           | Indicates if the `TreeNodeAdv.GetPath` method adds a separator at the end of the path string returned.                                                   |

## Code Examples
```csharp
// Example: Enabling LoadOnDemand in a TreeViewAdv
treeViewAdv1.LoadOnDemand = true;

// Handling BeforeExpand event to load subnodes
treeViewAdv1.BeforeExpand += (sender, e) =>
{
    // Add subnodes to the expanding node
    e.Node.Nodes.Add(new TreeNodeAdv("SubNode1"));
    e.Node.Nodes.Add(new TreeNodeAdv("SubNode2"));
};
```

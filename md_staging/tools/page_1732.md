# Essential Tools for Windows Forms

## Overview
- The RibbonControlAdv offers new color schemes to enhance the personalized experience in Windows Forms.
- The Office 2013 color schemes are now supported, providing three themes: White, LightGray, and DarkGray.
- The `Office2013ColorScheme` property can be utilized to customize the visual appearance of the RibbonControlAdv.

## Content

### 3.12.1.2.3.4 Office 2013 Color Schemes

The RibbonControlAdv includes new color schemes as available in the Office 2013 suite to provide a more personalized experience with the RibbonControlAdv.

RibbonControlAdv now supports three themes: White, LightGray, and DarkGray. These themes can be set through the `Office2013ColorScheme` property to tone down the bright white background of the ribbon.

The following code sample illustrates how to set the DarkGray theme for the Office 2013 style RibbonControl.

#### C#
```csharp
this.ribbonControlAdv1.Office2013ColorScheme = Office2013ColorScheme.DarkGray;
```

#### VB.NET
```vb
Me.ribbonControlAdv1.Office2013ColorScheme = Office2013ColorScheme.DarkGray
```

## API Reference

### Properties
- **Office2013ColorScheme**: Allows setting one of the three available Office 2013 themes (White, LightGray, DarkGray).

### Methods
- No additional methods are specified for this feature.

### Enumerations
- **Office2013ColorScheme**: Enum to define the supported color schemes.
  - **DarkGray**
  - **LightGray**
  - **White**

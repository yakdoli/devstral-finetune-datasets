## Node Level Style and Column Styles

### Overview

This page describes the application of styles to nodes and columns in the **MultiColumnTreeView** control. Node Level styles can be applied using the `TreeNodeAdv.BaseStyle` property, and column styles utilize `TreeColumnAdv` properties to control the appearance of columns.

### Content

#### Node Level Style

**Figure 882: Node Level Style**

Node styles can be applied to any node by using the `TreeNodeAdv.BaseStyle` property for a specific node. Refer to the **Style Settings table** in **Node Specific Style** for details.

#### Column Styles

**3.8.3.7.4 Column Styles**

- **Standard Column Style**: This is the default style applied to all columns of the **MultiColumnTreeView** control. Users can edit these style settings.

##### Column Style Properties

The following properties control the appearance of the columns:

| TreeColumnAdv Property            | Description                                                                 |
|------------------------------------|-----------------------------------------------------------------------------|
| AllowTextOverlap                  | Indicates whether the text can overlap or not. By default, it is false.   |
| AreaBackground                    | Gets or sets the background for the column area.                           |
| Background                         | Sets the background for the column (column header).                       |
| Font                               | Sets the foreground style for the columns.                                |
| HorizontalAlignment               | Sets the horizontal alignment of the text in the columns.                  |
| TextColor                          | Sets the text color for the columns.                                     |
| Vertical Alignment                | Sets the vertical alignment of the text in the columns.                   |
| BaseStyle                         | Sets the base style to be applied to the column.                          |
| Width                              | Specifies Column width.                                                   |
| Border3DStyle                     | Sets the 3D border style for the column.                                 |

Example code demonstrating the use of these styles could include setting the `BaseStyle` property of a `TreeNodeAdv` and modifying `TreeColumnAdv` properties to control column appearance.

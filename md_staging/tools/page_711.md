# Essential Tools for Windows Forms

**Figure 442: CurrencyEdit Control with its Components**

The properties which control the behavior of the Calculator button are as follows.

## CurrencyEdit Properties

| **CurrencyEdit Properties**         | **Description**                                                                                                                                                                                                                                                                                                  |
|-------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **ShowCalculator**                  | Specifies whether the calculator button is to be displayed. By default, it is `True`.                                                                                                                                                                                                                           |
| **CalculatorButton**                | Provides properties to customize the calculator button. See **ButtonEdit** control user guide to know about the settings.                                                                                                                                                                                         |
| **CalculatorLayoutType**            | Specifies the LayoutType for the Calculator control. The layout types are:<br/><br/> `Financial` (default) and `WindowStandard`.                                                                                                                                                                               |
| **PopupCalculatorAlignment**        | It sets the alignment of the popup calculator with respect to the ButtonEdit control. The options are:<br/><br/> `Right` (default) and `Left`.                                                                                                                                                                    |
| **CloseAction**                     | Specifies the action that will close the popup calculator. After performing calculations in the popup calculator, we may press `'='` to close the popup calculator. This is called closed action. We can select any operator instead of `'='` to close the popup calculator. The options are:<br/><br/> `Calcdigit0â€¦Calcdigit9`,<br/> `CalcOperatorNone`,<br/> `CalcOperatorMultiply`,<br/> `CalcOperatorPlus`,<br/> `CalcOperatorMinus` |

#### 3.9.1.4.2.1 Text Settings
This section describes how text alignment can be done for the GroupBar items.

#### Through Designer
The `TextAlign` property specifies the horizontal alignment of the GroupBar item text. The text can be aligned to the Center, Left, and Right. The default alignment is set to Center.

---

**Figure 900: GroupBar with TextAlign = "Left"**

This screenshot shows the GroupBar control with the `TextAlign` property set to "Left". The GroupBar control is displayed in the Designer view with properties visible in a property grid on the right. The following properties are visible in the property grid:

- **ShowChevron**: True

- **TabStop**: True

- **Text**: groupBar1

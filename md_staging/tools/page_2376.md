# Essential Tools for Windows Forms

| Property          | Description                                                                                                                                                                                                                         |
|-------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|                   | - RaisedOuter, <br> - SunkenOuter, <br> - RaisedInner, <br> - Raised, <br> - Etched, <br> - SunkenInner, <br> - Bump, <br> - Sunken, <br> - Adjust and <br> - Flat. |
| BorderColor       | Sets the border color for the Task page.                                                                                                                                                                                           |
| BorderSides       | Specifies the sides of the control which should have border.                                                                                                                                                                      |
| BorderSingle      | Specifies the 2D Border style for the Task page when BorderStyle property is set to FixedSingle. The available styles are, <br> - Dotted, <br> - Dashed, <br> - Solid, <br> - Inset and <br> - Outset.                         |


## Code Examples

### C#

```csharp
this.xpTaskPage1.BorderColor = System.Drawing.Color.SteelBlue
this.xpTaskPage1.BorderStyle =
System.Windows.Forms.BorderStyle.FixedSingle
```

### VB.NET

```vb
Me.xpTaskPage1.BorderColor = System.Drawing.Color.SteelBlue
Me.xpTaskPage1.BorderStyle =
System.Windows.Forms.BorderStyle.FixedSingle
```

## Notes

The code examples demonstrate how to set the border color and style for a Task page in a Syncfusion WinForms application using both C# and VB.NET. The `BorderColor` property sets the color of the border, while the `BorderStyle` property determines the type of border applied. When `BorderStyle` is set to `FixedSingle`, the `BorderSingle` property can be used to further customize the style of the border.

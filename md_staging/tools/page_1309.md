## Content

### Columns Editor

#### Figure 884: Column Style applied by using the Columns Editor

The Columns Editor window displays settings for columns in a grid or table-like structure. The **Columns List** on the left shows predefined columns such as "Text: Folders," "Text: Size," and "Text: Date Modified." The **Properties** section on the right provides detailed settings for each column, including:

- **Font**: Set to "Verdana, 8.25pt, style."
- **HelpText**: Not defined (empty).
- **HorizontalAlignment**: Center-aligned.
- **Text**: Display the column's label ("Size").
- **TextColor**: Set to "WindowText" (default black color).
- **VerticalAlignment**: Near alignment.
- **BaseStyle**: Inherited from "ColumnStyle1."
- **Visible**: Set to "True."
- **Width**: Specified as 105 units.
- **Show Column Preview**: Enabled.
- **Size Preview**: The editor allows the adjustment of column width, as shown by the "Size" option.

### 3.8.3.7.5 Subitem Styles

#### Subitem Styles

Standard Subitem style in the BaseStyles Collection Editor will be applied by default to all subitems that are added to the tree nodes. The user can also edit the default style settings.

#### Properties

The below properties lets you customize the subitems using the Styles editor.

| TreeNodeAdvSubItemStyleInfo Property | Description |
| --- | --- |
| Alignment | Sets the alignment of the Subitem text. |

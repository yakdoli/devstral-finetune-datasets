# Essential Tools for Windows Forms

This editor lets you add nodes, Subitems for the nodes and customize them using various property settings.

## Overview
- The TreeViewAdv NodeCollection Editor allows adding nodes and customizing their properties.
- The editor can be accessed during design time via the Context Menu, Tasks Windows, or through the MultiColumnTreeView.Nodes property in the property grid.

## Content

### TreeViewAdv NodeCollection Editor
The TreeViewAdv NodeCollection Editor window is displayed below:

![Figure 867: TreeViewAdv NodeCollection Editor](https://example.com/image-url-for-TreeViewAdvNodeCollectionEditor)

#### Features:
1. **Node Management**:
   - **Add Node**: Add a new node to the hierarchical structure.
   - **Add Child**: Add a child node to the selected node.
   - **Remove**: Remove the selected node.
   - **Select added node**: Toggle to automatically select the newly added node.
   - **Sort**: Sort the nodes in a specific order.
   - **OK**: Apply changes and close the editor.
   - **Cancel**: Close the editor without applying changes.

2. **Property Settings**:
   - **Appearance**:
     - **Background**: Node background color (`Solid; Transparent`).
     - **CheckColor**: Color of the checkbox (`ControlText`).
     - **CheckState**: State of the checkbox (`Unchecked`).
     - **Enabled**: Determines if the node is enabled (`True`).
     - **EnabledButtons**: Enables buttons for the node (`True`).
     - **Font**: Font used for text (`Verdana, 8pt`).
     - **Height**: Height of the node (`16`).
     - **HelpText**: Tooltip text displayed on hover.
     - **IntermediateCheckColor**: Color for intermediate check states (`ControlDark`).
     - **Multiline**: Allows text to wrap (`False`).
     - **OptionButtonColor**: Color of option buttons (`White`).
     - **SelectedOptionButton**: Color for selected option buttons (`Black`).
   - **Background**: The background setting for the node.

#### Accessing the Editor
This editor can be accessed using the following options:
- **Through the Context Menu of the control during design time.**
- **Tasks Windows.**
- **Using MultiColumnTreeView.Nodes property in the property Grid.**
- **Command at the bottom of the property grid.**

## API Reference
- **MultiColumnTreeView.Nodes**: Reference to the NodeCollection Editor, which allows management and customization of nodes.

## Code Examples

### C#
```csharp
// Accessing the NodeCollection Editor
```

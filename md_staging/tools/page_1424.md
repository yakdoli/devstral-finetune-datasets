# Essential Tools for Windows Forms

## Overview
- Describes how to use the XPTaskBarBox control to add and configure XPTaskBarItems in a Windows Forms application.
- Explains the process of editing task bar items using the XPTaskBarItem Collection Editor.
- Details the properties and customization options available for each XPTaskBarItem.

## Content

### Using the XPTaskBarBox and XPTaskBarItem Collection Editor

1. **Select the XPTaskBarBox**:
   - Once you have added the XPTaskBarBox to your Windows Forms application, select it in the form designer.

2. **Open the Properties Window**:
   - View the properties of the XPTaskBarBox in the Properties Window.

3. **Access the Items Property**:
   - Locate and click on the "Items" property in the Properties Window. This property is a collection that allows you to define the task bar items.

4. **Open the XPTaskBarItem Collection Editor**:
   - Click the "..." button next to the "Items" property to open the XPTaskBarItem Collection Editor.

5. **Add and Configure Task Bar Items**:
   - In the XPTaskBarItem Collection Editor:
     - **Add Items**: Use the "Add" button to create new XPTaskBarItems.
     - **Specify Properties**: For each item, configure the following:
       - **Text**: The text displayed on the task bar item.
       - **Image**: Use the ImageIndex property to assign an image.
       - **Tag**: Specify a unique Tag property to distinguish one item from another.
     - **Example Configuration**:
       - Name: XPTaskBarItem0
       - Text: XPTaskBarItem0
       - ImageIndex: (none)
       - ItemFont: Microsoft Sans Serif
       - Bounds: 5, 32, 90, 14
       - Enabled: True
       - ForeColor: (none)
       - Visible: True

6. **Save and Close**:
   - Click "OK" to save the changes and close the XPTaskBarItem Collection Editor.

## API Reference

### Properties and Methods

- **XPTaskBarBox Properties**:
  - `Items`: A collection of XPTaskBarItems that defines the task bar items.
  - `ImageList`: Specifies the image list for task bar item images.
  - `ImeMode`: Specifies the input method editor (IME) mode for the control.

- **XPTaskBarItem Properties**:
  - `Text`: The text displayed on the task bar item.
  - `ImageIndex`: The index of the image in the ImageList.
  - `Tag`: A unique identifier for the task bar item.
  - `Bounds`: Defines the size and position of the task bar item.
  - `Enabled`: Determines whether the task bar item is active.
  - `ForeColor`: The foreground color of the task bar item.

## Code Examples

```csharp
// Example: Adding an XPTaskBarItem to the XPTaskBarBox
Syncfusion.Windows.Forms.Tools.XPTaskBarBox xptaskbarbox1 = new Syncfusion.Windows.Forms.Tools.XPTaskBarBox();
Syncfusion.Windows.Forms.Tools.XPTaskBarItem xpTaskBarItem0 = new Syncfusion.Windows.Forms.Tools.XPTaskBarItem();

// Configure the task bar item
xpTaskBarItem0.Text = "TaskBarItem1";
xpTaskBarItem0.ImageIndex = 0; // Assuming you have an image list
xpTaskBarItem0.Tag = "UniqueTag";

// Add the task bar item to the collection
xptaskbarbox1.Items.Add(xpTaskBarItem0);
```

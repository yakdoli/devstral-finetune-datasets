## Essential Tools for Windows Forms

### Overview
- This page discusses the selection modes of the `TreeView` control in Windows Forms.
- It explains how to extend the selection of nodes using the `ExtendSelectionTo` method.

## Content

### Various Selection Modes of TreeView Control

#### Figure 1451: Various Selection Mode of TreeView Control
![Various Selection Mode of TreeView Control](#)
*This figure illustrates three selection modes: Single Level, Multi-Select Same Level, and MultiSelectAll.*

### Extending the Selection

We can extend the selection of the nodes using the `ExtendSelectionTo` method.

#### Method Description

| Methods                          | Parameter                                                                                                                                         |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| ExtendSelectionTo                | Extends the selection of the node to a specified node. <br/> `SelNode` - Represents a `treeNodeAdv`. |
| ExtendSelectionTo (Overloaded)   | `SelNode` - Represents a `treeNodeAdv`. <br/> `removeCurrentMultipleSelection` - Indicates whether or not any current selection should be removed. |

#### Note
*This method will be effective only when the `SelectionMode` is `MultiSelectSameLevel` or `MultiSelectAll`.*

### Code Examples

```csharp
// Extend Selection using below method
this.treeViewAdv1.ExtendSelectionTo(this.treeNode1);
// Overloaded Method
this.treeViewAdv1.ExtendSelectionTo(this.treeNode1, false);
```

## API Reference

### Methods
- **ExtendSelectionTo**
  - **Parameter**: `SelNode` - Represents a `treeNodeAdv`.
  - **Description**: Extends the selection of the node to a specified node.

- **ExtendSelectionTo (Overloaded)**
  - **Parameters**:
    - `SelNode` - Represents a `treeNodeAdv`.
    - `removeCurrentMultipleSelection` - Indicates whether or not any current selection should be removed.

## Overview

- Specifies the configuration and styling options for Wizard page controls in Windows Forms.
- Lists available 3D and 2D border styles, color settings, and event handling for Wizard controls.
- Details events raised by the Wizard control and their functionalities.

## Content

### Border Styles and Properties

| Property         | Description                                                                                                  |
|------------------|--------------------------------------------------------------------------------------------------------------|
| Border3DStyle    | Specifies the 3D border style for Wizard page. The available styles are,                                     |
|                  | • RaisedOuter,                                                                                              |
|                  | • SunkenOuter,                                                                                              |
|                  | • RaisedInner,                                                                                              |
|                  | • Raised,                                                                                                   |
|                  | • Etched,                                                                                                   |
|                  | • SunkenInner,                                                                                              |
|                  | • Bump,                                                                                                     |
|                  | • Sunken,                                                                                                   |
|                  | • Adjust,                                                                                                   |
|                  | • Flat.                                                                                                     |
| BorderColor      | Sets the border color for the Wizard page.                                                                  |
| BorderSides      | Specifies the sides of the control which should have border.                                                |
| BorderSingle     | Specifies the 2D Border style for the Wizard page when BorderStyle property is set to FixedSingle. The available styles are, |
|                  | • Dotted,                                                                                                   |
|                  | • Dashed,                                                                                                   |
|                  | • Solid,                                                                                                    |
|                  | • Inset,                                                                                                    |
|                  | • Outset.                                                                                                   |

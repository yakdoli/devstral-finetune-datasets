#### 3.12.2.1.2.3 Tab Groups

The `RibbonControlAdv` allows to create `TabGroups` using the `TabGroups` property available for the `RibbonControlAdv`.

##### Creating TabGroup Through Designer

1. Clicking the `TabGroup` property will pop up a window like the one below and using this, number of groups can be added and customized using the Color, Name and Visible properties provided to the right of the window.

## API Reference

### Namespaces: Syncfusion.Windows.Forms.Tools

#### RibbonControlAdv
- **Property:** `TabGroups`
  - Type: `System.Collections.Generic.List<TabGroup>`
  - Description: Represents a collection of TabGroups in the RibbonControlAdv.
  - Default: `null`
  - Added: v11.4.0.26

### TabGroup
- **Property:** `Color`
  - Type: `System.Drawing.Color`
  - Description: The color of the TabGroup.
- **Property:** `Name`
  - Type: `System.String`
  - Description: The name of the TabGroup.
- **Property:** `Visible`
  - Type: `System.Boolean`
  - Description: Indicates whether the TabGroup is visible.

---

## Code Examples (C#)

```csharp
// Adding a TabGroup at runtime
using System;
using Syncfusion.Windows.Forms.Tools;

RibbonControlAdv ribbon = new RibbonControlAdv();
TabGroup newTabGroup = new TabGroup();
newTabGroup.Color = System.Drawing.Color.Yellow;
newTabGroup.Name = "New Tab";
newTabGroup.Visible = true;
ribbon.TabGroups.Add(newTabGroup);
```

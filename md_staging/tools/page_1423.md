## Overview
- Demonstrates the process of adding an XPTaskBar Box to a Windows Forms application using the Syncfusion.Windows.Forms.Tools.XPTaskBar control.
- Explains two methods: using verbs and through the Property Grid.

## Content

### Adding XPTaskBar Box through Verbs

The following figure illustrates how to add an XPTaskBar Box using verbs in the Windows Forms designer:

![Figure 966: Adding XPTaskBar Box through Verbs](image:Figure 966)

In the figure:
1. Right-click on the form in the designer.
2. From the context menu, select the "Add Task Box" option.

### Adding XPTaskBar Box through Property Grid

The following figure shows how to add an XPTaskBar Box via the Property Grid:

![Figure 967: Adding XPTaskBar Box through Property Grid](image:Figure 967)

In the figure:
1. Identify the XPTaskBar control in the Property Grid.
2. Click the "Add Task Box" button within the Property Grid to add a task box.

### Step-by-Step Process

#### Using Verbs:
1. Open the Windows Forms application in the designer.
2. Right-click on the form to open the context menu.
3. Select the "Add Task Box" option from the menu.
4. A new task box is added to the form, and its default settings are applied.

#### Using the Property Grid:
1. Select the XPTaskBar control from the designer.
2. Open the Property Grid for the XPTaskBar control.
3. Click the "Add Task Box" button in the Property Grid.
4. The task box is added, and it can be customized further from the Property Grid.

## API Reference

### Namespace: Syncfusion.Windows.Forms.Tools
#### Class: XPTaskBar
Properties:
- AllowDrop: Implements the task box's drag-and-drop functionality.
  - Type: boolean
  - Default: `false`
  - Description: Enables or disables the ability to drag and drop task boxes within the XPTaskBar.

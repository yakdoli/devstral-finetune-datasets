## Code Examples
### C# Example
```csharp
using Syncfusion.Windows.Forms.Tools;

// Customizing ToolstripEx
ToolstripEx myToolstrip = new ToolstripEx();
myToolstrip.BackColor = Color.LightBlue;
myToolstrip.ForeColor = Color.Black;

// Adding Controls
myToolstrip.Add(new ToolstripComboBox("ComboBox"));
myToolstrip.Add(new ToolstripLabel("Label"));
myToolstrip.Add(new ToolstripSplitButton("SplitButton"));
```

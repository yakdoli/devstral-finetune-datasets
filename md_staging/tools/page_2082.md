# Essential Tools for Windows Forms

## Overview

- This page explains how to create and control tab groups with MDI children in a single form using Syncfusion Windows Forms tools.

### Content

The figure below demonstrates TabGroups with MDI Children in a Single Form:

**Figure 1399: TabGroups with MDI Children in a Single Form**

The below topics will guide you on how to create tab groups and set borders for the tab groups.

**3.18.2.7.1 Creating Tab Groups**

The below given steps will guide you to create and control tab groups.

1. To the application, add Tools.Windows and Shared.Base Syncfusion assemblies.
2. Add 2 more forms and rename them as TabGroup1_Form and TabGroup2_Form. (The application now contains three forms (i.e.) Form1, TabGroup1_Form, and TabGroup2_Form).
3. In Form1, add the namespace Syncfusion.Windows.Forms.Tools.

## Essential Tools for Windows Forms

### Color Schemes in OfficeForm

#### Background Color for Office2007 Form

The background of the Office2007 form can be set to match the color scheme applied to the form. To make this effective, set the `UseOffice2007SchemeBackColor` property to `true`.

Figure 1109: Blue, Silver, and Black ColorSchemes in OfficeForm

```
this.UseOffice2007SchemeBackColor = true;
```

### Background Color for Office2007 Form

The background of the Office2007 form can be same, as the color scheme applied to the form. Set `UseOffice2007SchemeBackColor` property to `true` to make this effective.

```csharp
// C#
this.UseOffice2007SchemeBackColor = true;
```

```vb.net
' VB.NET
Me.UseOffice2007SchemeBackColor = True
```

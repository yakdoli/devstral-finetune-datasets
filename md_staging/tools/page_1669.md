# Essential Tools for Windows Forms

## Overview
- Demonstrates customization of the `DigitalClock` control, specifically focusing on its shape and color.
- Explains how to change the foreground color of the `DigitalClock` using the `ForeColor` property.

## Content

### 3.11.6.4.1.2 Color customizations

#### 3.11.6.4.1.2.1 Foreground color

The foreground color for the DigitalClock can be changed using the `ForeColor` property. This color will be reflected in the text of the control.

#### Code Examples

```csharp
this.clock1.ForeColor = System.Drawing.Color.Yellow;
```

```vb
Me.clock1.ForeColor = System.Drawing.Color.Yellow
```

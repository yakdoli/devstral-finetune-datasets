## Essential Tools for Windows Forms

### Overview
- Node Level Styles can be added and customized using the BaseStyles Collection Editor.
- These styles are applied by default to the first level nodes.
- A specific node can also be styled using the `TreeNodeAdv.BaseStyle` property.

## Content

### Node Level Styles

The BaseStyles Collection Editor allows you to define and edit node level styles. Below is how to add and configure a style for NodeLevel1.

#### Figure 1476: Node Level Styles added by using the BaseStyles Collection Editor

![](https://via.placeholder.com/100x100.png?text=Figure+1476)

- **Properties for style NodeLevel1:**
  - **Appearance:**
    - **Background:**
      - **Style:** Gradient
      - **BackColor:** SandyBrown
      - **ForeColor:** Cornsilk
      - **GradientStyle:** BackwardDiagonal
      - **GradientColors:** Syncfusion.Drawing.Brush
    - **CheckColor:** ControlText (Unchecked)
    - **ClosedImgIndex:** 1
    - **Enabled:** True

By default, this style will apply to the first level nodes as shown in the next image.

#### Figure 1477: NodeLevelStyle applied by default to the First Level Nodes

![](https://via.placeholder.com/100x100.png?text=Figure+1477)

This demonstrate the application of the NodeLevelStyle to the parent nodes (`Parent1`, `Parent2`, `Parent3`) and their respective children (`Node1`, `Node2`, `Node3`).

### Applying a Style to a Specific Node

You can customize the style for a specific node by using the `TreeNodeAdv.BaseStyle` property. This allows you to apply the defined style to a particular node instead of the default level style.

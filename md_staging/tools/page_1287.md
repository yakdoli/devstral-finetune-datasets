## API Reference

### Properties
- **AccessibleRole**: Specifies the role of the control for accessibility purposes.
- **BackgroundColor**: Sets the background color of the control.
- **Border3DStyle**: Defines the 3D style of the control's border.

### Methods
- **AddNode**: Adds a new node to the tree view.
- **AddColumn**: Appends a column to the grid.

## Essential Tools for Windows Forms

### ThemeDataEnabled

| ThemeDataEnabled | Specifies whether the CurrencyTextBox control uses XP themes, when BorderStyle is set to Fixed3D. |
| --- | --- |

#### Code Examples

##### C#

```csharp
this.currencyTextBox1.ThemesEnabled = true;
```

##### VB.NET

```vb
Me.currencyTextBox1.ThemesEnabled = True
```

### Figure 516: BorderStyle = "Fixed3D"; ThemesEnabled = True / False

![Figure 516: BorderStyle = "Fixed3D"; ThemesEnabled = True / False](https://i.imgur.com/image.png)
#### Overview

- 1. ThemeDataEnabled allows configuring the visual theme for a CurrencyTextBox control.
- 2. BorderStyle influences the border appearance of the control.
- 3. Example demonstrates enabling themes in both C# and VB.NET.

### Subset of Border Styles

#### 3.5.8.6.5.2 Border Styles

The below properties describe various properties available to set the border for the CurrencyTextBox control.

| CurrencyTextBox Properties | Description |
| --- | --- |
| BorderStyle | Sets the style of the border. The options include:<br>- FixedSingle,<br>- Fixed3D and<br>- None (Default). |
| Border3DStyle | Sets 3D border style of the CurrencyTextBox when the BorderStyle is in Fixed3D. The options include:<br>- Raised,<br>- RaisedOuter,<br> |

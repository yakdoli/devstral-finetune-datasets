# Essential Tools for Windows Forms

## Overview
- Describes the bar properties and their functions in Windows Forms.
- Explains how to customize toolbars and their behavior at design and run time.
- Highlights options for hiding, reordering, and customizing bar items.

## Content

### Bar Properties

| Bar Property                       | Description                                                                                                                                                                                                                                                            |
|------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Items**                          | Collection of bar items representing the menu items in the toolbar. This invokes the customize dialog at design time.                                                                                                                                                    |
| **AllowCustomizing**               | Indicates whether dragging and dropping of bar items into the bar is allowed or not, at run time.                                                                                                                                                                   |
| **AllowItemsReorderOnShrunk**      | When the toolbar shrunk to show more bar items, the bar items will be automatically reordered, when `AllowItemsReorderOnShrunk` property is set to true. The default value is true.                                                                                 |
| **Bar Styles**                     | Specifies the `BarStyle`.                                                                                                                                                                                                                                              |
| **AllowHiding**                    | Specifies if the toolbar can be hidden by the user at run time. By default, this property is true. When set to false, the user will not be able to hide the toolbar at run time, as the checkbox will be disabled at run time. |

## API Reference

### Properties

- **Items**: Represents the collection of bar items in the toolbar.
- **AllowCustomizing**: A Boolean property to enable or disable customization of bar items at run time.
- **AllowItemsReorderOnShrunk**: A Boolean property to allow automatic reordering of bar items when the toolbar is shrunk.
- **BarStyles**: Specifies the style of the bar.
- **AllowHiding**: A Boolean property to allow or disallow the toolbar from being hidden by the user.

## Code Examples

### Example of Setting `AllowHiding` Property

```csharp
// Disabling the ability to hide the toolbar
Bar bar = new Bar();
bar.AllowHiding = false;
```

### Example of Customizing AllowItemsReorderOnShrunk

```csharp
Bar bar = new Bar();
bar.AllowItemsReorderOnShrunk = true;
```

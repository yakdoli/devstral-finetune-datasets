## Essential Tools for Windows Forms

### Overview

- Describes the essential tools and features for Windows Forms.
- Focuses on style settings and properties for the `ToolStripEx` control.
- Includes details on `RightToLeft` and `BorderSettings` properties.

### Content

#### 3.12.2.1.1.2.2 Style Settings

This section discusses the style settings available for the `ToolStripEx`.

##### Border Settings

The following table explains the properties available for border settings:

| Property    | Description                                                                 |
|-------------|-----------------------------------------------------------------------------|
| `RightToLeft` | Indicates whether the item should draw right to left for RTL languages. |

---

| Property    | Description                                                                 |
|-------------|-----------------------------------------------------------------------------|
| `BorderStyle` | Sets the border style for the control. The options are, <br> - None, <br> - Etched, <br> - StaticEdge. |

##### Code Examples

- **C#**:
  ```csharp
  this.toolstripEx1.BorderStyle = ToolstripBorderStyle.Etched;
  ```
  
- **VB.NET**:
  ```vb
  Me.toolstripEx1.BorderStyle = ToolstripBorderStyle.Etched
  ```

# Essential Tools for Windows Forms

## Overview
- This page discusses the essential tools and dashboard for Windows Forms in Syncfusion Essential Studio.
- It highlights how users can access and interact with various sample applications for WinForms development.

## Content

### Main Dashboard Features
The dashboard provides a central interface for managing and interacting with Syncfusion's Windows Forms tools and samples. The interface includes the following key sections:

#### User Interface
- **ASP.NET MVC**, **ASP.NET**, **Windows Forms**, **WPF**, **Silverlight**, **Windows Phone**, **Mobile MVC** are listed as menu options for easy navigation.

#### Highlighted Features
- **User Interface Edition for Windows Forms**: Integrates UI edition into applications with ease.
- **9 Comprehensive Products**: Contains over 100 components to ease development and improve productivity.

#### Sample Accessibility
The dashboard allows users to explore and interact with sample applications in multiple ways:

#### Navigation
- **Sales**, **Contact Support**, **www.syncfusion.com**, **Release Notes**

#### Administration Access
- **Administrator Level Access Required**

### Sample Exploration

#### Default Display
- **Tools samples** will be displayed by default.

#### Note
- **You can view the samples in any of the following three ways:**

1. **Run Samples** - Click to view locally installed samples.
2. **Online Samples** - Click to view online samples.
3. **Explore Samples** - Explore BI Web samples on disk.

- The dashboard includes options for **Download Source Code**, **Online Documentation**, and **Release Notes** for users to access further information.

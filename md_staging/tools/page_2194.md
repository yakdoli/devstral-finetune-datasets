# Essential Tools for Windows Forms

## Overview
- Demonstrates adding "Edit", "Copy", and "Cut" menu items to the Context Menu in TreeViewAdv.
- Explains editing functionality using the "Edit" menu item and the `BeginEdit` method.

## Content

### Editing the nodes using "Edit" Menu Item

#### Figure 1452: "Edit", "Copy", and "Cut" menu items added to the Context Menu
![Figure 1452: "Edit", "Copy" and "Cut" menu items added to the Context Menu](attachment:Figure_1452.png)

We can include editing functionality when you click the Edit menu item using the EditItem_Click event. In the below example, it calls the BeginEdit method and begins editing the node that is selected.

#### treeViewAdv Methods

| treeViewAdv Methods           | Description                                                                                      |
|-------------------------------|--------------------------------------------------------------------------------------------------|
| BeginEdit                     | Edits the selected node.                                                                        |
| BeginEdit(Overloaded)         | Edits the specified node that is passed as the parameter. The parameter is,<br>node - Indicates the particular node to edit. |

## API Reference

### Namespace: Syncfusion.Windows.Forms.Tools

#### Class: TreeViewAdv

##### Methods
- `BeginEdit()`: Edits the selected node.
- `BeginEdit(TreeNodeAdv node)`: Edits the specified node that is passed as the parameter.

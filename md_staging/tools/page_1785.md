# Essential Tools for Windows Forms

## Overview

- Provides details about progressbar manipulation using ToolstripProgressBar.
- Explains Tooltip settings and their configurations.
- Discusses Right-to-Left (RTL) support for language-specific rendering.
- Focuses on the ToolStripCheckBox and its functionalities.

## Content

### Progressbar Manipulation

| PerformStep() | Advances the current position of the progressbar by the value specified in ToolstripProgressBar.Step property. |
| --- | --- |

### Tooltip Settings

| Property   | Description                                                                                                                                                                                                                                                                                                                                 |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| AutoToolTip | - When set to true, will display the text set in the Text property as the item's tooltip.<br>- When set to false, will display the text set in the ToolTipText property.                                                                                                     |
| ToolTipText | Sets the text for the tooltip when AutoToolTip is set to false.                                                                                                                                                                                                               |

### RTL Support

| Property          | Description                                                                                                                     |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| RightToLeft       | Indicates whether the item should draw right to left for RTL languages.                                                          |
| RightToLeftLayout | Indicates whether the control layout is right to left.                                                                          |

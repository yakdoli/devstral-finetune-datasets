#### 3.12.1.2.2.1 Appearance Settings

This section discusses various appearance and behavior settings of the Ribbon form.

### Appearance Settings

The appearance of the ribbon form can be controlled using the below properties.

#### Property Summary

| Property       | Description                                                                                          |
|----------------|------------------------------------------------------------------------------------------------------|
| **Appearance** | Sets the appearance of the form. The values are, <br> - Normal <br> - Office2007 (Default) <br> - Office 2010 |
| **ColorScheme** | Specifies the office color scheme of the Ribbon form. The color schemes are, <br> - Blue, <br> - Black, <br> - Silver, <br> - Managed (Default). |

---

## API Reference

### Appearance Property

- **Description**: Sets the appearance of the form.
- **Possible Values**:
  - Normal
  - Office2007 (Default)
  - Office 2010

### ColorScheme Property

- **Description**: Specifies the office color scheme of the Ribbon form.
- **Possible Values**:
  - Blue
  - Black
  - Silver
  - Managed (Default)

---

## Code Examples

### C# Example

```csharp
// Setting appearance and color scheme
this.RibbonControlAdv.Appearance = Syncfusion.Windows.Forms.Tools.RibbonAppearance.Normal;
this.RibbonControlAdv.ColorScheme = Syncfusion.Windows.Forms.Tools.RibbonColorScheme.Blue;
```

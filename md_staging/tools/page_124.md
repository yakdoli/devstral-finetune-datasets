# Essential Tools for Windows Forms

## Overview
- Demonstrates the DockingManager feature in Windows Forms.
- Explains how to transform a ListBox control into a docking window.
- Provides a detailed look at the properties of docked controls.

## Content

### DockingManager Feature

#### Transforming a ListBox Control into a Docking Window
The DockingManager allows the transformation of a ListBox control into a docking window, as shown in **Figure 43**.

![Figure 43: ListBox control transformed into a Docking Window](https://i.imgur.com/example.png)

#### Docking-Specific Properties
DockingManager provides specific properties to docked controls, which are listed under the Syncfusion Docking category of those controls. These properties let you:
- Specify icons for the caption.
- Edit the caption labels.
- Auto hide the controls and more.

#### Properties of Docked Control

The image below shows the properties of a docked ListBox control. These properties are available under the **Syncfusion Docking** category.

![Figure 44: Properties of Docked Control](https://i.imgur.com/example.png)

#### Appearance Properties
DockingManager also offers extensive appearance properties, whose settings will be applied to all docked controls.

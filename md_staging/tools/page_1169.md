##### Grouping Bar Items
**Design Time**

XPMenus allows you to group certain bar items using the "Begin a Group" option in the designer, code, and programmatically.

This feature enables developers to organize related bar items into logical groups for enhanced usability and visual clarity.

![Bar Item Without Group](UnGroupedBar.png "Bar Item without Grouping")

The following example demonstrates how to group bar items programmatically:

```csharp
// Example of grouping bar items programmatically
using Syncfusion.Windows.Forms.Tools.XPToolbarMainItem;

XPToolbarMainItem toolbarMainItem = new XPToolbarMainItem("Group");
toolbarMainItem.AddRange(new BarItem[] {
    barItem1, barItem2, barItem3
});

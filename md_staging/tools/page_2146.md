## Content

### 3.21.1 Features Overview

The TreeView package provides an advanced **TreeViewAdv** control with numerous styles and extensive customization capabilities. The features of the control are listed below.

#### Features

- **Look and Feel** - The control can be applied with impressive look and feel by providing different options for the users to customize the appearance of the control.

### Notes:

- The TreeViewAdv control offers advanced customization options compared to the standard TreeView control.
- The control includes various styles and themes to match different user interface requirements.
- The appearance of the TreeViewAdv can be customized to enhance user experience.

## API Reference

### Classes and Members

- **TreeViewAdv** - Extends the functionality of the TreeView control.
  - Properties:
    - Appearance: Customizes the visual style of the tree nodes.
    - Nodes: Provides access to the collection of nodes in the tree.
  - Methods:
    - PopulateNode: Used to populate node data.
    - ExpandAll: Expands all visible nodes.
  - Events:
    - NodeMouseClick: Triggered when a mouse click occurs on a node.

### Code Examples

```csharp
TreeViewAdv treeView = new TreeViewAdv();
treeView.PopulateNode += (sender, args) =>
{
    // Populate node data here
};
treeView.ExpandAll();
```

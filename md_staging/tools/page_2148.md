# Essential Tools for Windows Forms

## Overview
- TreeViewAdv provides support for Office2007 Scrollbars with all three color schemes.
- Detailed features include tree nodes tooltips, drag and drop functionalities, and contextual menu options.
- Flexibility in style architecture for defining custom node styles.

## Content

### Office2007 ScrollBars

TreeViewAdv provides support for Office2007 Scrollbars with all the three color schemes.

![Office 2007 ScrollBars](./resources/Figure_1430_Office_2007_ScrollBars.png)
**Figure 1430: Office 2007 ScrollBars**

#### Blue
- The left-side scrollbar with a blue color scheme.
#### Silver
- The middle-side scrollbar with a silver color scheme.
#### Black
- The right-side scrollbar with a black color scheme.

### ToolTip
Allows us to show and hide tooltips for the nodes wherever necessary.

### Drag and Drop
Provides support for advanced drag and drop of the nodes.

### Context Menus
TreeViewAdv has the ability to associate context menus with the option to show and hide wherever necessary. This pops up when the right mouse button is clicked on a tree node.

### Styles Architecture
Supports a flexible style architecture to let users define the styles for nodes at different levels of the Tree. It helps the users to specify the styles for a specific node or for a class of nodes.

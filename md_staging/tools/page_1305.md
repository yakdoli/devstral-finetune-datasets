### Node Level Style

#### Setting a Common Style for all the Nodes in a Level

TreeView lets you add node level styles using the BaseStyles Collection Editor. By default, this style will be applied to all the nodes in the Level1 for NodeLevel1 style, Level2 for NodeLevel2 style, and so on.

#### Figure: BaseStyles Collection Editor

<figure>
  <img alt="BaseStyles Collection Editor interface" src="image.png" />
  <figcaption>
    The editor interface displays two main sections:
    - **Base Styles**: Lists existing styles like "Standard - Column", "NodeLevel1", "BaseStyle1", etc.
    - **Properties for style NodeLevel1**: Detailed appearance settings for NodeLevel1, including:
      - Background: Gradient; Vertical; Papaya
      - CheckColor: ControlText
      - CheckState: Unchecked
      - Enabled: True
      - EnabledButtons: True
      - Font: Microsoft Sans Serif, 8.25pt
      - Height: 16
      - IntermediateCheckColor: ControlDark
      - Multiline: False
      - OptionButtonColor: White
  </figcaption>
</figure>

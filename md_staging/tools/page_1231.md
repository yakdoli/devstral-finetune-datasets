# Essential Tools for Windows Forms

## Overview
- The document describes the supported GUI styles for Windows Forms, including XP, Office 2003, Office XP, and VS2005 styles.

## Content

### Supported GUI styles

The supported Visual styles are:

- **Office 2003 look**: This can be enabled by setting `Style` to `Office2003`.

![Figure 838: Office 2003 Visual Style](https://example.com/office2003_style.png)

Figure 838: Office 2003 Visual Style

- **OfficeXP look**: This can be enabled by setting `Style` to `OfficeXP`.

![Figure 839: Office XP Visual Style](https://example.com/officexp_style.png)

Figure 839: Office XP Visual Style

- **VS2005 look**: This can be enabled by setting `Style` to `VS2005`.

![Figure: VS2005 look](https://example.com/vs2005_style.png)

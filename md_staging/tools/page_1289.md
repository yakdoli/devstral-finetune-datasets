#### Content

##### Tree Node Adv Subitem Collection Editor
The TreeNodeAdvSubItem Collection Editor provides a graphical interface to manage the subitem properties of a tree node in a Windows Forms application. This editor allows developers to:

- Add or remove subitems.
- Adjust the appearance properties such as alignment, background, and text color.
- Modify border styles and visibility settings.

##### Styles Editor
- **Purpose**: The Styles Editor enables administrators or developers to customize the look and feel of the TreeNodeAdv controls.
- **Functionality**: It provides the flexibility to define default styles and to add new styles that can be applied to nodes and subitems.
- **Property Settings**: Detailed settings for each style, including appearance properties and inheritance, are discussed in the **Styles Architecture** section, which outlines the underlying structure and mechanisms governing these styles.

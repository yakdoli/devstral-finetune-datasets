## Essential Tools for Windows Forms

### Overview
- Details the properties of the ToolstripLabel control.
- Explains how to configure text display, font, color, alignment, direction, and relative image placement.

### Content

#### Property Descriptions

| Property              | Description                                                                                                                     |
|-----------------------|---------------------------------------------------------------------------------------------------------------------------------|
| Text                  | Sets the text for the ToolstripLabel. This text will be displayed only if the `DisplayStyle` is `Text` or `ImageAndText`.       |
| Font                  | Sets the font style for the display text.                                                                                     |
| ForeColor             | Sets the foreground color for the display text.                                                                               |
| TextAlign             | Specifies the alignment of the text in the item. The options are:                                                                |
|                       | - TopLeft,                                                                                                                     |
|                       | - TopCenter,                                                                                                                   |
|                       | - TopRight,                                                                                                                    |
|                       | - MiddleLeft,                                                                                                                  |
|                       | - MiddleCenter,                                                                                                                |
|                       | - MiddleRight,                                                                                                                 |
|                       | - BottomLeft,                                                                                                                  |
|                       | - BottomCenter,                                                                                                                |
|                       | - BottomRight.                                                                                                                 |
| TextDirection         | Specifies the direction of drawing the text. The options are:                                                                  |
|                       | - Horizontal - Text is placed horizontally,                                                                                 |
|                       | - Vertical90 - Text is placed vertically,                                                                                 |
|                       | - Vertical270 - Text is placed vertically at 270 degrees.                                                                 |
| TextImageRelation     | Specifies the relative location of the image to the text on the item. The options are:                                           |
|                       | - Overlay - Image and text share the same space in the control,                                                            |
|                       | - ImageAboveText - Image will be placed above the text,                                                                 |
|                       | - TextAboveImage - Text will be placed above the image,                                                                 |
|                       | - ImageBeforeText - Image will be placed before the text.                                                                 |

### API Reference

#### Properties
- **Text**: String property for setting the text to be displayed.
- **Font**: Font property for setting the font style.
- **ForeColor**: Color property for setting the foreground color.
- **TextAlign**: Enumeration property for setting text alignment.
- **TextDirection**: Enumeration property for setting text direction.
- **TextImageRelation**: Enumeration property for setting the relationship between text and image.

### Code Examples

#### Example: Configuring ToolstripLabel
```csharp
using System;
using System.Drawing;
using System.Windows.Forms;

class Program
{
    static void Main(string[] args)
    {
        // Create a ToolstripLabel
        ToolStripLabel label = new ToolStripLabel();

        // Set properties
        label.Text = "Sample Text";
        label.Font = new Font("Arial", 12, FontStyle.Bold);
        label.ForeColor = Color.Blue;
        label.TextAlign = ContentAlignment.MiddleLeft;
        label.TextDirection = TextDirection.Horizontal;
        label.TextImageRelation = ToolStripTextImageRelation.Overlay;

        // Add label to the toolbar
        ToolStrip toolbar = new ToolStrip();
        toolbar.Items.Add(label);

        // Display the form
        Form form = new Form();
        form.Controls.Add(toolbar);
        Application.Run(form);
    }
}
```

## Essential Tools for Windows Forms

### Setting ToolTip Text using ToolTipInfo.Body.Text Property

#### Overview
- Demonstration of how to set the text displayed in a ToolTip using the `ToolTipInfo.Body.Text` property.

#### Content

**Figure 832: Setting ToolTip Text using ToolTipInfo.Body.Text Property**

![](images/screenshot_2023-10-10_194501_image.png)

The ToolTip Editor window displays an example of setting a ToolTip for a control labeled "Save." The properties related to the ToolTip are visible on the right side of the window. The "Text" property is set to "Save," indicating the text to be displayed when the ToolTip is triggered.

**C# Code Example:**
```csharp
toolTipInfo1.Body.Text = "Save";
this.superToolTip1.SetToolTip(this.barItem2, toolTipInfo1);
```

**VB.NET Code Example:**
```vb
toolTipInfo1.Body.Text = "Save"
Me.superToolTip1.SetToolTip(Me.barItem2, toolTipInfo1)
```

This code snippet demonstrates how to programmatically set the `Body.Text` property of a `ToolTipInfo` object to "Save" and then associate this ToolTip with a specific `barItem2` control using the `SetToolTip` method.

### API Reference

**Properties Used:**
- `ToolTipInfo.Body.Text`: Sets the text to be displayed in the ToolTip.
- `superToolTip1.SetToolTip`: Method to assign the ToolTip to a control.

**Parameters:**
- `barItem2`: The control to which the ToolTip is to be assigned.
- `toolTipInfo1`: The `ToolTipInfo` object containing the ToolTip properties.

### Code Examples

The examples provided illustrate the use of the `Body.Text` property to set the text content of a ToolTip. The code in both C# and VB.NET is intended to be placed within the appropriate event handler or initialization method where the ToolTip needs to be defined and assigned to the control.

## Essential Tools for Windows Forms

### Overview
- Demonstrating how to associate a `ContextMenuStrip` with a `RichTextBox` in Windows Forms.
- Highlighting the design-time configuration through property settings.
- Sample C# and VB.NET code snippets to assign the `ContextMenuStrip`.

### Content

#### Design-Time Association

The properties panel is used to configure the `RichTextBox` to associate it with a `ContextMenuStripEx` control. The following image illustrates the process:

![Figure 1247: Associating ContextMenuStripEx to RichTextBox](https://i.imgur.com/jZK8X.png)

#### Code Examples

The `ContextMenuStrip` can be programmatically linked to the `RichTextBox1` control using the following code:

```csharp
// C#
this.richTextBox1.ContextMenuStrip = this.contextMenuStripEx1;
```

```vb
' VB.NET
Me.richTextBox1.ContextMenuStrip = Me.contextMenuStripEx1
```

#### Runtime Behavior

At runtime, when the user right-clicks the control, the associated menu items will be displayed as shown in the example below:

![Runtime Context Menu Display](https://i.imgur.com/jZK8X.png)

## Summary

By associating a `ContextMenuStrip` with a `RichTextBox`, developers can provide context-sensitive menu functionality for enhanced user interaction in Windows Forms applications. The process can be managed either through the designerâ€™s properties panel or dynamically through code.

## Essential Tools for Windows Forms

### Overview
- Introduces the `EditableList` control, which allows in-place editing when double-clicked.
- Describes the key features and functionality of the `EditableList` control.
- Explains the process to create and use the `EditableList` control in a user application.

## Content

### Features

#### 3.5.8.7.1 Features
The `EditableList` displays a list control that allows in-place editing when the user double-clicks on the list control. It contains the following features:

- **Appearance and Behavior Properties**: The `EditableList` listbox provides various appearance and behavior properties.
- **AutoCompletion Support**: The `EditableList` supports AutoCompletion from the list.

### Creating Editable List

#### 3.5.8.7.2 Creating Editable List
To use an `EditableList` control in your application, all you need to do is drag-and-drop the `EditableList` control from the controls toolbox onto your form. To add items, expand the `ListBox` node and open the **String Collection Editor**.

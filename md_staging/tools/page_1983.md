# Essential Tools for Windows Forms

## Overview
- This section provides detailed instructions on customizing the TabItems in TabControlAdv.
- It explains how to rename TabItems at runtime using the LabelEdit property.
- Includes examples in C# and VB.NET for programmatically enabling the renaming feature.

## Content

### 3.17.4.1.1 TabItems Customization

TabItems can be customized using the properties given below.

#### Renaming TabItems

TabControlAdv comes with the renaming functionality similar to MS Excel. Users can edit TabControlAdv's text at run-time using the **LabelEdit** property, which has to be set to **True**.

| TabControlAdv Property | Description |
|-------------------------|-------------|
| LabelEdit              | Specifies whether the text of the tab item is editable. Default value is **False**. |

To bring the text of the tab item to the edit mode, the following can be done:

- Select the text of the tab item to be edited and right-click on the tab to bring the text to edit mode. The text can now be edited and it can be saved by pressing the Return key.
- Also, double-clicking on a tab makes the text change to edit mode.

![Figure 1346: Renaming the Tab Item](image1.png)

After editing the text, to come out of the edit mode, press the Enter key or click the Left Mouse button.

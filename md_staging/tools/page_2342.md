## Overview
- This section guides you through various options available in the designer to manage pages in a wizard control, including adding, removing, navigating to previous, and moving to the next page.

## Content

### 3.22.1.4.4.1 Options to Add Page, Remove Page, Previous page and Next Page

This section will guide you with various options available in the designer to add page, remove page, go to previous page and next page.

#### Using Property Grid

The Property Grid provides a convenient way to manage pages in a wizard control.

- **Adding Page:**
  ![Property Grid View](https://example.com/property-grid-image.jpg)  
  *Figure 1509: Adding Page Using "WizardPages" Property and "Add Page" Command*

  - Select the wizard control in the designer.
  - In the Property Grid, locate the **WizardPages** property.
  - Click the "..." button next to **WizardPages** to edit the array of pages.
  - Use the commands **Add Page**, **Remove Page**, **Previous Page**, and **Next Page** to manage the pages.

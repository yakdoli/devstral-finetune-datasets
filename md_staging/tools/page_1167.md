# Essential Tools for Windows Forms

In the designer, right-click on the bar item which you want to delete and select "Delete" option from the Context Menu.

**Figure 782: Delete option in the Context Menu**

You can remove the BarItem from a submenu using any of the below two methods also.

## BarItem Methods

| BarItem Methods | Description |
| --- | --- |
| Remove | Removes the first occurrence of the specific object.<br><br>obj - System.Object to remove from System.Collections.ArrayList. |
| RemoveAt | Removes the bar item from the ParentBarItems Collection based on the Bar item index(index).<br>The parameter is,<br><br>index - Index of the bar item. |

## Code Examples

```csharp
[C#]
this.parentBarItem1.Items.Remove(this.barItem1);
or
this.parentBarItem1.Items.RemoveAt(1);
//where '1' refers to the index of the BarItem in its parentBarItems
```

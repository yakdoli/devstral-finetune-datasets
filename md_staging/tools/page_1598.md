# Essential Tools for Windows Forms

## Overview
- This section discusses the customization of panel size and spacing properties in Windows Forms applications using the `StatusBarAdv` control.
- It explains how to set the `CustomLayoutBounds` property to define a custom rectangle for panel display.

---

## Panel Size

### Overview
The rectangle used to display the panels can be customized using the provided property.

### Property Details
The `CustomLayoutBounds` property is used to indicate a custom rectangle that the layout will use to display the panels.

| StatusBarAdv Property       | Description                                                   |
|-----------------------------|---------------------------------------------------------------|
| `CustomLayoutBounds`        | Indicates a custom rectangle that the layout will use to display the panels. |

### Setting Programmatically
It can be set programmatically using the following code snippets:

#### C#
```csharp
this.statusBarAdv1.CustomLayoutBounds = new System.Drawing.Rectangle(5, 2, 100, 20);
```

#### VB.NET
```vb
Me.statusBarAdv1.CustomLayoutBounds = New System.Drawing.Rectangle(5, 2, 100, 20)
```

## API Reference
### Properties
- `CustomLayoutBounds`: Used to set a custom rectangle for the panels displayed in the layout.

### Method Signature
- `CustomLayoutBounds`: `System.Drawing.Rectangle`

### Returns
- A `Rectangle` object defining the custom bounds for the panels.

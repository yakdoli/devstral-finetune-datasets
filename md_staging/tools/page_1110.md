### Summary
- This section explains how to add BarItems in Windows Forms using the `Toolbars` and `Commands` interface.
- After inserting BarItems, you can modify their properties in the VS .NET design-time property grid.
- The process involves selecting the BarItem from the Commands list and accessing its properties to customize its behavior and appearance.

### API Reference
- **BarManager**: The main control for managing multiple toolbars and menu strips within the Windows Forms application.
- **BarItem**: Represents a single item within a toolbar or menu strip.
- **Properties**:
  - **Text**: The text displayed on the BarItem.
  - **Visible**: Determines whether the BarItem is visible.
  - **Tag**: A property to hold custom information about the BarItem.

### Code Examples
#### C#
```csharp
// Creating a new BarItem
Syncfusion.Windows.Forms.Tools.XPMenus.BarItem newItem = new Syncfusion.Windows.Forms.Tools.XPMenus.BarItem();
newItem.Text = "New Item";
newItem.Visible = true;

// Adding the BarItem to the BarManager
mainFrameBarManager1.Items.Add(newItem);
```

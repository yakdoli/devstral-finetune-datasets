### 3.5.4.5.3.2 ColorPickerUIAdv Appearance

This section covers the below topics:

#### 3.5.4.5.3.2.1 Border Styles

The border for the ColorPickerUIAdv control can be Fixed Single, Fixed3D, or None, which is set using the `BorderStyle` property. By default, the border style is None.

```csharp
[C#]
this.colorPickerUIAdv1.BorderStyle =
System.Windows.Forms.BorderStyle.Fixed3D;
```

```vbnet
[VB.NET]
Me.colorPickerUIAdv1.BorderStyle =
System.Windows.Forms.BorderStyle.Fixed3D
```

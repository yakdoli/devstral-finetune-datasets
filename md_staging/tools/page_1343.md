# Essential Tools for Windows Forms

## Overview

- GroupBar items with customizable background and foreground colors.
- Methods to reset group bar properties to their default values.

## Content

### Code Samples

```csharp
this.groupBar1.HeaderBackColor = System.Drawing.Color.LavenderBlush;
this.groupBar1.HeaderForeColor = System.Drawing.Color.Silver;
```

```vb
Me.groupBar1.HeaderBackColor = System.Drawing.Color.Red
Me.groupBar1.HeaderForeColor = System.Drawing.Color.Maroon
```

### Resetting Properties

The methods given below are used to reset the above properties.

| Methods                | Description                                                       |
|------------------------|-------------------------------------------------------------------|
| ResetHeaderFont        | Resets the HeaderFont property to its default value.            |
| ResetHeaderBackcolor   | Resets the HeaderBackColor property to its default value.      |
| ResetHeaderForeColor   | Resets the HeaderForeColor property to its default value.      |

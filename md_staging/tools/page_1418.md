### Overview

This section provides an overview of the XPTaskBar control and its features. Here is a list of the features included in the XPTaskBar:

#### Features

- **Office2007 Style for XPTaskBarBox**: The XPTaskBarBox in the XPTaskBar control follows the Office2007 style.
  
- **Custom Colors**: Custom colors can be applied for the XPTaskBar control. For more details, refer to the **Visual Style** topic.

#### Behavior Settings

- **Drag-and-Drop Functionality**: The XPTaskBar includes drag-and-drop functionality, which allows for dragging XPTaskBar items within a XPTaskBar Box or between different XPTaskBar Boxes.

# Essential Tools for Windows Forms

## Border Settings

The `BorderStyle` property of `TabControlAdv` can be used to set the border styles for the `TabPages`.

The three types of border styles are given below.

- FixedSingle
- Fixed 3D
- None

### TabControlAdv Property: BorderStyle

| TabControlAdv Property | Description                                                                                   |
|-------------------------|-----------------------------------------------------------------------------------------------|
| BorderStyle            | Gets / sets the border styles for the tabpages. It includes the following styles:<br>• FixedSingle<br>• Fixed3D<br>• None |

# Essential Tools for Windows Forms

## Overview
- Describes RTL (Right-To-Left) support in ContextMenuStripEx.
- Explains adding ToolStripItems to ContextMenuStripEx.
- Details the unique properties of ToolStripMenuItem.

## Content

### 3.12.2.5.5 RTL Support

ContextMenuStripEx has RTL support which can be enabled using the `RightToLeft` property.

#### Code Examples
```csharp
this.contextMenuStripEx1.RightToLeft = Yes;
```

```vb
Me.contextMenuStripEx1.RightToLeft = Yes
```

**Figure 1257**: RightToLeft = "Yes"

### 3.12.2.5.6 ToolStripsItems

The following are the ToolStripItems which can be added as menu items to the ContextMenuStripEx control.

#### 3.12.2.5.6.1 ToolStripItem - Menuitem

The ToolStripMenuItem supports all the general properties of a ToolStripItems. This section will walk you through the unique properties with their description for the ToolStripMenuItem.

#### Unique Property: Checked

| Property    | Description                                                                                                          |
|-------------|----------------------------------------------------------------------------------------------------------------------|
| Checked     | Indicates whether the item is in the checked state. <br> <br> **Note:** This will be displayed only if `ContextMenuStripEx.ShowCheckMargin`. |

## API Reference

### Properties

- **Checked**: Indicates whether the item is in the checked state.

  **Note:** This will be displayed only if `ContextMenuStripEx.ShowCheckMargin`.

## Code Examples

#### Example in C#
```csharp
this.contextMenuStripEx1.RightToLeft = Yes;
```

#### Example in VB.NET
```vb
Me.contextMenuStripEx1.RightToLeft = Yes
```

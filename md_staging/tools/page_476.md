### Figure 274: Tasks Window of DateTimePickerAdv Control
The figure displays the Tasks window of the DateTimePickerAdv control, illustrating various settings related to its configuration.

- **Culture**: (Default) configured to the default culture.
- **Appearance**:
  - Checkboxes for:
    - Show CheckBox
    - Show Drop Down Button
    - Show UpDown Button
    - Show UpDown Button on Focus
  - Themes Enabled is checked, with the Visual Style "Office2007" applied.
  - Dropdown alignment is set to "Right."
  - Dropdown colors are configurable: Normal, Pressed, and Selected.
- **Behavior**:
  - Checked State is enabled.
  - Is Null Date is not selected.
  - Null String is set to "No date is selected."

- **Design**:
  - Name is "dateTimePickerAdv1."

## Essential Tools for Windows Forms

### Overview
- Adds a `RibbonControlAdv` control and implements required `ToolStrapTabItems` and `ToolStrapEx` items.
- Demonstrates how to integrate `RibbonPanelMergeContainer` into a `ChildForm`.
- Shows the procedure for adding a `ToolStripEx` to the `RibbonPanelMergeContainer`.
- Explains steps to associate a `ChildForm` with an `MDIParent`.

### Content

#### Using RibbonControlAdv and ToolStrip Components
In a Ribbonform, add a `RibbonControlAdv` control and the required `ToolStrapTabItems` and the `ToolStrapEx` items.

From the ToolBox, add a `RibbonPanelMergeContainer` to the `ChildForm`. `ToolStripEx` can be added into this by right Clicking on it.

![Figure 1210](./image.png)
*Figure 1210: ToolStrip added to the RibbonPanelMergeContainer in the Child Form*

Now add this `ChildForm` to the `RibbonForm` that is the `MDIParent` of the `ChildForm` in the default manner.

**C#**
```csharp
ChildForm frm = new ChildForm();

frm.Show();
```

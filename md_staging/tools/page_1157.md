# Essential Tools for Windows Forms

## Overview
- Describes the use of Bar Items in Windows Forms.
- Focuses on editing text and alignment of Bar Items using properties.
- Demonstrates code examples in C# and VB.NET for setting properties.

## Content

### 3.7.4.1.2.3.4 BarItem Text

Text for a bar item can be edited through the `Text` property. The alignment of text can be specified in the `TextAlignment` property.

#### BarItem Properties

| ParentBarItem Property | Description |
|------------------------|-------------|
| **Text**              | Sets text for the Bar item. |
| **TextAlignment**     | Sets the text alignment. Options are: <br><ul><li>Near (Default)</li><li>Far</li><li>Center</li></ul> |

#### Example Code

**[C#]**

```csharp
this.barItem2.Text = "Center";
this.barItem2.TextAlignment = Syncfusion.Windows.Forms.Tools.XPMenus.TextAlignment.Center;
```

**[VB.NET]**

```vbnet
Me.barItem2.Text = "Center"
Me.barItem2.TextAlignment = Syncfusion.Windows.Forms.Tools.XPMenus.TextAlignment.Center
```

# Essential Tools for Windows Forms

## Content

### Scroll Settings in ScrollersFrame

The horizontal and vertical scrollers have a `Value` property, which represents the current position of the scrollbox on the scrollbar control at runtime. This value can be changed using the `HorizontalSmallChange` and `VerticalSmallChange` properties.

| **Property**             | **Description**                                                                                      |
|--------------------------|------------------------------------------------------------------------------------------------------|
| `HorizontalSmallChange`  | Gets / sets a value to be added or subtracted from the `Value` property when the horizontal scroll box is moved a small distance. Default value is 1. |
| `VerticalSmallChange`    | Gets / sets a value to be added or subtracted from the `Value` property when the vertical scroll box is moved a small distance. Default value is 1.   |

#### Example Code

##### C#

```csharp
this.scrollersFrame2.VerticalSmallChange = 25;
this.scrollersFrame2.HorizontalSmallChange = 25;
```

##### VB.NET

```vb.net
Me.scrollersFrame2.VerticalSmallChange = 25
Me.scrollersFrame2.HorizontalSmallChange = 25
```

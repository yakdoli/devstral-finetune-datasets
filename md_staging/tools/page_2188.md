# Essential Tools for Windows Forms

## Drag Drop Events

### Node Selection

#### Overview
- Describes how the `SelectionMode` property of `TreeViewAdv` affects the drag-and-drop functionality.
- Explains the selection of single nodes, same-level nodes, or multiple nodes during drag-and-drop operations.

#### Content

During drag and drop operation of the tree nodes, a single node or same level nodes or multi level nodes can be selected and dragged based on the selection mode set for the treeview control. SelectionMode property is used for this purpose.

| TreeViewAdv Properties    | Description                                                                                                                                                                         |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| SelectionMode             | Specifies the selection mode of the treeview.<br>**Options are:**<br>- **Single**: The user can only select one node at a time and implement the drag-drop operation in the TreeViewAdv (Default).<br>- **MultiSelectSameLevel**: The user can only select nodes of the same level, i.e. only child nodes or only parent nodes.<br>- **MultiSelectAll**: The user can select multiple nodes for implementing the DragDrop operation in the TreeViewAdv. |

#### Code Examples

**[C#]**
```csharp
this.treeViewAdv1.SelectionMode = TreeSelectionMode.MultiSelectSameLevel;
```

**[VB.NET]**
```vb
Me.treeViewAdv1.SelectionMode = TreeSelectionMode.MultiSelectSameLevel
```

## API Reference

### TreeViewAdv Properties
- **SelectionMode**: Specifies the selection mode of the treeview.

### TreeSelectionMode Enum
- **Single**: The default mode where only one node can be selected at a time.
- **MultiSelectSameLevel**: Allows selection of nodes at the same level.
- **MultiSelectAll**: Allows selection of multiple nodes from any level.

## Overview

- The `Renderer` property provides information about the current `Syncfusion.Windows.Forms.Tools.ITabRenderer` used by the `TabControl` to render the `TabPanel`.
- Explains how to enable custom tabs with designer support using `ITabRenderer`.
- Provides a step-by-step guide to adding custom tab styles to `TabControlAdv` at design time.

## Content

### Renderer Property

The `Renderer` property returns the current `Syncfusion.Windows.Forms.Tools.ITabRenderer` used by the `TabControl` to render the `TabPanel`.

### Enabling Custom ITabRenderer at Design Time

Once you have a `Custom ITabRenderer` implementation, you can make it available to the `TabControlAdv` at design time. To do so:

1. Select the `Add Custom Tabs` entry in the drop-down list that pops-up from the `TabStyle` property editor. This will insert a new `DesignTimeTabTypeLoader` component into your forms designer.

### Summary

This section explains how to use the `Renderer` property, customize tabs, and enable custom tab styles at design time using `ITabRenderer` and `DesignTimeTabTypeLoader`.

## API Reference

### Properties

- **Renderer**: Gets or sets the `ITabRenderer` used by the `TabControl` to render the `TabPanel`.

## Code Examples

### Example: Enabling Custom Tabs

```csharp
// Example of enabling custom tabs
using Syncfusion.Windows.Forms.Tools;

// Assuming CustomITabRenderer is a valid implementation
tabControlAdv1.Renderer = new CustomITabRenderer();

// Adding custom tab style
tabControlAdv1.DesignTimeTabTypeLoader1.AddCustomTabs(TabStyle.Custom);
```

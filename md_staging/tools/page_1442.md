## Essential Tools for Windows Forms

### Figure 978: Collapse and Header Image for XPTaskBar

This section discusses the mouse hover settings of the XPTaskBar control.

The position of the mouse with respect to the control can be known using the properties given below.

| XPTaskBar Box Property | Description |
|-------------------------|-------------|
| HitTaskBoxArea         | Specifies whether the mouse is moved over the TaskBox area. |
| HeaderHit              | Indicates whether the mouse is currently over the header portion. |

### Code Examples

#### C#

```csharp
this.xpTaskBarBox1.HitTaskBoxArea = true;
this.xpTaskBarBox1.HeaderHit = true;
```

#### VB.NET

```vb
Me.xpTaskBarBox1.HitTaskBoxArea = True
Me.xpTaskBarBox1.HeaderHit = True
```

## API Reference

### Properties

- **HitTaskBoxArea**: Type: `bool`  
  Description: Specifies whether the mouse is moved over the TaskBox area.  
  Default: `false`  

- **HeaderHit**: Type: `bool`  
  Description: Indicates whether the mouse is currently over the header portion.  
  Default: `false`  

## Summary

This section provides detailed information about the mouse hover settings of the XPTaskBar control in Windows Forms. The properties `HitTaskBoxArea` and `HeaderHit` are used to determine the position of the mouse relative to the control. Code examples in both C# and VB.NET are provided to demonstrate how to interact with these properties.

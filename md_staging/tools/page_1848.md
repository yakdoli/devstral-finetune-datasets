## Overview
- Adding items to the StatusStripEx control.
- Using the Items Collection Editor to customize the look of StatusControl items.
- Programmatically adding a ToolStripStatusLabel via code.

## Content

### Overview
**Adding Items to the StatusStripEx**

To add items to the StatusStripEx control, access the **Items** property of the control to open the **Items Collection Editor**. Use this editor to add customized **StatusControl** items. This editor allows you to modify the look and feel of the items using the properties provided on the right side.

**Figure 1230: Adding Items through Items Collection Editor**

- **Note**: A shortcut to add **ToolStripStatus** items is through the **Tasks Window**. Refer to **Smart Tag options** for more details.

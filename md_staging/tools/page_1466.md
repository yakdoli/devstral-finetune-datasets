# Essential Tools for Windows Forms

## Overview
- Describes the NavigationView BarCollection Editor and how to create a NavigationView control programmatically.
- Details steps for incorporating the Tools Windows namespace for programmatic control creation.

## Content

### BarCollection Editor

#### Figure 989: NavigationView BarCollection Editor

The figure shows the interface for the BarCollection Editor, a tool used for managing the appearance and behavior of NavigationView controls. The editor displays properties such as Appearance and Behavior, allowing users to adjust settings like Disabling Image, Image Index, Text, AutoExpand, Enabled, and Visible options.

#### Creating a NavigationView Control Programmatically

To create a NavigationView control programmatically, follow the steps below:

1. **Include the Tools Windows namespace to the .cs / .vb file.**

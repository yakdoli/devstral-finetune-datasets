# Essential Tools for Windows Forms

## Overview
- **Vertical Layout**: Determines whether TaskBar Boxes should be aligned vertically or horizontally in the XPTaskBar control. The default layout is Vertical.
- **Column Width on Horizontal Alignment**: Specifies the width for each column in the Horizontal Layout mode.
- **Properties**: Includes VerticalLayout and ColWidthOnHorizontalAlignment.

## Content

### Vertical Layout
| **Property Name** | **Description** | **Default Value** | **Data Type** | **Usage Example** |
|--------------------|-----------------|--------------------|---------------|-------------------|
| VerticalLayout    | Gets/sets the value which determines whether the TaskBar Boxes should be aligned vertically or horizontally in the XPTaskBar control. The default layout is Vertical. | Normal | Boolean [true/false] | Public Property VerticalLayout As Boolean<br>this.xpTaskBar1.VerticalLayout = true; |

### Column Width on Horizontal Alignment
| **Property Name** | **Description** | **Default Value** | **Data Type** | **Usage Example** |
|--------------------|-----------------|--------------------|---------------|-------------------|
| ColWidthOnHorizontalAlignment | Specifies the width for each column in the Horizontal Layout mode. | Normal | Integer value | public int ColWidthOnHorizontalAlignment { get; set; }<br>this.xpTaskBar1.ColWidthOnHorizontalAlignment = 100; |

**Note**: When setting `ColWidthOnHorizontalAlignment` to `false`, additional actions may be required for proper functionality.

## API Reference
### Properties
- **VerticalLayout**: Determines the alignment orientation of TaskBar Boxes within the XPTaskBar control.
  - **Type**: Boolean
  - **Default Value**: True (Vertical)
  - **Usage**: `this.xpTaskBar1.VerticalLayout = true;`

- **ColWidthOnHorizontalAlignment**: Specifies the width for each column in the Horizontal Layout mode.
  - **Type**: Integer
  - **Default Value**: Not specified
  - **Usage**: `this.xpTaskBar1.ColWidthOnHorizontalAlignment = 100;`

## Code Examples
```csharp
// Setting VerticalLayout property
this.xpTaskBar1.VerticalLayout = true;

// Setting ColWidthOnHorizontalAlignment property
this.xpTaskBar1.ColWidthOnHorizontalAlignment = 100;
```

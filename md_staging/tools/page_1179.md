# Essential Tools for Windows Forms

## Overview
- Describes the behavior of the command bar when docked.
- Details properties that control hiding and showing various elements of the CommandBar.
- Explains properties related to the dropdown functionality of the CommandBar.

## Content

### Command Bar Display
Display the text that is set through the CommandBar.Text property when the command bar is in docked position.

### Hide / Show
#### CommandBar Property Description
| CommandBar Property         | Description                                                                 |
|-----------------------------|-----------------------------------------------------------------------------|
| HideChevron                 | When set to true hides the chevron for the CommandBar.                    |
| HideCloseButton             | Hides the Close button for the floating CommandBar when set to true.      |
| HideDropDownButton          | Shows / hides the dropdown button.                                       |
| HideGripper                 | Shows / hides the drag gripper.                                          |

#### Figure 797: CommandBar in Floating and Docked mode illustrating various buttons and Gripper
- **Docked CommandBar**: Contains the drag gripper, chevron, menu button, and command bar.
- **Chevron**: Indicated at the top right of the docked CommandBar.
- **Menu Button**: Present next to the chevron in the docked CommandBar.
- **Command Bar**: Shown in both the docked and floating modes.
- **Floating CommandBar**: Includes the close button, dropdown menu, and drag gripper.
- **Close Button**: Indicates the button used to close the floating CommandBar.

#### PopUp for the DropDown
Provides details about the properties that control the dropdown behavior.

| CommandBar Property         | Description                                                                 |
|-----------------------------|-----------------------------------------------------------------------------|
| PopupContainer              | Indicates the PopupContainer control that is displayed when the dropdown button is clicked. |
| PopupMenu                   | Indicates the Popup menu on clicking the dropdown button.                     |

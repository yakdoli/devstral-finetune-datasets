# Essential Tools for Windows Forms

## Overview

- FontListBox is a list box derived control that gets automatically populated with the fonts installed on the user's system.
- FontListBox control provides a user interface to display and allow selection of fonts installed in the system.

## Content

### FontListBox Control

![Figure: FontListBox Control](./image?name=Figure_583_FontListBox_Control)

**Figure 583: FontListBox Control**

This figure illustrates the FontListBox control in action, displaying a list of fonts installed on the system. It also shows how selecting a font changes the appearance of a label and simulates a "Traffic Signal" effect.

### See Also

#### Features

- **FontListBox displays and allows selection of fonts.** It has the following features:

    - The FontListBox control automatically fills the listbox with the fonts installed in the system.
    - Has options in selecting the fonts such as single-select, multi-select, or not selectable. [See Selection Mode topic](../Selection_Mode).
    - Vertical and horizontal scrollbars available for the FontListBox control.

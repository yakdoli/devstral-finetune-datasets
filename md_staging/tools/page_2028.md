# Essential Tools for Windows Forms

## Overview
- This page explains how to enable and use scroll buttons in the `TabControlAdv` control to navigate through tab pages when the number of tabs increases.

## Content

### ScrollButton

The `TabControlAdv` has the ability to add scroll buttons on the `TabPanel` when the number of `Tab` items increases. These scroll buttons allow the user to navigate through the tab pages in the front and back. For this, the `ShowScroll` property of the `TabControl` must be set.

#### TabControlAdv Property

| Property         | Description                                               |
|------------------|-----------------------------------------------------------|
| ShowScroll       | Specifies whether to show the scroll button.             |

- **By default, scroll buttons are set to 2D, 3D, and Workbook tab styles.**

![Scroll buttons displayed for the TabControlAdv](image.png)
*Figure 1383: Scroll buttons displayed for the TabControlAdv*

- **For all other styles, scroll buttons are set to normal button-look.**

![Scroll buttons with Normal Button-Look](image.png)
*Figure 1384: Scroll buttons with Normal Button-Look*

- **ScrollButtons can have a VS .NET - TabbedMDI scroll button-look when the `VSLikeScrollButton` property is set to `true`.**

## API Reference

- **Properties**
  - `ShowScroll` - Specifies whether to show the scroll button.
  - `VSLikeScrollButton` - Specifies whether to enable the VS .NET - TabbedMDI look for scroll buttons.

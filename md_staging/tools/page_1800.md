# Essential Tools for Windows Forms

## Content

### Caption Settings

Caption for a ToolStripEx can be enabled using the **ShowCaption** property. Text for the Caption is set using the **Text** property.

![Selecting ToolStripEx control using Select Option](https://example.com/figure_1190.png)

**Figure 1190: Selecting ToolStripEx control using Select Option**

#### Code Examples

**[C#]**

```csharp
this.toolStripEx1.ShowCaption = true;
this.toolStripEx1.Text = "Standard Items";
```

**[VB.NET]**

```vb
Me.toolStripEx1.ShowCaption = True
Me.toolStripEx1.Text = "Standard Items"
```

## API Reference

- **Properties:**
  - **ShowCaption**: Enables or disables the caption display for the ToolStripEx control.
  - **Text**: Sets the text displayed as the caption for the ToolStripEx control.

## Essential Tools for Windows Forms

The `TabPrimitiveHost` property allows to customize the navigation and close buttons by defining it through the `TabPrimitives` Property Collection. The tab primitives can be added, and each primitive can be assigned with the type to be used, which includes primitives to traverse to the First / Previous / Next / LastTab, Next / PreviousPage and Close / DropDown options.

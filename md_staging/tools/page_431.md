### Alignment and Wrapping of Text

#### Overview
- **Control**: The alignment and text wrapping of the dates inside the grid cells are controlled using specific properties.
- **Properties**:
  - `HorizontalAlignment`
  - `VerticalAlignment`
  - `WrapText`

#### Detailed Description

The alignment and text wrapping of the dates inside the grid cells are controlled using the following properties:

| MonthCalendarAdv Properties | Description |
|-------------------------------|-------------|
| **HorizontalAlignment**      | Specifies the horizontal alignment of the dates inside a grid cell. The options are:   <br>- Left <br>- Center <br>- Right |
| **VerticalAlignment**        | Specifies the vertical alignment of the dates inside a grid cell. The options are:    <br>- Top <br>- Middle <br>- Bottom |
| **WrapText**                 | Indicates whether the grid can wrap the text inside grid cells. By default, it is false. |

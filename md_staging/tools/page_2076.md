### Close Button

#### Section Title: 3.18.2.5.2 Close Button

The `CloseButtonVisible` property controls the visibility of the close button:

- **CloseButtonVisible**: When set, the close button will be either visible or hidden.
- **ShowCloseButtonForActiveTabOnly**: Enables the close button for the active tab only.
- **ShowCloseButton**: Displays the close button for all tabs.

Additionally, individual tab close buttons can be enabled with the following snippet.

#### Code Example

```csharp
[C#]
// Individual Close Buttons enabled.
```

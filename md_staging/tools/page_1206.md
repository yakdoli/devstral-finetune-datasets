# Essential Tools for Windows Forms

## Overview
- Details the addition of separators in `PopupMenu` or grouping items within a `BarManager`.
- Focuses on the `SeparatorIndices` property for managing separators when the `ParentBarItem` is not part of a `BarManager`.
- Illustrates the use of the `Int32 Collection Editor` for adding separators.

## Content

### Adding Separators for a PopupMenu or Grouping Item within a BarManager

**Figure 822:** Adding Separator for a `PopupMenu` or Grouping item contained within a `BarManager`

If the ParentBarItem is not contained within a `BarManager`, edit the **SeparatorIndices** property of the `ParentBarItem` indicating the item indices in the items list where you want the separators to be introduced.

#### Int32 Collection Editor

**Figure 823:** Adding Separators through the `SeparatorIndices` Collection Editor when `BarManager` is Not Used

- The **Members** list shows indices `2` and `5` where separators will be placed.
- Two properties (Value) are shown as `2`.

#### Separators AFTER BarItem2 and BarItem5

**Figure 824:** Separators after `BarItem2` and `BarItem5`
- The context menu `PopupMenu` displays items with `Separators` introduced after `BarItem2` and `BarItem5`.

- A `PopupMenu` represents a context menu that can be shown over any control when the user right clicks the application. It must be associated with a control whose contents will be displayed in the popup.
- The `BarManager` is similar to the .NET `ContextMenu`, allowing multiple menus to be displayed.
- `PopupMenusManager` is used in the designer to set the display properties of the menu.

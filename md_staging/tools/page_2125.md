# Essential Tools for Windows Forms

## Overview
- The page explains how to use the `TabSplitterContainer` control in Windows Forms.
- Demonstrates adding primary pages to the control using both the designer and programmatically.
- Highlights the use of Syncfusion's tools and namespaces for enhanced functionality.

## Content

### Programmatically Adding Primary Pages
It can be created programmatically using the following steps:

1. **Create or open a Windows Forms project**:
   - Begin by setting up your project environment.

2. **Add Syncfusion's Tools.Windows and Shared.Base assemblies to the application**:
   - Reference the necessary Syncfusion assemblies.

3. **Add the Syncfusion.Windows.Forms.Tools namespace**:
   - Include the appropriate namespace for functionality.

```csharp
using Syncfusion.Windows.Forms.Tools;
```

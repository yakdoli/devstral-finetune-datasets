# Essential Tools for Windows Forms

## Overview
- Demonstrates the use of AutoComplete control in Windows Forms.
- Explains how to configure AutoCompleteSource using "HistoryList".
- Describes the process of specifying custom auto completion text using the String Collection Editor.

## Content

### Search Wizard

The interface displayed shows an **AutoComplete control** in use. The user is prompted to **"Enter the Topic Name to Search"**, and a list of suggested URLs appears based on the input. This feature uses **AutoCompleteSource = "HistoryList"** to populate the suggestions.

#### Custom Source
The **Custom Source** section explains how to specify a set of auto completion text using the **String Collection Editor**. This editor is invoked using the **Control.AutoCompleteCustomSource** property.

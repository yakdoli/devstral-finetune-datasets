# Essential Tools for Windows Forms

## Overview
- The Alignment property is used to specify the type of layout logic (simple or constraint-based).
- Note: Alignment is applied only in the direction of the layout flow.

## Content

### Alignment Property
The Alignment property determines how layout components are aligned in the direction of the flow.

#### Table: FlowLayout Property

| FlowLayout Property | Description                                                                                          |
|----------------------|------------------------------------------------------------------------------------------------------|
| Alignment            | Specifies the alignment of layout components in the direction of flow.<br><br>The options included are as follows:<br>Center,<br>Near,<br>Far and<br>ChildConstraints. |

#### Code Examples

**C#**
```csharp
this.flowLayout1.Alignment =
    Syncfusion.Windows.Forms.Tools.FlowAlignment.Near;
```

**VB.NET**
```vb.net
Me.flowLayout1.Alignment =
    Syncfusion.Windows.Forms.Tools.FlowAlignment.Near
```

### Note
- Alignment is applied only along the direction of flow. For example, if the LayoutMode property is set to 'Horizontal' and the Alignment property is set to 'Center', then the rows will be centered horizontally.

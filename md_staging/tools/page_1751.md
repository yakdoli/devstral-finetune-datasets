## Gallery

### Overview
- The Gallery feature in Essential Tools RibbonControlAdv allows you to add a collection of items to a gallery.
- A gallery can be added to a ToolStripTabItem using the Items Collection Editor.
- To add a Gallery item, select it in the dropdown and add it to the control.

#### Adding Controls to the Gallery

Using the `Gallery.Item` property, the Items Collection Editor opens, allowing you to add items to the gallery.

```csharp
this.toolStripGallery1.Items.Add(toolStripGalleryItem1);
this.toolStripGallery1.Items.Add(toolStripGalleryItem2);
this.toolStripGallery1.Items.Add(toolStripGalleryItem3);
```

```vb.net
Me.toolStripGallery1.Items.Add(toolStripGalleryItem1)
Me.toolStripGallery1.Items.Add(toolStripGalleryItem2)
Me.toolStripGallery1.Items.Add(toolStripGalleryItem3)
```

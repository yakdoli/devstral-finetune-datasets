# Essential Tools for Windows Forms

## Overview
- Specifies layout mode for Child controls with options: Default and Fill.
- Explains behavior of Child controls based on Container size and layout mode.

## Content

### LayoutMode

| Property              | Description                                                                                                                                                                                                                                                                                                                                                                                                              |
|-----------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **LayoutMode**        | Specifies the layout mode for the Child controls. The default value is set to 'Default'. <br> The options included are as follows. <br> Default and Fill.                                                                                                                |

#### Behavior Based on Layout Mode

- When the layout mode of CardLayout is set to **'Default'**, the Child control is simply centered within the Container when the Container's size is bigger than the Child control's preferred size.
- However, if the Container's size is smaller than the Child controls's preferred size, the Child control's size will shrink down to its minimum size. When shrunk, you have an option to specify whether the preferred width / height aspect ratio should be maintained for that Child control, which is specified using the extended **MaintainAspectRatio** property of each Child.

- When the layout mode is set to **'Fill'**, it simply resizes the Child control to fill the entire Container area.

### Code Examples

#### C#

```csharp
this.cardLayout1.LayoutMode = Syncfusion.Windows.Forms.Tools.CardLayoutMode.Fill;
```

#### VB.NET

```vb
Me.cardLayout1.LayoutMode = Syncfusion.Windows.Forms.Tools.CardLayoutMode.Fill
```

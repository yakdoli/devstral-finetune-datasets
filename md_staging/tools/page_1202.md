# Essential Tools for Windows Forms

## Overview
- Demonstrates reusing `ParentBarItem` added through `BarManager`.
- Explains how to customize the `PopupMenu` by dragging `BarItems` into it.
- Provides steps to access the `Customize` dialog for menu customization.

## Content

### Reusing ParentBarItem added through BarManager

**Figure 818: Reusing ParentBarItem added through BarManager**

![](image_url_for_Figure_818)

In the presence of a `BarManager`, the user can just drag `BarItems` into the popup menu. Right-click on the `popupMenu1` and select **Customize** to invoke this dialog and drag the required menu items.

---

### Customizing the PopupMenu

**Figure 819: Customize Dialog Box**

![](image_url_for_Figure_819)

The **Customize** dialog box provides options to manage tools and commands for the popup menu. Categories such as `Form1`, `File`, and `Edit` are displayed on the left, while specific commands like `Edit`, `Cut`, `Copy`, `Paste`, and `Clear Events` are listed on the right. Users can drag these commands into the popup design area to add them to the menu.

## API Reference

### Configuration Properties

- **Name**: Displays the name of the element (`popupMenu1`).
- **GenerateMember**: Indicates whether the member is generated (`True`).
- **IgnoreWorkingArea**: Specifies if the working area is ignored (`False`).
- **Modifiers**: Access modifiers (`Private`).
- **ParentBarItem**: Determines the parent element (`parentBarItem1`).

### Methods and Events
- **SynchronousPopup**: Configures the popup behavior.

## Code Examples

### C# Example

```csharp
// Example of configuring the popupMenu
this.popupMenu1.ParentBarItem = this.parentBarItem1;
this.popupMenu1.SynchronousPopup = true;
```

### VB.NET Example

```vb
' Example of configuring the popupMenu
Me.popupMenu1.ParentBarItem = Me.parentBarItem1
Me.popupMenu1.SynchronousPopup = True
```

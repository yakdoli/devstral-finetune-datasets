### Flat Appearance
**Dropdown can be given flat appearance using FlatDropDown property. By default it is false.**

#### C#

```csharp
this.dateTimePickerAdv2.FlatDropDownButton = true;
```

#### VB.NET

```vb
this.dateTimePickerAdv2.FlatDropDownButton = true;
```

### Note
These settings will be effective only when DateTimePickerAdv.Style is Default.

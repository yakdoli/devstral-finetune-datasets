# Essential Tools for Windows Forms

## Overview
- Discusses the launcher settings properties.
- Explains how to specify the visibility and style of the launcher in the control.
- Demonstrates using C# and VB.NET code examples to configure launcher styles.

## Content

The below properties deal with the launcher settings.

| Property      | Description                                                                 |
|---------------|-----------------------------------------------------------------------------|
| ShowLauncher  | Specifies the visibility of the Launcher in the control.                |
| LauncherStyle | Sets the Style for the launcher. The styles are, <br> - Office12, <br> - Office2007 |

### Code Examples

#### [C#]

```csharp
this.toolStripEx1.ShowLauncher = true;
this.toolStripEx1.LauncherStyle = LauncherStyle.Office2007;
```

#### [VB.NET]

```vb
Me.toolStripEx1.ShowLauncher = True
Me.toolStripEx1.LauncherStyle = LauncherStyle.Office2007
```

### Grip Style

The toolstrip can hold a grip, which can be visible by setting the GripStyle property. We can enable GripStyle easily using the smart tag of the ToolstripEx control.

| Property     | Description                                               |
|--------------|-----------------------------------------------------------|
| GripStyle    | Specifies the grip style for the ToolStripEx control.    |

---

## API Reference

### Properties

- **ShowLauncher**  
  - Type: Boolean
  - Description: Specifies whether the Launcher is visible in the control.
  - Default: false

- **LauncherStyle**  
  - Type: LauncherStyle
  - Description: Specifies the style of the launcher.
  - Default: Office12

- **GripStyle**  
  - Type: GripStyle
  - Description: Specifies the grip style for the ToolStripEx control.
  - Default: Auto

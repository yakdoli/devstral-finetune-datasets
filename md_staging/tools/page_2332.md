# Essential Tools for Windows Forms

## Overview
- Select the WizardControl and use the WizardPages property in the Property Editor to reorder pages.
- Reordering is done through the collection editor using Up and Down arrow keys.
- Designer functionality for moving pages to the front or back is available.

## Content

### Reordering Wizard Pages
- **Selecting the WizardControl and Property Editor**
  - Select the `WizardControl` and choose the `WizardPages` property in the Property Editor. This brings up the collection editor where you can reorder the pages using the Up and Down arrow keys.
  - **Figure 1503: Reordering Wizard Pages**
    ![Figure 1503: Reordering Wizard Pages](https://.../figure_1503.png)  
      - The collection editor displays the list of `WizardControlPage` members, where you can reorder pages as indicated in the image.
      - The `wizardControlPage` properties provide controls for appearance, such as `BackColor`, `BackgroundColor`, and `Border3DStyle`.

### Using the Designer
- In the designer, right-click on a page and choose 'Bring To Front' or 'Send To Back' options to move the page to the beginning or to the end of the collection, respectively.

# Essential Tools for Windows Forms

## Overview

- Describes the `ScrollerType` property for setting the scroller type in the Gallery.
- Explains the available scroller types: `StandardType` and `CompactType`.
- Demonstrates the appearance of a gallery item display for both scroller types, including Caption text, BackColor, and ItemDisplayStyle properties.

---

## Content

### ScrollerType Property

| Property       | Description                                                                                           |
|----------------|-------------------------------------------------------------------------------------------------------|
| ScrollerType   | Sets the scroller type for the Gallery. The types available are:                                |
|                | <ul><li>StandardType</li><li>CompactType</li></ul>                                                 |

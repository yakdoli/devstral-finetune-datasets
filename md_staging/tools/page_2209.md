# Essential Tools for Windows Forms

## Overview

- Inactive Nodes painted with OrangeRed and Ivory Background with SteelBlue Foreground
- Foreground setting of tree node text

## Content

### Foreground Settings

#### Foreground Settings for the tree node text

Using the Text and TextColor property, individual node's text can be edited and colored respectively.

Using the Font and the ForeColor properties of the control, the text and text color of the nodes can be set for all the nodes in the TreeView control.

#### Notes
- **Font Style for Individual Nodes**: The font style for individual nodes can be set using the Font property available for individual nodes using `NodeCollection Editor`.
- **ForeColor Property Reflection**: When you set the ForeColor property for TreeViewAdv control, it will get reflected in the Node's TextColor property. User can change the color for specific nodes using `TreeNodeAdv.TextColor` property.

#### TreeViewAdv Properties

| TreeViewAdv Properties | Description                          |
|-------------------------|--------------------------------------|
| Font                    | Specifies the Font style of the node text. |
| ForeColor               | Specifies the text color of the nodes. |

| TreeViewAdv Properties | Description                          |
|-------------------------|--------------------------------------|
| Text                    | Sets text for the node.             |
| TextColor               | Sets the color for the specific node text. |

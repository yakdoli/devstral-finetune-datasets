### Adding GroupBar Items Using the Property Browser

- To add a GroupBar Item using the property browser, follow the procedure below:

1. In the Properties window, select the `GroupBarItems` property. This will open the **GroupBarItem Collection Editor**.
2. Click **Add** in the GroupBarItem Collection Editor. A GroupBar Item will be added to the GroupBar control.

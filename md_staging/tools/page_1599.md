# Essential Tools for Windows Forms

## Overview
- Introduces the alignment settings for Wrapping panels and child controls to be added to the StatusBarAdv control.
- Explains how alignment can be customized using the `Alignment` property.
- Demonstrates C# and VB.NET code examples to set alignment options.

## Content

### 3.11.4.3.6 Alignment Settings

The panels and child controls that are added to the StatusBarAdv control can be aligned according to the needs of the user using the property given below.

| StatusBarAdv Property     | Description                                                                                                                                                  |
|---------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Alignment                 | Determines the alignment of the panels. The options included are given below.                                                                                |
|                           | - Center,                                                                                                                                                   |
|                           | - Near,                                                                                                                                                     |
|                           | - Far,                                                                                                                                                     |
|                           | - ChildConstraints.                                                                                                                                         |

#### C# Code Example
```csharp
this.statusBarAdv1.Alignment = Syncfusion.Windows.Forms.Tools.FlowAlignment.Center;
```

#### VB.NET Code Example
```vb
Me.statusBarAdv1.Alignment = Syncfusion.Windows.Forms.Tools.FlowAlignment.Center
```

#### Alignment Options
![Alignment Options](image.png)

**Figure 1062: Alignment Options**

If the `Alignment` property is set to 'ChildConstraints', the positioning and resizing of the panels and child controls can be set by calling the `SetHAlign()` method.

## API Reference

### Properties
#### Alignment
- **Type**: Syncfusion.Windows.Forms.Tools.FlowAlignment
- **Description**: Determines the alignment of the panels.
- **Options**:
  - Center
  - Near
  - Far
  - ChildConstraints

### Methods
#### SetHAlign
- **Description**: Used to set the horizontal alignment for panels and child controls when `ChildConstraints` is applied.

## Code Examples

### Setting Alignment to Center Using C#
```csharp
this.statusBarAdv1.Alignment = Syncfusion.Windows.Forms.Tools.FlowAlignment.Center;
```

### Setting Alignment to Center Using VB.NET
```vb
Me.statusBarAdv1.Alignment = Syncfusion.Windows.Forms.Tools.FlowAlignment.Center
```

# Essential Tools for Windows Forms

## Overview
- Describes the TreeNodePrimitives Collection Editor.
- Explains the various types of primitives available for customizing nodes.
- Provides details on setting the position of node contents using the Primitive Collection Editor.

## Content

### Figure 1434: TreeNodePrimitives Collection Editor

The TreeNodePrimitives Collection Editor is shown in the figure above. It allows users to customize various properties of a node such as its images, font, and other attributes. Below are some key features and functionalities:

- **Types of Primitives Available:**
  - LabelPrimitive
  - LeftImagePrimitive
  - RightImagePrimitive
  - CheckBoxPrimitive
  - OptionButtonPrimitive
  - CustomControlPrimitive

#### Primitive Collection Editor
The Primitive Collection Editor is used to set the **index** property for each node, allowing fine control over the arrangement and positioning of node contents.

---

### LabelPrimitive
- **LabelPrimitive** is used to display the text of the label.

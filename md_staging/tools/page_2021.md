## Interactive Features

This section discusses the CloseButton features of TabControlAdv.

It includes the below topics:

### 3.17.9.1 Interactive Features
#### 3.17.9.1.1 CloseButton Settings

TabControl can have CloseButtons for all the TabPages as in Internet Explorer 7 to close the corresponding TabPages irrespective of the Style set.

![CloseButton Example](image.png)

## API Reference
- **Namespace:** Syncfusion.Windows.Forms.Tools
- **Class:** TabControlAdv

### Properties
- **ThemesEnabled**: Enables or disables themes for the TabControlAdv.

## Code Examples

### Example: Enabling Themes for TabControlAdv

```vbnet
Me.tabControlAdv1.ThemesEnabled = True
```

### Example: Adding Close Buttons to TabPages

```vbnet
Me.tabControlAdv1.CloseButton = True
```

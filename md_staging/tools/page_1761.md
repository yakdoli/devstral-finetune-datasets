# Essential Tools for Windows Forms

## Overview
- Discusses the properties and behavior of the ToolStripButton in Windows Forms.
- Describes how to add and customize Tool_stripButtons within a ToolStripEx.
- Highlights the use of Foreground Settings like Text, Font, ForeColor, and TextAlign to control display properties.

## Content

### Button

#### Adding ToolStripButtons to ToolStripEx
- **Description**: 
  - ToolStripButton can be added to a ToolStripEx directly or through a panel.
  - Figure 1172 shows an example of ToolStripButtons added to the ToolStripEx:
  
  ![Figure 1172: ToolStripButtons Added to the ToolStripEx](https://example.com/image1172.png)

#### Properties Controlling Appearance and Behavior
- The following properties control the appearance and behavior of the Tool_stripButton. These are categorized under "Foreground Settings."

##### Foreground Settings
| Property    | Description                                                                 |
|-------------|-----------------------------------------------------------------------------|
| **Text**    | Sets the Text for the ToolStripButton. This text will be displayed, only if the DisplayStyle is Text or ImageAndText. |
| **Font**    | Sets the font style for the display text.                                   |
| **ForeColor** | Sets the fore color for the display text.                                 |
| **TextAlign** | Specifies the alignment of the text in the item. The options are,         |
|               | • TopLeft,                                                                 |
|               | • TopCenter,                                                               |
|               | • TopRight,                                                                |
|               | • MiddleLeft,                                                              |
|               | • MiddleCenter,                                                           |
|               | • MiddleRight,                                                            |
|               | • BottomLeft,                                                              |

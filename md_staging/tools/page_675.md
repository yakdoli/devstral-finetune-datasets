# Essential Tools for Windows Forms

## Overview
- GradientPanelExt includes a margin for all four sides with a customizable spacing gap.
- The `BorderGap` property is used to control the space between the border and margins.
- The document provides examples for setting the `BorderGap` programmatically in both C# and VB.NET.

## Content

### GradientPanelExt Property: BorderGap
| GradientPanelExt Property | Description |
|-----------------------------|-------------|
| BorderGap                  | Used to get or set the gap between the border and the margins. |

The border gap for the GradientPanelExt can be set programmatically as shown below:

#### C#
```csharp
gradientPanelExt1.BorderGap = 40;
```

#### VB.NET
```vb.net
Private gradientPanelExt1.BorderGap = 40
```

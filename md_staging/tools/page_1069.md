### Configuring Child Controls

The following settings can be used to configure the Child controls of the GridLayout Manager.

#### ParticipateInLayout

To prevent a Child control from being laid out using the GridLayout Manager, the following property can be used.

| Child Control Property           | Description                                                                                                         |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| ParticipateInLayout              | Specifies whether the Child control should participate in the GridLayout. The default value is set to 'True'. |

The methods associated with the above property are given below.

---

## API Reference

### Properties
- **ParticipateInLayout**  
  *Description*: Controls whether the Child control should participate in the GridLayout.  
  *Default Value*: `True`  

## Code Examples
```csharp
// Example to disable a child control from participating in GridLayout
button1.ParticipateInLayout = false;
```

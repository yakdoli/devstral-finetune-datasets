# Essential Tools for Windows Forms

## Content

### Setting Orientation

The TrackBarEx control allows you to set its orientation to either horizontal or vertical. Here's how to do it in both C# and VB.NET.

#### C#

```csharp
this.trackBarEx1.Orientation = Orientation.Horizontal;
```

#### VB.NET

```vbnet
' To set the control to be vertically oriented
Me.trackBarEx1.Orientation = Orientation.Vertical
' To set the control to be horizontally oriented
Me.trackBarEx1.Orientation = Orientation.Horizontal
```

### TrackBar Value

#### Overview
- **TrackBarEx** slides between the **minimum** and **maximum** values, which are specified in the **Minimum** and **Maximum** properties.
- The properties and their descriptions are listed in the table below.

#### Properties and Description

| Property    | Description                                                                 |
|-------------|-----------------------------------------------------------------------------|
| Minimum     | Specifies the minimum value of the trackbar. Default is 10.              |
| Maximum     | Specifies the maximum value of the trackbar. Default is 20.              |

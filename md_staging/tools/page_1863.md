# Essential Tools for Windows Forms

## Overview
- This section explains how to access the Items Collection Editor either through the Item property or via the Edit Items... command.
- Discusses adding menu items through the Items Collection Editor in the Designer.
- Explains how to add ToolStripMenuItem (Cut) to the menu list using code.

## Content

### Accessing Items Collection Editor

**Figure 1245: Options for Accessing Items Collection Editor through Item property or Edit Items... Command**

![Items Collection Editor](./image.png)

The **Items Collection Editor** allows you to select and add items to a list for configuring menu elements like `MenuItem`, `ComboBox`, `Separator`, and `TextBox`. The right panel displays properties such as `Behavior`, `Data`, and `Design`, where you can adjust settings like `UseWaitCursor`, `AllowDrop`, and `Enabled`.

**Figure 1246: Adding Menu Items through Items Collection Editor in Designer**

This figure demonstrates how to add menu items using the Designer interface. The left pane shows the available options for adding different item types, while the right side lists their properties.

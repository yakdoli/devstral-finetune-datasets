# Essential Tools for Windows Forms

## Overview
- Describes the design-time verb for adding a tab in the TabControlAdv control.
- Steps to add a tab programmatically at design time.

## Content

### Step-by-Step Guide: Adding a Tab at Design Time

#### Figure 1342: Add Tab Design-Time Verb

![Design-Time Verb for Adding a Tab](#fig1342)

- **Figure Description**: The image shows the design-time interface for the `TabControlAdv` control. The context menu lists options such as "Add Tab," "Remove Tab," and other control management features.

2. **In the Properties window**, select the TabPages property. The TabPageAdv Collection Editor will be opened. Click Add, a tabpage will be added to the TabControlAdv.

## API Reference

### Properties
- **TabPages**: Collection property to manage the tab pages within the `TabControlAdv`.

### Methods
- **Add**: Adds a new tab to the `TabControlAdv`.

### Events
- None explicitly shown in the context of this page.

## Code Examples

#### Adding a Tab Programmatically
```csharp
// Assuming tabControlAdv is the instance of TabControlAdv
tabControlAdv.TabPages.Add();
```

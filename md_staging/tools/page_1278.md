# Essential Tools for Windows Forms

## Overview
- Describes the process of setting text, `BorderSingle` and `BorderStyle` properties for subitems using the `TreeNodeAdvSubItem` Collection Editor.
- Explains how to customize subitems through various properties in the SubItems Collection dialog.

### Properties for Customizing the SubItems

#### The below properties lets you customize the subitems using the SubItems Collection dialog.
The following table lists the properties available for customizing `TreeNodeAdvSubItem`:

| TreeNodeAdvSubItem Property | Description |
|------------------------------|-------------|
| Alignment                   | Sets the alignment of the SubItem text. |
| Background                   | Sets the background for the subitem. |
| LineAlignment                | Sets the vertical alignment of the subitem text. |
| Text                         | Sets the text for the SubItem. |
| TextColor                    | Sets the color for the SubItem text. |
| Visible                      | Sets the visibility of the subitem. |
| BaseStyle                    | Specifies the BaseStyle that it should inherit from. |

## Code Examples (multi-language supported)
```csharp
// Example of configuring a subitem
TreeNodeAdvSubItem subItem = new TreeNodeAdvSubItem();
subItem.Text = "Example Text";
subItem.TextColor = System.Drawing.Color.Black;
subItem.Visible = true;
subItem.Alignment = TextAlignment.Center;
subItem.Background = System.Drawing.Color.LightGray;
subItem.BaseStyle = BaseStyles.None;
```

#### 3.14.2.2 Setting Channel Height

You can set the required height of the channel over which the slider moves. In the following example, the change in height of the channel from 4 to 6 pixels is illustrated.

The following code snippet illustrates how to set the Channel height.

**a. When the Channel height is set to 4 pixels:**

```csharp
rangeSlider.ChannelHeight = 4;
```

<figure>
  <img src="ChannelHeight4.png" alt="Design View of RangeSlider when the height is set to 4" />
  <figcaption>Figure 1313: Design View of RangeSlider when the height is set to 4</figcaption>
</figure>

**b. When the Channel height is set to 6 pixels:**

```csharp
rangeSlider.ChannelHeight = 6;
```

## API Reference
- **Property**
  - **ChannelHeight**
    - **Type**: `int`
    - **Description**: Determines the height of the channel where the slider moves.

## Code Examples

### Example 1: Setting Channel Height to 4

```csharp
rangeSlider.ChannelHeight = 4;
```

### Example 2: Setting Channel Height to 6

```csharp
rangeSlider.ChannelHeight = 6;
```

### Additional Information
- **Default Value**: ChannelHeight is typically set to a default value, which may vary based on the theme or application settings.

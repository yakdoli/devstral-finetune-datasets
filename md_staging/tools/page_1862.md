# Essential Tools for Windows Forms

## Overview
- Describes how to add menu items using a ContextMenuStripEx control in a Windows Forms application.
- Highlights the use of both the designer interface and property grid for adding menu items.
- Explains the process of integrating menu options into a form.

## Content

### Adding Menu Items Through Designer

Drag and drop a ContextMenuStrip to the form. Click "Type Here" to add the items. This displays a collection of menu items, using which the user can add the menu items.

![Figure 1244: Adding Menu Items Through Designer](https://i.imgur.com/Example.png)

**Figure 1244: Adding Menu Items Through Designer**

Items can also be added using `ContextMenuStripEx.Items` property or the **Edit Items...** command in the property grid.

![Property Grid for ContextMenuStripEx](https://i.imgur.com/Example.png)

### Property Grid for ContextMenuStripEx

The property grid view allows users to add items to the `ContextMenuStripEx` control by selecting the **Edit Items...** option from the **Items** section.

## API Reference

### ContextMenuStripEx.Items

- **Type**: Collection
- **Description**: Represents the collection of items displayed in the context menu strip.

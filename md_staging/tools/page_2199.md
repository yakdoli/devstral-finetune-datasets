# Essential Tools for Windows Forms

## Overview
- TreeViewAdv control supports Office2007 Look and Feel for ScrollBars.
- The control integrates with Office2007 Scrollbars using three color schemes: Blue, Silver, and Black.
- The Office2007ScrollBarsColorScheme property is used to select the scrollbars color scheme.
- FillSplitterPane property provides support for using the control inside a dynamic splitter window and sharing scrollbars with the parent window.

## Content

### Office2007 Look and Feel for ScrollBars

**TreeViewAdv** provides support for Office2007 Scrollbars with all three color schemes.

#### Figure 1454: Scrollbars with Sizing Grip
- TreeViewAdv integrates with Office2007 Scrollbars.

#### Figure 1455: Office 2007 Scrollbars
- The control supports Blue, Silver, and Black color schemes for Office2007 Scrollbars.

#### Color Scheme Selection
The Office2007ScrollBarsColorScheme property can be used to select the desired color scheme.

#### Splitter Window Support
When the control is used inside a splitter window and shares scrollbars with the parent control or window, setting the `FillSplitterPane` property to `true` toggles this functionality.

#### TreeViewAdv Properties

| TreeViewAdv Properties    | Description                                                                 |
|---------------------------|-----------------------------------------------------------------------------|
| FillSplitterPane           | Provides support for using the control inside a dynamic splitter window and sharing the scrollbars with the parent window. |

#### Code Example (C#)
```csharp
this.treeViewAdv1.AllowIncreaseSmallChange = true;
this.treeViewAdv1.FillSplitterPane = true;
```

### Summary
- TreeViewAdv supports Office2007 Scrollbars via the Office2007ScrollBarsColorScheme property.
- Integrated support for using TreeViewAdv inside splitter windows is enabled via the FillSplitterPane property.

### API Reference

#### Properties
- **FillSplitterPane**: A boolean property that enables sharing scrollbars with the parent window when used inside a splitter window.
- **Office2007ScrollBarsColorScheme**: A property for selecting the Office2007 Scrollbars color scheme.

## Code Examples (C#)
The following code enables both the increased small change and splitter pane support:

```csharp
this.treeViewAdv1.AllowIncreaseSmallChange = true;
this.treeViewAdv1.FillSplitterPane = true;
```

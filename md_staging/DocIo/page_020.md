# Essential DocIO

## Overview
- This page provides guidelines on how to view and browse through the samples of Essential DocIO using the ASP.NET MVC Sample Browser.
- It includes information on where to locate the source code for DocIO under different platforms.

## Content

### ASP.NET MVC Sample Browser

Figure 9 shows the samples displayed in the ASP.NET MVC Sample Browser. These samples are categorized and organized to demonstrate various features of Essential DocIO.

#### Sample Displayed in the Browser
The sample browser interface includes:

- **Navigation Menu**: On the left side, it features categories like `ProductShowCase`, `Getting Started`, `Editing`, `Formatting`, and more.
- **Featured Samples**: On the right side, highlighted samples such as `Employee Report`, `Table of Content`, `Clone and Merge`, `Forms`, and `Update Fields` are displayed.
- **Description**: The main section provides a detailed description of the library, stating:
  - High-performance server-side .NET library that can read, write, and manipulate Microsoft Word files.
  - Full-fledged object model similar to Microsoft Office COM libraries.
  - Fully integrated into ASP.NET MVC.

#### Instructions for Browsing
1. **Note:** You can view the samples in any one of the three options displayed.
2. **Select any DocIO sample and browse through the features.**

### Source Code Location

The source code for DocIO under different platforms is available at the following default locations:

- **Windows**: `[System Drive]:\Program Files\Syncfusion\Essential Studio\[Version Number]\Windows\DocIO.Windows\Src`
- **ASP.NET**: `[System Drive]:\Program Files\Syncfusion\Essential Studio\[Version Number]\Web\DocIO.Web\Src`
- **WPF**: `[System Drive]:\Program Files\Syncfusion\Essential Studio\[Version Number]\WPF\DocIO.WPF\Src`
- **ASP.NET MVC**: `[System Drive]:\Program Files\Syncfusion\Essential Studio\[Version Number]\MVC\DocIO.MVC\Src`

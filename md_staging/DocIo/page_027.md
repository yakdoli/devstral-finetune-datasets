# Essential DocIO

## Overview
- Guides through the process of creating an ASP.NET Web Site and deploying Essential DocIO.
- Explains the steps to create a WPF Application in Microsoft Visual Studio.

## Content

### ASP.NET Web Site
Figure 12: ASP.NET Web Site template selected in the New Web Site Dialog Box  

An ASP.NET application is created.

1. Now you need to deploy Essential DocIO into this ASP.NET application. Refer **ASP.NET** topic for detailed info.

### WPF Application

1. Open Microsoft Visual Studio. Go to **File** menu and click **New Project**. In the **New Project** dialog box, select **WPF Application** template, name the project and click **OK**.

### Example Steps for WPF Application  
- **Step 1:** Launch Microsoft Visual Studio.
- **Step 2:** Navigate to the **File** menu.
- **Step 3:** Select **New Project**.
- **Step 4:** In the **New Project** dialog box, choose the **WPF Application** template.
- **Step 5:** Provide a name for the project.
- **Step 6:** Click **OK** to create the project.

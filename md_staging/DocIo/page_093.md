## Content

### Essential DocIO Features
The Essential DocIO library has a simple yet powerful object model that provides the ability to customize the document to a great extent. The following are some of the important features available in Essential DocIO:

- **Create and edit a new document.**
- **Modify existing MS Word documents.**
- **Ability to read and write BuiltIn and Custom document properties.**
- **API to manipulate Page Setup settings.**
- **Inserting document page Header and Footer.**
- **Read and write paragraphs in a document.**
- **Formatting text in the document.**

# Essential DocIO

## Overview
- Overview of Essential DocIO, a .NET library for creating, reading, and manipulating Microsoft Word files.
- Features a full-fledged object model similar to Microsoft Office COM libraries.
- Built from scratch in C# without using COM interop, making it suitable for systems without Microsoft Word installed.
- Demonstrates various document creation and manipulation capabilities.

## Content

### WPF

**Syncfusion Essential DocIO WPF**

Essential DocIO is a .NET library that can read and write Microsoft Word files. It features a full-fledged object model similar to the Microsoft Office COM libraries. It does not use COM interop and is built from scratch in C#. It can even be used on systems that do not have Microsoft Word installed.

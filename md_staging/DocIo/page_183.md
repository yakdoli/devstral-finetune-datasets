# Essential DocIO

## Overview
- You can add comments to a Word document.
- The process involves selecting the text to which you want to apply the comment, then using the Insert menu feature.
- The Comment option is used to insert comments into the document.

## Content

### Adding Comments to a Document

To add a comment to a document:
1. Select the text to which you want to apply the comment.
2. Open the `Insert` menu.
3. Click on the `Comment` option.

#### Interface Overview

The `Insert` menu includes various options for adding features to a document. The `Comment` option is highlighted under this menu. 

**Figure 63: Comment option in Insert Menu**

![Figure 63: Comment option in Insert Menu](https://i.imgur.com/example_image_63.png)

Once the comment is added, it appears as a separate annotation within the document, pointed to the relevant text.

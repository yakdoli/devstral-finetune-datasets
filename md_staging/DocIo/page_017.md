# Essential DocIO

## Overview

- Demonstrates the feature of document protection and manipulation capabilities provided by Essential DocIO.
- Shows how to generate and protect Word documents with various security settings.

## Content

### Document Protection Sample

#### Description
This sample illustrates Essential DocIO's support for protecting Word documents. The resultant document requires MS Word Viewer or MS Word to be viewed. The following features are highlighted:

- **Protection Type**: Allows specifying how the document can be edited, such as allowing only form fields or completelyRead-only settings.
- **Password**: Enables setting a password to restrict access or editing.
- **Save As**: Options to save the document in various Word versions (e.g., Word 97-2003, Word 2007, Word 2010).

#### Interface
The interface includes input fields for entering the protection type and password, as well as dropdowns to select the save format for the generated document. There is also a button labeled "Generate Document" to initiate the document generation process.

#### Sample Demonstration
The image shows the interface for configuring and generating a protected Word document. It includes fields and options for setting protection settings and generating the document.

### ASP.NET Web Application

#### Steps to Create an ASP.NET Web Site
1. **Open Microsoft Visual Studio**.
2. **Go to File menu and click New Web Site**.
3. **In the New Website dialog box, select ASP.NET Web Site template**.
4. **Name the website** and click `OK`.

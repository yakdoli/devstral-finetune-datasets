# Document Properties Dialog Box

## Overview
- Provides a comprehensive view of the Document Properties dialog box with its various tabs and their functionalities.
- Describes the editable and non-editable properties of a document, allowing users to view and manage document metadata.

## Content

### Figure 25: Document Properties Dialog Box
The Document Properties dialog box contains the following tabs:

- **General (not editable)**: 
  - Contains general document information such as file name, type, size, location, creation, modification, and access date.

- **Summary (editable)**: 
  - Allows setting or modifying the document properties such as author name, title, company, subject, keywords, and so on.

- **Statistics (not editable)**: 
  - Displays the statistics of the document such as creation, modification, access, and print dates, revision number, characters, words, pages number, and so on.

- **Contents (not editable)**: 
  - Displays the content of the document.

- **Custom (editable)**: 
  - Enables users to create their own properties by combining their name, type, and value. 
  - For example, marking a document as "Checked" can be done by creating a custom property with the Name as "Checked", Type as "Text", and Value as "your name".

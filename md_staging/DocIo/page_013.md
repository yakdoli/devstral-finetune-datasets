# Essential DocIO

## Content

### Overview

- This section provides details on viewing the DocIO samples for various platforms, focusing on the Windows Forms platform.
- It explains how to access the sample browser within the Dashboard for Windows Forms.

### Steps to View DocIO Samples in Windows Forms

#### Windows

1. **Dashboard Window**:  
   Open the Dashboard window.  
   In the **Reporting Edition** panel, locate and click on **Run Samples for Windows Forms**.  
   This action will display the **Windows Forms Sample Browser** window.

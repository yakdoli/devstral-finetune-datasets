## Content

### Filtering Columns in Grid Grouping Control

The **GroupingGridFilterBarExt** class provides support to filter a column in the Grid Grouping control by its display member instead of the value member. This is accomplished by implementing a custom filter bar cell by replacing the default filter bar cell.

#### Figure: Filtering a column in the Grid Data Bound Grid by its Display Member
- The figure illustrates a Grid Data Bound Grid where the "Supplier" column is filtered by its Display Member, showing a dropdown list of supplier names.

#### Code Example: Wiring the GroupingGridFilterBarExt to the Grid Grouping Control

1. **Using C#**

```csharp
private GroupingGridFilterBarExt gGFilter;
this.gGFilter = new GroupingGridFilterBarExt();
this.gGFilter.WireGrid(this.gridGroupingControl1);
```

2. **Using VB.NET**

```vb
Private gGFilter As GroupingGridFilterBarExt
Me.gGFilter = New GroupingGridFilterBarExt()
Me.gGFilter.WireGrid(Me.gridGroupingControl1)
```

### Notes
- This functionality is particularly useful in scenarios where you need to filterGrid data based on user-friendly display text rather than underlying values.
- Ensure that the Grid Grouping control is properly initialized before wiring the `GroupingGridFilterBarExt`.

# Essential Grid for Windows Forms

## Overview
- Provides information about the properties, methods, and events of the GridPDFConverter class.
- Focuses on configuring the display of headers and footers in PDF documents generated using the Grid control.

## Content

### Properties
#### ShowHeader
- **Description**: This property gets or sets a value indicating whether the header should be displayed in the PDF document. Default value is set to `false`.
- **Purpose**: Allows users to control whether a header is included in the PDF document.

#### Example Usage

##### Using C#
```csharp
pdfConvertor.ShowHeader = true;
```

##### Using VB.NET
```vb
pdfConvertor.ShowHeader = True
```

#### ShowFooter
- **Description**: This property gets or sets a value indicating whether the footer should be displayed in the PDF document. Default value is set to `false`.
- **Purpose**: Allows users to control whether a footer is included in the PDF document.

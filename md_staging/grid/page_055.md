### Query Builder Interface

The Query Builder interface allows you to construct and manage database queries visually. Key components include:

- **Suppliers Table**: The query builder presents available columns from the Suppliers table.
- **Column Selection**: You can select specific columns (e.g., Address, City, CompanyName, ContactName) or choose to include all columns using the `*` (All Columns) option.
- **Generated SQL Query**: The interface displays a preview of the SQL query being built, showing the selected fields and table.

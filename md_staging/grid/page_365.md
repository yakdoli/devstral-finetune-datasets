## Overview
- Functions to calculate tangent values, hyperbolic tangent values, and formatting numerical values as text.

## Content

### TAN
#### Overview
Returns the tangent of the given angle.

#### Syntax
```plaintext
TAN(number)
```
where:
- `number` is the angle in radians for which, you want the tangent.

---

### TANH
#### Overview
Returns the hyperbolic tangent of a number.

#### Syntax
```plaintext
TANH(number)
```
where:
- `number` is any real number.

#### Remarks
- The formula for the hyperbolic tangent is,
  \[
  \tanh(z) = \frac{\sinh(z)}{\cosh(z)}
  \]

---

### TEXT
#### Overview
Converts a value to text in a specific number format.

#### Syntax
```plaintext
TEXT(value, format_text)
```
where:
- `value` is the number to be converted.
- `format_text` is the specified number format for the text output.

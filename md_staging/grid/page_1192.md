### 4.15.1 Zooming Grid Controls

To enable the zooming support for a control, you need to pass the control as a parameter to implement zooming functionality. Then, you need to define the percentage value of zooming by using the `zoomGrid` method. The following code examples illustrate how to zoom the entire grid of the `GridControl`, `GridDataBoundGrid`, and `GridGrouping` controls.

#### For GridControl

##### [C#]
```csharp
ZoomGrid zoom = new ZoomGrid(this.gridControl1);
zoom.zoomGrid(this.trackBar1.Value.ToString());
```

##### [VB]
```vb
Dim zoom As New ZoomGrid(Me.gridControl1)
zoom.zoomGrid(Me.trackBar1.Value.ToString())
```

## API Reference

### Overview
- Enables zooming functionality for grid controls.
- Passes the control as a parameter to implement zooming.
- Defines the zoom percentage using the `zoomGrid` method.

### Methods
- `zoom_grid(value: string)`: Zooms the grid based on the percentage value passed as a string.

### Parameters
| Name    | Type  | Description                          | Default | Required |
|---------|-------|----------------------------------------|---------|----------|
| value   | string | Specifies the zoom percentage as a string. | None    | Yes      |

### Returns
- Type: `void`
- Description: Applies the zoom effect to the grid.

### Exceptions
- Throws an exception if the zoom percentage is not a valid string representation of a percentage.

### Code Examples

#### Example 1: Zooming a GridControl
```csharp
ZoomGrid zoom = new ZoomGrid(this.gridControl1);
zoom.zoomGrid("150%");
```

#### Example 2: Zooming a GridDataBoundGrid
```csharp
ZoomGrid zoom = new ZoomGrid(this.gridDataBoundGrid1);
zoom.zoomGrid("200%");
```

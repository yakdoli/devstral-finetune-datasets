# Essential Grid for Windows Forms

## Overview
- This section explains the **CellDrawn Event**.
- The event handler receives an argument of type `GridDrawCellEventArgs`.
- Describes the invocation of the event in both C# and VB.NET.

## Content

### 4.3.4.14.2.3 CellDrawn Event

**Description:**
The `CellDrawn` event occurs for every cell after the `grid` has drawn the specified cell. The event handler receives an argument of type `GridDrawCellEventArgs` containing data related to this event. This event can be invoked as follows:

#### C#
```csharp
this.gridGroupingControl.TableControlCellDrawn+= new Syncfusion.Windows.Forms.Grid.GridDrawCellEventHandler(TableControl_CellDrawn);
```

#### VB.NET
```vb
AddHandler Me.gridGroupingControl.TableControlCellDrawn, AddressOf TableControl_CellDrawn
```

#### GridTableControlCellDrawnEventArgs Properties
The following `GridTableControlCellDrawnEventArgs` properties provide information specific to this event:

| Properties             | Description                                                                   |
|------------------------|-------------------------------------------------------------------------------|
| Bounds                 | Cell boundaries including borders and margins.                               |
| Renderer               | The `GridCellRendererBase` associated with the cell.                         |
| Graphics               | The `Graphics` context.                                                      |
| IsBackgroundErased     | Specifies if the cell background has already been drawn with the interior as specified in the style object. |
| ColIndex               | Specifies the column index of the cell.                                      |
| RowIndex               | Specifies the row index of the cell.                                         |
| Cancel                 | Specifies a value to indicate if this event should be canceled.            |

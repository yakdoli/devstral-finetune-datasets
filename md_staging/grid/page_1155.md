# Essential Grid for Windows Forms

## Overview

- Describes the `ThemeChanged` event and its parameters.
- Explains how to apply the Metro theme to a control using the `GridVisualStyles` property.
- Provides code examples in C# and VB to demonstrate the application of the Metro theme.

## Content

### Events

| Event        | Parameters                         | Description                                                                 |
|--------------|------------------------------------|-----------------------------------------------------------------------------|
| **ThemeChanged** | Object sender, EventArgs | Occurs when the **ThemesEnabled** property is changed.                   |

### 4.10.2 Applying Metro Theme to a Control

You can apply the Metro theme to an application by setting the `GridVisualStyles` property as **Metro**. The following code example illustrates how to apply the Metro theme for the GridGrouping control.

#### C# Code Example

```csharp
this.gridGroupingControl1.GridVisualStyles = GridVisualStyles.Metro;
```

#### VB Code Example

```vb
Me.gridGroupingControl1.GridVisualStyles = GridVisualStyles.Metro
```

## API Reference

### Properties

- **GridVisualStyles**: Specifies the visual style applied to the control. It can be set to **Metro** to apply the Metro theme.

## Code Examples

### C# Example

```csharp

// ...

```

### VB Example

```vb
Imports Syncfusion.Windows.Forms.Grid

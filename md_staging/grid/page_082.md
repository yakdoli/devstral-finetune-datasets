# Essential Grid for Windows Forms

## 3.1.4.1 Basic Grid Data Bound Grid

### Overview
- Learn how to use the designer to place a Grid Data Bound Grid on a form.
- Create a new Windows Application project in Visual Studio .NET.
- Add a SQLDataAdapter to the form to create a connection to the NorthWind database using the Data Adapter Wizard.

### Content

#### Creating a New Windows Application
1. **Create a New Project**:
   - Open Visual Studio .NET.
   - Navigate to the **File -> New -> Project** menu option.
   - Select **Visual C# Projects**.
   - Choose **Windows Application** from the templates.
   - Name the project **DBGridTutorial**.
   - Specify the location: `C:\CSToVB\samples\cs`.
   - Click **OK** to create the project.

   ![Figure: Creating a New Windows Application](https://image.png)

2. **Designer Surface**:
   - You now have an empty form on the design surface.
   - Open the **Data** section of your toolbox.
   - Drag a **SQLDataAdapter** onto your form.
   - This will open the **Data Adapter Wizard**.

3. **Using the Wizard**:
   - Use the wizard to create a connection to the **NorthWind database**.
   - This database is installed as part of the **.NET Framework ADO.NET samples**.

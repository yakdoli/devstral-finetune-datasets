# Essential Grid for Windows Forms

## Overview

- The Essential Grid is a powerful grid control with detailed customization options, including multi-level undo-redo, shared scrollbar support, and unmatched extensibility.
- Various samples and demonstrations are provided, showcasing Pivot Grid, Formula Grid, Virtual Grid, and more.
- Grid grouping is demonstrated through samples like Grouping in Outlook, LINQ to SQL, and Hierarchical View.

**Description:**

Essential Grid is a powerful grid control that is implemented using a unique styles architecture. This grid control, which is similar to Microsoft Excel, allows extremely detailed customization down to the cell level. Multi-level undo-redo, shared scrollbar support, data view separation, floating cells, more than 18 cell types, and unmatched extensibility are the unique features of the Essential Grid.

### Data Grid

#### Overview

- The Data Grid Grouping Control is a column-row-oriented grid used to bind to a data source.
- Essential Grouping allows easy grouping, sorting, filtering, and summarizing of data.
- The control can display true nested grids with hierarchical data and multiple, unrelated tables in one grid.

# Essential Grid for Windows Forms

## Overview
- Demonstrates how to wire filters to a grid's column using code.
- Explains the various documentation segments provided by Syncfusion for essential grid controls.

## Content

### Code Examples

#### C#
```csharp
this.gridGroupingControl1.TableDescriptor.Columns[0].Appearance.FilterBar.CellType = "DynamicFilterCell";
```

#### VB
```vb
Me.gridGroupingControl1.TableDescriptor.Columns(0).Appearance.FilterBar.CellType = "DynamicFilterCell";
```

Other filters can be wired to the grids column by using the previous code.

# Essential Grid for Windows Forms

## Data Source Configuration Wizard

### Choose Your Database Objects

The `Data Source Configuration Wizard` allows you to select the database objects you want to include in your dataset. The screen shot below shows the available options:

- **Tables:**  
  - Categories  
  - Customers  
  - Employees  
  - Order Details  
  - Orders  
  - Products  
  - Shippers  
  - Suppliers  

- **Views:**  
  - Category Sales for 1995  
  - Current Product List  
  - Invoices  

The `DataSet name` field is pre-populated with `NWinDDataset`.

### Action Buttons
- `< Previous`  
- `Next >`  
- `Finish`  
- `Cancel`

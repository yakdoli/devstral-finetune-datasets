# Essential Grid for Windows Forms

## Overview
- The GridControl Designer allows modification of cell contents/styles and general grid properties.
- Figure 45 shows the Grid Control Context Menu with various options for grid manipulation.

## Content

### GridControl Designer
The GridControl Designer window is displayed. By using the GridControl Designer, the cell contents/styles, as well as general grid properties can be modified.

#### Grid Control Context Menu (Figure 45)
The context menu includes options such as:
- **View Code**
- **Bring to Front**
- **Send to Back**
- **Align to Grid**
- **Lock Controls**
- **Edit**
- **Touch It**
- **Edit base styles**
- **Select 'Form1'**
- **Cut**
- **Copy**
- **Paste**
- **Delete**
- **Properties**

### Interaction with GridControl
Using these context menu options, developers can perform tasks such as:
- Editing grid properties directly.
- Manipulating the grid visually by bringing it to the front or sending it to the back.
- Aligning controls to the grid.
- Locking controls to prevent accidental movement.
- Accessing and modifying the base styles of the grid.

### WinForms-specific Features
- The GridControl Designer integrates seamlessly with Windows Forms, providing a rich set of options for customizing and managing grid behavior.

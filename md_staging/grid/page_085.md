# Essential Grid for Windows Forms

## Overview
- Guides the user through the Data Adapter Configuration Wizard interface.
- Details the process of selecting the Query Builder and configuring SQL statements for dataset operations.
- Provides instructions on table selection in the Add Table dialog.

## Content

### Data Adapter Configuration Wizard
The Data Adapter Configuration Wizard is designed to help generate SQL statements. The Select statement entered will be used to create Insert, Update, and Delete statements. Users have the option to either manually type in their SQL Select statement or use the Query Builder for a graphical design experience.

#### Interface Description
- **Generate the SQL statements**: The Select statement entered here is used to create Insert, Update, and Delete statements.
- **Input Field**: Enter the SQL Select statement or use the Query Builder to design the query.
- **Data Loading**: Specifies what data the data adapter should load into the dataset.

Figure 51: Select the Query Builder

- **Buttons**:
  - **Advanced Options...**: Provides additional customization options.
  - **Query Builder...**: Opens the graphical query builder interface.
  - **Cancel**: Cancels the wizard process.
  - **Back**: Returns to the previous step.
  - **Next >**: Proceeds to the next step in the wizard.
  - **Finish**: Completes the wizard process.

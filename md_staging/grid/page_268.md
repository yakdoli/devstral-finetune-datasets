### 4.1.4.6.6.40 COSH

Returns the hyperbolic cosine of a number.

#### Syntax

**COSH(number)**, where:

- **number** is any real number for which you want to find the hyperbolic cosine.

#### Remarks

The formula for the hyperbolic cosine is,

\[
\cosh(z) = \frac{\mathrm{e}^z + \mathrm{e}^{-z}}{2}
\]

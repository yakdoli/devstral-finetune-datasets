## Code Examples

Below are some examples demonstrating the usage of the Essential Grid for Windows Forms.

### Example: Binding Data to the Grid

```csharp
using Syncfusion.Windows.Forms.Grid;

gridControl1.DataSource = dataSource; // dataSource is your data source object
```

### Example: Customizing Cell Styles

```csharp
gridControl1.GetCellStyles[0,0].BackColor = Color.LightGray;
gridControl1.GetCellStyles[0,0].Text = "Custom Text";
```

### Example: Handling Grouping Events

```csharp
gridGroupingControl1.GroupAdded += (sender, args) => {
    // Handle GroupAdded event
    MessageBox.Show("Group Added!");
};
```

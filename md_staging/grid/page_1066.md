### 4.3.4.17.1 Searching for text in multiple columns

This feature can be used for performing searches either in specified columns or in all of the grid columns. Only the rows containing the search text will be filtered and displayed.

Searches can be performed in two modes:

1. Search for text in all columns.
2. Search for text in particular columns.

### Method Table

| Method            | Prototypes                                                                                  | Description                                                               |
|-------------------|---------------------------------------------------------------------------------------------|---------------------------------------------------------------------------|
| Search()         | this.gridGroupingControl.TableDescriptor.Search("SearchText", List<GridColumnDescriptor>); | It is called for performing the searching operation in the columns. |

## API Reference

- **Namespace**: The name of the namespace containing the `gridGroupingControl` class.
- **Class**: `gridGroupingControl`
- **Method**: `Search()`
  - **Parameters**:
    - `SearchText`: The text to search for in the grid columns.
    - `List<GridColumnDescriptor>`: A list specifying the columns to search in.
  - **Returns**: Performs the searching operation in the specified columns.
  - **Description**: Filters and displays only the rows containing the search text.

## Code Examples

```csharp
// Example: Searching in all columns
this.gridGroupingControl.TableDescriptor.Search("SearchText", null);

// Example: Searching in particular columns
List<GridColumnDescriptor> columnsToSearch = new List<GridColumnDescriptor> { column1, column2 };
this.gridGroupingControl.TableDescriptor.Search("SearchText", columnsToSearch);
```

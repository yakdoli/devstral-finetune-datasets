# Essential Grid for Windows Forms

## Overview
- Demonstrates the process of setting up a data source for the GridGroupingControl.
- Explains how to configure the appearance and look and feel of the control.
- Provides instructions on setting the Anchor property to allow resizing with the form.

## Content

### Data Setup
- **Data Source Configuration**:
  - The GridGroupingControl's data source is set to `dataset11.Suppliers`.
  - The GridGroupingControl displays a single item from the dataset.

### Appearance and Look and Feel
- Options are available for previewing and editing the schema, as well as copying, pasting, and saving the schema.
- The control's appearance can be customized by copying, pasting, saving, or choosing a specific Look and Feel option.
- The "Office2007Blue" Skin is selected for the GridGroupingControl.

### Documentation and Support
- Links to Online Documentation, Evaluation Center, Forums Support, and Direct-Trac Support are provided for additional resources.

### Setting the Anchor Property
1. Set the `Anchor` property of the Grid Grouping control to `All` so that the control can be easily resized with the form.
2. This configuration ensures the control resizes properly when the form is resized.

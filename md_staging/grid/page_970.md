## Essential Grid for Windows Forms

### Overview
- A demonstration of a Microsoft Outlook 2007-like interface implemented using Grid Grouping Control.
- Features include support for Office 2007 themes, grouping of columns, custom selection navigation, dynamic formatting, and event handling.

## Content

### Figure 362: Microsoft Outlook 2007-like Interface implemented by using Grid Grouping Control
This figure demonstrates a user interface similar to Microsoft Outlook 2007, showcasing the capabilities of the Grid Grouping Control in managing mail items, dates, sizes, and other attributes.

#### Features
The sample demonstrates the following major features:

- **Office 2007 themes - Blue, Black, and Silver** to see the effects of the application.
- **Grouping of columns** in the mails display area. Right-click on the column headers and select `ArrangeBy` in the column.
- **Custom selection navigation** with all arrow keys.
- **Custom grouping** for dates, size, priority, and flag fields.
- **Dynamic formatting** when switching between column-set view and normal view.
- **Events** when deleting records.

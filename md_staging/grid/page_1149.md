# Essential Grid for Windows Forms

## Overview
- Demonstrates the Office2010 Silver theme in the `Binding Array List Demo`.
- Explains the properties and events related to setting up and customizing Essential Windows Grids.
- Includes detailed descriptions of `GridVisualStyles`, `GridOfficeScrollbars`, and `Office2010ScrollbarColorScheme` properties, as well as the `ThemeChanged` event.

## Content

### Figure 466: Office2010 Silver theme

The following screenshot showcases the Office2010 Silver theme applied to a grid:

**Figure: Office2010 Silver theme**
- **Grid Layout**: Displays an array list with 14 items, including columns for `ID`, `Name`, and `Button`.
- **Customization Options**: Allows selection of theme and scrollbar styles using dropdowns for `Office2010Theme` and `MS-OfficeScrollBars`.

### Tables for Properties and Events

#### Properties

**Table 16: Properties Table**

| Property                       | Description                                                                 | Data Type                          |
|--------------------------------|---------------------------------------------------------------------------|------------------------------------|
| GridVisualStyles              | - This is an Enumeration type property.<br>- This property is used to get or set the VisualStyles (skins) like Office2010, Office2007, Office2003. | Syncfusion.Windows.Forms.GridVisualStyles |
| GridOfficeScrollbars          | - This is an Enumeration type property.<br>- This property is used to get or set the Office like scrollbars. | Syncfusion.Windows.Forms.OfficeScrollBars |
| Office2010ScrollBarsColorScheme | - This is an Enumeration type property.<br>- This property is used to get or set the style of Office2010 scroll bars | Syncfusion.Windows.Forms.Office2010ColorScheme |

#### Events

The following event is used when applying the Office2010 theme to Essential Windows Grids:

- **Event**: ThemeChanged
  - **Parameters**: Object sender, EventArgs e
  - **Description**: Occurs when the ThemesEnabled property is changed.

## API Reference

### Properties
- `GridVisualStyles`: Manages different visual styles for the grid.
- `GridOfficeScrollbars`: Manages Office-like scrollbars for the grid.
- `Office2010ScrollbarColorScheme`: Manages the color scheme for Office2010 scrollbars.

### Events
- `ThemeChanged`: Triggered when the ThemesEnabled property is changed.

## Code Examples

```csharp
// Example of applying the Office2010 theme
gridGridVisualStyles = GridVisualStyles.Office2010;
gridGridOfficeScrollbars = GridOfficeScrollbars.Office2010;
gridOffice2010ScrollbarColorScheme = Office2010ColorScheme.Silver;

// Event handling for ThemeChanged
gridThemeChanged += (sender, e) => 
{
    // Handle ThemeChanged event
};
```

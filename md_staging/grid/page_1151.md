# Essential Grid for Windows Forms

## Overview
- Explains how to access and configure the Grid Style Demo in the Syncfusion Windows Forms Grid.
- Details enhanced visual styles for Windows Forms Grid controls.
- Instructs on scheduling samples and opening sample projects.

## Grid Style Configuration

```csharp
// Setting Office2010 ScrollBars Color Scheme
Me.gridGroupingControl1.Office2010ScrollBarsColorScheme = Office2010ColorScheme.Black;
```

### Sample Link
To get the Schedule samples from the dashboard:

1. **Open Essential Studio Dashboard**:
   - Select Start -> All Programs -> Syncfusion -> Essential Studio <<Version Number>> -> Dashboard.

2. **Select Run Locally Installed Samples**:
   - From the Windows Forms drop-down list on the User Interface pane.

3. **Expand Grid samples**:
   - In the left panel of the sample browser.

4. **Select Grid Style Demo**:
   - Expand the "Appearance" subsection.

5. **Click Run Sample**:
   - In the right panel.

#### To open sample project:
1. **Navigate to the sample location**:
   - `<<Sample Installation Location>>\Syncfusion\Essential Studio<<Version Number>>\Windows\Grid.Windows\Samples\2.0\Appearance\Grid Style Demo`

2. **Open the sample projects**:
   - The location contains subfolders CS and VB to open projects based on the application language.

## 4.9 Enhanced Visual Styles for the Syncfusion Windows Grids

**Feature Overview**:
This feature allows the application of enhanced visual styles for the Windows Forms Grid controls, including Grid control, GridGrouping control, GridDataBoundGrid, GridList control, GridRecordNavigation control, associated scrollbars, combo box drop-down container, filter (dynamic filter and filter bar), selected row, and group drop area.

By default, the `EnableLegacyStyle` property value is `true`.

### Use Case Scenarios
You can use this control to apply a skin to the entire control, including scrollbars, combo boxes, drop-down containers, filters, and more.

### Properties

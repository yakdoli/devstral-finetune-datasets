# Essential Grid for Windows Forms

## Overview
- This section explains how to handle cell click events in the `GridGroupingControl` using C# and VB.NET.
- Provides details about the `GridCellClickEventArgs` properties and includes examples for handling cell click events.

## Content

### Event Handling

The following examples demonstrate how to attach event handlers to the `TableControlCellClick` event for both C# and VB.NET:

#### C#

```csharp
this.gridGroupingControl1.TableControlCellClick += new 
Syncfusion.Windows.Forms.Grid.GridCellClickEventHandler(TableControl_CellClick);
```

#### VB.NET

```vb
AddHandler Me.gridGroupingControl1.TableControlCellClick, AddressOf 
TableControl_CellClick
```

### GridCellClickEventArgs Properties

The following `GridCellClickEventArgs` properties provide information specific to the cell click event:

| **Properties**       | **Description**                                                                         |
|----------------------|-----------------------------------------------------------------------------------------|
| `IsOverImage`       | Indicates if the mouse was over the image in a static cell when the mouse was released. |
| `MouseEventArgs`    | The `MouseEventArgs` originating this event.                                           |
| `ColumnIndex`       | Specifies the column index of the cell.                                                |
| `RowIndex`          | Specifies the row index of the cell.                                                   |
| `Cancel`            | Specifies a value to indicate if this event should be canceled.                       |

### Example

#### C#

```csharp
private void TableControl_CellClick(object sender, GridCellClickEventArgs e)
{
    Console.WriteLine("CellClicked");
}
```

#### VB.NET

```vb
Private Sub TableControl_CellClick(ByVal sender As Object, ByVal e As GridCellClickEventArgs)
    Console.WriteLine("CellClicked")
End Sub
```

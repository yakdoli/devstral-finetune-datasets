## Grid Properties for Windows Forms

### Overview
- This section lists the properties of the `Grid` control in Windows Forms, detailing their descriptions, access type, and data types.
- Focuses on configuring various aspects of grid display and behavior, including formatting, text display, and interaction properties.

### API Reference

#### Namespace: Syncfusion.Windows.Forms.Grid
##### Properties

- **DecimalSeperator**:  
  - Type: char  
  - Description: Gets or sets the character to use when a decimal separator position is specified.

- **Description**:  
  - Type: string  
  - Description: Gets / sets the text that is shown in check box or pushbuttons.

- **DisplayMember**:  
  - Type: string  
  - Description: Indicates the property to display for the items in this control.

- **Enabled**:  
  - Type: bool  
  - Description: Specifies if the cell can be activated as the current cell or if the cell should be skipped when moving the current cell.

- **FloatCell**:  
  - Type: bool  
  - Description: Gets / sets if text can float into the boundaries of a neighboring cell.

- **FloodCell**:  
  - Type: bool  
  - Description: Gets / sets if this cell can be flooded by a previous cell.

- **Font**:  
  - Type: GridFontInfo  
  - Description: The font for drawing text.

- **Foreground**:  
  - Type: Color  
  - Description: The foreground color used to display text and graphics in the control.

- **ForeSegments**:  
  - Type: bool  
  - Description: Determines if the foreground is segmented (Progress bar).

- **Format**:  
  - Type: string  
  - Description: Contains format information of a cell.

### Code Examples

#### Example: Setting Grid Properties in C#

```csharp
using Syncfusion.Windows.Forms.Grid;

Grid control = new Grid();
control.DecimalSeperator = '.';
control.Description = "Check me";
control.DisplayMember = "ItemName";
control.Enabled = true;
control.FloatCell = false;
control.FloodCell = false;
control.Font = new GridFontInfo("Arial", 12);
control.Foreground = Color.Black;
control.ForeSegments = false;
control.Format = "0.00";
```

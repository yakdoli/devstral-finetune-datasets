### Events

#### Overview of Events

The following events occur during specific interactions or processes with the grid control:

- **Grid Querying for Style Information**
- **User Interaction with Cells**
- **Card Model Saving Information**
- **User Clicking a Push Button**

#### Events List

| Event            | Description                                                                 | Arguments                                                                                                   | Type   |
|------------------|--------------------------------------------------------------------------- |-----------------------------------------------------------------------------------------------------------|-------|
| `QueryCardCellInfo` | Occurs when the card model queries for style information about a specific cell. | `public<br/>QueryCardCellInfoEventArgs(GridQueryCellInfoEventArgs e, GridCardView cardView)` | Event |
| `CellClick`       | Occurs when the user clicks inside a cell.                               | `public<br/>CardCellClickEventArgs(GridCellClickEventArgs e, GridCardView cardView)`     | Event |
| `SaveCardCellInfo` | Occurs when the card model is about to save style information about a specific cell. | `public<br/>SaveCardCellInfoEventArgs(GridSaveCellInfoEventArgs e, GridCardView cardView)` | Event |

## API Reference

### Methods

#### `IsValueCell`
##### Description
Indicates whether the specified cell is a value cell.

##### Parameters
- `rowIndex`: The row index of the cell.
- `colIndex`: The column index of the cell.

##### Return Type
- `bool`: Returns `true` if the cell is a value cell; otherwise, `false`.

#### `IsCardCaption`
##### Description
Indicates whether the specified cell is a caption cell.

##### Parameters
- `rowIndex`: The row index of the cell.
- `colIndex`: The column index of the cell.

##### Return Type
- `bool`: Returns `true` if the cell is a caption cell; otherwise, `false`.

#### `GetCardCellType`
##### Description
Specifies the type of the specified card cell.

##### Parameters
- `rowIndex`: The row index of the cell.
- `colIndex`: The column index of the cell.

##### Return Type
- `CardCellType`: Returns the type of the card cell.

### Events

#### `QueryCardCellInfo`
- **Description**: Triggered when the card model queries for style information about a specific cell.
- **Arguments**: `GridQueryCellInfoEventArgs e`, `GridCardView cardView`

#### `CellClick`
- **Description**: Triggered when the user clicks inside a cell.
- **Arguments**: `GridCellClickEventArgs e`, `GridCardView cardView`

#### `SaveCardCellInfo`
- **Description**: Triggered when the card model is about to save style information about a specific cell.
- **Arguments**: `GridSaveCellInfoEventArgs e`, `GridCardView cardView`

#### `PushButtonClick`
- **Description**: Triggered when the user clicks a push button.
- **Arguments**: `GridCellPushButtonClickEventArgs e`, `GridCardView cardView`

# Overview

This page provides an overview of advanced functionalities available in the **Essential Grid for Windows Forms** component. Specifically, it highlights the built-in support for adding summaries, applying filters, and sorting data within the Grid Grouping Control. These features enable users to manipulate and analyze data effectively in Windows Forms applications.

## Content

### Support to Add Summaries

#### Description
The grid grouping control supports the addition of summaries to help users aggregate and interpret data efficiently.

#### Figure: Support to add Summaries  
![Figure 247: Support to add Summaries](#)

The grid displays data related to football teams, including statistics such as losses, ties, wins, and years. The interface allows users to view aggregated summaries of the data at the bottom of the grid.

#### Key Points:
- The grid allows users to display summarized data for easy aggregation.

### Filtering

#### Description
The grid grouping control comes with in-built support for the **FilterBar**, which enables users to display subsets of records based on user-defined criteria.

#### Figure: Filtering support in the Grid Grouping Control  
![Figure 248: Filtering support in the Grid Grouping Control](#)

In this example, the grid shows a set of company records with columns for **Address**, **City**, and **Company Name**. Users can filter the data to view specific subsets of records based on criteria they define.

#### Key Points:
- The filter bar feature helps in displaying a subset of records that satisfy user-defined conditions.

### Sorting

#### Description
The grid grouping control allows you to sort the data against an unlimited number of columns, either in ascending or descending order. When editing or modifying data, the grid automatically adjusts the record values to maintain the sorting order.

#### Figure: Sorting support in the Grid Grouping Control  
![Figure: Sorting support in the Grid Grouping Control](#)

This figure illustrates how the grid sorts data based on attributes such as **CustomerID**, **Company Name**, **Contact Name**, and **Contact Title**. The header arrows indicate the sorting direction.

#### Key Points:
- The grid provides the ability to sort data against multiple columns, ensuring that records remain sorted even after modifications.

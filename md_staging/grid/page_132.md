### Overview
- This section covers information on various topics related to grid controls.

#### Grid Control
- The Grid control is a general-purpose GridControl class that can hold its own data or virtually bind to an external data source.

### 4.1 Grid Control

**Description**  
The Grid control is a general-purpose GridControl class. You can use it by allowing it to hold its own data or virtually binding it to an external data source.

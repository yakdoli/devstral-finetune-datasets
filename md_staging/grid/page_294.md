### 4.1.4.6.6.99 GAMMALN

Returns the natural logarithm of the gamma function, ã(x).

#### Syntax

GAMMALN(x), where:

- **x** is the value for which you want to calculate GAMMALN.

#### Remarks

- x must be positive.
- GAMMALN is calculated as follows,  
  \[
  GAMMALN = LN(\Gamma(x))
  \]

where:  
\[
\Gamma(x) = \int_{0}^{\infty} e^{-u} u^{x-1} du
\]

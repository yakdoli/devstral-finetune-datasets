## Creating and Using Data Sets in Grids

### Generating a Dataset

#### Overview
- Steps to generate a dataset in Windows Forms, including selecting existing or creating a new dataset.
- Adding tables to the dataset using the "Generate Dataset" dialog.

#### Workflow

1. **Generate Dataset Dialog**:
   - Open the "Generate Dataset" dialog.
   - Choose an existing dataset: `WindowsFormsApplication1.NORTHWNDDataSet`.
   - Select the "Customers (sqlDataAdapter1)" table to include in the dataset.

   ![Figure 399: Generating Data Set](images/figure_399.png)

2. **Automatic Fill Method Execution**:
   - In the `Form_Load` event handler, the `Fill` method will be automatically called for the `SqlDataAdapter`.
   - This method passes the dataset generated in the previous step.

   ```csharp
   private void Form1_Load(object sender, EventArgs e)
   {
       // TODO: This line of code loads data into the 'nORTHWNDDataSet.Customers'
       this.customersTableAdapter.Fill(this.nORTHWNDDataSet.Customers);
   }
   ```

   ![Figure 400: Fill method called automatically in the Form_Load Event Handler](images/figure_400.png)

3. **Adding GridListControl**:
   - Drag a `GridListControl` object from the toolbox and place it onto the form.

- This page details the process of generating a dataset, setting up the `SqlDataAdapter`, and integrating the `GridListControl` in a Windows Forms application.
- It covers the UI-based steps to configure the dataset and provides code examples for automating data loading during runtime.

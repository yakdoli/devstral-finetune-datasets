# Essential Grid for Windows Forms

## Overview

- Describes the GridControl Designer for Windows Forms.
- Details the property window settings and customization options for the Grid control.
- Explains steps to save/load XML formats and apply changes.

## Content

### GridControl Designer

**Description:** GridControl Designer allows you to save/load XML formatted files and Soap templates.

**Figure 73: Property Window**

![Property Window](./property-window.png)

The Property Window shows various settings that can be customized for the Grid control, including appearances, backcolor, borders, and font styles. The current selection is highlighted, and the properties are displayed on the right.

**Steps to Apply Changes:**

1. Make the necessary changes in the GridControl Designer.
2. When changes are complete, exit the designer. You will be prompted to save the changes to the Grid control.
3. Click **OK** to apply these settings to the grid control.

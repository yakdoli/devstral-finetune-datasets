### 4.3 Grid Grouping Control

Grid Grouping control supports grouping with summaries, filters, and expression columns; it can be used with flat data sources, hierarchical data sources, or data sources with multiple nested tables.

#### 4.3.1 Creating Grid Grouping Control

This section will provide the step-by-step procedure to create a Grid Grouping control through designer and through a programmatic approach in a .NET application.

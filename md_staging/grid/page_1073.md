# Essential Grid for Windows Forms

## Overview
- Explains how to set a data source to a `gridListControl` using the `DataSource` property.
- Provides a step-by-step guide to configuring the `gridListControl` to display data from a `BindingSource`.

## Content

### Setting Up the Data Source

Figure 402 illustrates the process of setting the `DataSource` property for the `gridListControl`:

- **Grid Properties Pane**: The properties of the `gridListControl` are displayed, indicating it is of the type `Syncfusion.Windows.Forms.Grid`.
- **DataSource Property**: The dropdown menu for the `DataSource` property is expanded, showing options such as:
  - None
  - `customersBindingSource`
  - Other Data Sources
- **Currently Data Bound**: The grid indicates that the current data source is `customersBindingSource`.

### Steps to Implement

1. Open the properties window of the `gridListControl`.
2. Locate the `DataSource` property.
3. Select the desired `BindingSource` from the dropdown menu.
4. Ensure that the data source matches the required format and structure.
5. Configure any additional properties as needed.

### 4.3.2 Feature Summary

Essential Grid Grouping control possesses advanced features such as Outlook style grouping and is highly optimized to handle large amounts of data. This section lists the various features of the grid grouping control in brief.

- **Filter by Display Member**: A new addition to Grid Helper Utilities. It allows filtering of grid data by Display Member instead of Value Member. With this feature, the filter bar's drop-down lists display the member strings of the filtering column instead of its value member strings. You can use this feature with both Grid Data Bound Grid and Grid Grouping control.

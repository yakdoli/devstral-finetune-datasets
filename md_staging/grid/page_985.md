### 4.3.4.11 Navigation Bar

The Grid Grouping control comes with an in-built Navigation Control that facilitates easy browsing through records. The navigation bar includes buttons for moving to first, next, previous, and last records, as well as an option to add a new record. A label on the navigation bar shows the current record number and the total record count.

#### NavigationBar Setup
- **Enabling NavigationBar:** Configure by setting `ShowNavigationBar` to `true`.
- **Customizing Appearance:** Adjust default appearance via appropriate properties.
- **Enabling Tooltips:** Enable tooltips by setting `ShowNavigationBarTooltips` to `true`. Ensure `ShowNavigationBar` is set to `true` for tooltips to function.

#### Grid Grouping Control Properties
| Grid Grouping Control Property      | Description                                                         |
|-------------------------------------|---------------------------------------------------------------------|
| ShowNavigationBar                  | Enables or disables the navigation bar.                          |
| ShowNavigationBarTooltips          | Enables tooltips for the navigation bar when set to `true`.       |

# Essential Grid for Windows Forms

## Overview
- Demonstrates the use of the Grid Folder Browser to navigate and display file system directories.
- Highlights the Multiple Document Interface (MDI) support provided by "Essential Grid."

## Content

### WinForms Grid Folder Browser Demo

The Grid Folder Browser Demo provides a visual representation of navigating and displaying file system directories.

#### Figure 201: Grid Folder Browser Demo
This figure shows a window titled "Grid Folder Browser Demo" displaying a file system structure in a grid format. Key columns include:

- **Logical Drives:** Path is shown as `C:\`.
- **Name:** Displays file and folder names.
- **Size:** Indicates the size in bytes, MB, or KB.
- **Path:** Full path to each item.
- **Modified Time:** Shows the last modification time.

**Example Data:**
- **C:\\** → Size: 3 B, Modified Time: 9/9/2009 5:32:23 AM.
- **027cdc818b806e1fdfbe04** → Size: 6 MB, Modified Time: 8/23/2009 11:11:09.
- **Downloads** → Size: 9 B, Modified Time: 8/19/2009 4:43:38.
- **Documents and Settings** → Size: 25 MB, Modified Time: 7/18/2009 12:49:32.
- **Windows OS folders such as Inetpub**, **j2sdk1.4.1_02**, **MSOCache**, **Program Files**, and others, indicating typical Windows folder structure.

### 4.1.4.28 Multiple Document Interface (MDI) Support

#### Overview
Essential Grid supports Multiple Document Interface (MDI) by enabling users to work with multiple grid controls simultaneously. This feature allows multiple windows to reside under a single parent window, facilitating a more organized and intuitive user interface.

#### Description
- **MDI Support:** Through Essential Grid, applications can manage multiple documents or forms within a single parent window.
- **User Interaction:** Users can interact with multiple grid controls at the same time, enhancing productivity.
- **Integration:** This feature is fully integrated within the framework, ensuring seamless operation and customization.

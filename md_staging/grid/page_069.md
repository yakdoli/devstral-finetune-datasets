## Content
### Opening the Add Connection Dialog
The **Add Connection** dialog box is used to configure database connections. Follow these steps to set up the connection:

1. **Open the Dialog Box**: The dialog will appear without the database file name pre-filled.
2. **Browse for the Database File**:
   - Click the **Browse** button.
   - Navigate to the path: **C:\Syncfusion\EssentialStudio\[Version Number]\Windows\Data\NWIND.mdb**.
   - Click **OK** to confirm the selection.
3. **New Connection Setup**:
   - Click **New Connection** to proceed with creating a new connection.
   - Click **Next** to move forward in the configuration process.

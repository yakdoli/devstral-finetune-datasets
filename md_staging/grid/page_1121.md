## Overview
- Demonstrates how to filter a column in the Grid Grouping control by its display member.
- Explains the functionality of the Grid Format Cell Dialog for dynamic cell formatting options.

## Content

### Filtering a Column by Display Member

In this section, we illustrate how to filter a column in the Grid Grouping control by its display member. The following code snippet shows how to implement this functionality in VB.NET.

```vb
Private gGFilter As GroupingGridFilterBarExt
Me.gGFilter = New GroupingGridFilterBarExt()
Me.gGFilter.WireGrid(Me.gridGroupingControl1)
```

### 4.7.8 Grid Format Cell Dialog

The Grid Format Cell Dialog, similar to the Excel-like Format Cell Dialog, enables users to format the cells dynamically. It provides options to customize the cell font family, font color, font size, font style, font effects, background, alignment, text format, and so on.

# Essential Grid for Windows Forms

## Overview
- Covers properties related to text alignment, formatting, styling, and visual effects for cells and progress bars in the grid control.
- Provides options for text shadow, trimming, and separators for numbers and time.
- Includes parameters for customizing the appearance of progress bars within cells.

## Content

### Properties and Descriptions
The following table lists the properties with their descriptions, types, and data types:

| Property           | Description                                                                                                                                                   | Type    | Data Type                  |
|--------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|---------|----------------------------|
| -                  | text cannot be parsed and converted to the type specified with CellValueType                                                                          | -       | -                          |
| Tag                | A custom tag you can associate with a cell                                                                                                               | -       | object                     |
| Text               | Gets / sets the value as a string                                                                                                                          | -       | string                     |
| TextAlign          | Align text left of button elements (which is typical for combo boxes). Or align text right of button elements                                           | -       | GridTextAlign             |
| TextShadow         | Determines if the text shadow is visible                                                                                                                 | -       | bool                       |
| TextStyle          | Determines the style of the text                                                                                                                           | -       | ProgressBarTextStyles     |
| ThousandSeparator   | The character to use when a thousand separator position is specified                                                                                  | -       | char                       |
| TimeSeparator      | The character to use when a time separator position is specified                                                                                       | -       | char                       |
| Trimming           | Indicates how text is trimmed when it exceeds the edges of the cell text rectangle                                                                     | -       | StringTrimming             |
| TubeEndColor       | The outer color of the tube of the foreground (Progress bar)                                                                                            | -       | Color                      |
| TubeStartColor     | The middle color of the tube of the foreground                                                                                                           | -       | Color                      |

## Code Examples

Example of setting properties programmatically:

```csharp
gridModel.CellStyle.TextAlignment = GridTextAlign.Center;
gridModel.CellStyle.TextShadow = true;
gridModel.CellStyleTubeEndColor = Color.Red;
gridModel.CellStyleTubeStartColor = Color.Green;
```

## API Reference

### Namespace: Syncfusion.Windows.Forms.Grid

#### Properties
- `Tag`: Associated custom data with a cell.
- `Text`: Gets or sets the cell text as a string.
- `TextAlign`: Specifies the alignment of the text within the cell.
- `TextShadow`: Enables or disables the shadow effect for the text.
- `TextStyle`: Specifies the style applied to the text.
- `ThousandSeparator`: Specifies the character used for separating thousands in numbers.
- `TimeSeparator`: Specifies the character used for separating hours, minutes, and seconds.
- `Trimming`: Specifies how to handle text that exceeds the cell boundaries.
- `TubeEndColor`: Specifies the outer color of the progress bar.
- `TubeStartColor`: Specifies the middle color of the progress bar.

### 1.3.4.1.2 Adding Sort Icon Placement to an Application

To enable this feature, use the following code:

#### [C#]

```csharp
this.gridGroupingControl.SortIconPlacement = SortIconPlacement.Top;
```

#### [VB]

```vb
Me.gridGroupingControl.SortIconPlacement = SortIconPlacement.Top
```

# Essential Grid for Windows Forms

## Overview

- In this section, discover how to set up an external data source and display it using a virtual grid. Learn to display and edit data, pushing changes back to your data source.
- Explore topics such as creating a project, setting up a data source, and implementing a virtual grid for Windows Forms applications.

## Content

### Creating the Project and Data Source

#### Creating a New Project

1. **Step 1: Open Visual Studio .NET**
   - Open the File menu and select **New Project**.
   - Choose the Windows Application project template. Use either VB.NET or C# to create a new Windows Forms project.

   ![Creating New Project](https://i.imgur.com/VirtualGridTutorial.png)
   *Figure 63: Creating New Project*

   - Name the project **VirtualGridTutorial**.
   - Specify the location where the project will be created, for example, **C:\A_SyncWork\incident\VirtualGridTutorial**.

2. **Step 2: Create the Data Source**
   - **Objective**: Set up any external data source that can return values based on row-column parameters.
   - **Requirements**: The external data source must know how to return values based on the row-column parameters passed to it.
   - Additional details on the data source requirements are upcoming in subsequent sections.

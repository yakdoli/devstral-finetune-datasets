# Essential XlsIO

## Overview
- Demonstrates how to set up a new Silverlight application with Syncfusion XlsIO integration.
- Details on hosting the application in a new website and referencing required assemblies.
- Instructions for integrating XlsIO into an ASP.NET MVC application.

## Content

### New Silverlight Application Setup
1. **Creating a New Silverlight Application**:  
   Click the checkbox to host the Silverlight application in a new website. The `New Silverlight Application` dialog box appears (see Figure 18).  
   **Figure 18: New Silverlight Application dialog box**  

2. **Host the Application in a New Website**:  
   Click OK to host the Silverlight application in a new website. A new Silverlight application is created.

3. **Open the Main Form in the Designer**:  
   Open the main form of the application in the designer.

4. **Add References to Required Assemblies**:  
   Add references to the following assemblies:  
   - `Syncfusion.XlsIO.Silverlight.dll`  
   - `Syncfusion.Compression.Silverlight.dll`  

   For more information on deploying XlsIO, refer to the **Deploying Essential XlsIO** topic to know how to add XlsIO to the created application.

### ASP.NET MVC Application Integration

#### Creating an MVC Application  
Refer to the **Grid MVC -> Getting Started -> Creating an MVC application** topic to know how to create an MVC application.

#### Integrating XlsIO into the MVC Application  
To know how to add XlsIO to this application, refer to the **Adding Essential XlsIO to an MVC Application** topic under the Getting Started section.

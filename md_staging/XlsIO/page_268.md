# Essential XlsIO

## Repeating Row Label on Each Page

XlsIO allows setting the row label on each page, while printing, allowing users to view the header on each page.

### Code Example

```csharp
pivotTable.RepeatItemsOnEachPrintedPage = false;
```

### Formatting the Pivot Table By using Excel 2007

Excel 2007 provides a set of built-in styles that allow formatting the pivot table row and column header. When your cell pointer is inside the pivot table, you will have two new ribbon tabs under PivotTable Tools - Options and Design. On the Design ribbon, the Pivot Table Styles gallery offers 85 built-in formats for pivot tables.

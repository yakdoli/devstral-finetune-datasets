# Essential XlsIO

## Overview

- A table describing various data export flags used in the XlsIO component, including their names and descriptions.
- Links to additional information about the export options for XlsIO.

## Content

### Data Export Flags

The following table describes the various datatable export flags available in the XlsIO component.

| Member Name                       | Description                                                                                                                                                                                                 |
|------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| None                              | No datatable exports flags.                                                                                                                                                                                |
| ColumnNames                       | Represents the ColumnNames datatable export flag.                                                                                                                                                         |
| ComputedFormulaValues             | Represents the ComputedFormulaValues datatable export flag.                                                                                                                                               |
| DetectColumnTypes                 | Indicates that XlsIO should try to detect column types.                                                                                                                                                   |
| DefaultStyleColumnTypes           | When DetectColumnTypes is set and this flag is set too, it means that <default column style> must be used to detect style. If this flag is not set, but DetectColumnTypes is set, then the first cell in the column will be used to detect column type. |

### Additional Information

For more information on this feature, see: [http://www.syncfusion.com/products/reporting-edition/xlsio/features#export-options](http://www.syncfusion.com/products/reporting-edition/xlsio/features#export-options)

## Essential XlsIO

### WinForms Application

A windows application is created.

1. Open the main form of the application in the designer.
2. Add the Syncfusion controls to your VS.NET toolbox if you haven't done so already [This is done automatically when you install Essential Studio].
3. Refer to the *Deploying Essential XlsIO* topic to know how to add XlsIO to the created application.

### Web Application

1. Open Microsoft Visual Studio. Go to the File menu and click New Project. In the New Project dialog box, select the ASP.NET Web Application template, name the web application, and click OK.

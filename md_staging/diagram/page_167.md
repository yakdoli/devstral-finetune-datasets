# Essential Diagram for Windows Forms

## Overview
- Describes the FillStyle Dialog Box and its functionality in setting fill styles for Connection Point Lines.
- Explains the LineStyle property and its customization options for Connection Point Lines.

## Content

### FillStyleDialog

**Figure 100: FillStyle Dialog Box**

The FillStyleDialog window is displayed, providing options for customizing fill styles. The interface includes sections for "Colors" and "Brush style." 

- **Colors**:
  - **Color**: Allows selection of a color using the "Choose color" button, with an alpha factor set to 0.
  - **Fore color**: Allows selection of a foreground color using the "Choose color" button, with an alpha factor set to 255.

- **Brush style**:
  - **Brush type**: Set to "Solid," with options including "Texture," "LinearGradient," "Hatch," and "PathGradient."

- **Preview**: Displays a preview of the current fill style, showing a rectangular shape with a dashed outline.

- **Buttons**: Contains "OK" and "Cancel" buttons to confirm or discard changes.

### LineStyle

- This property allows customization to set the style for the Connection Point Lines, similar to other line types. 

#### Customization Properties:
- **Width**: Set to 1.
- **Color**: Blue.
- **DashCap**: Round.
- **DashOffset**: 0.
- **DashStyle**: Dash.
- **EndCap**: Round.
- **InheritContainer**: True.
- **LineColor**: Blue (represented by a color box).
- **LineJoin**: Round.
- **LineWidth**: 1.
- **MeasureUnit**: Pixel.
- **MiterLimit**: 10.

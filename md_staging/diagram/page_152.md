## 4.5.1.8 OrgChart Layout Manager

### Overview
- Event arranges all the nodes in a parent-child relationship using the new OrgLineConnector.
- The OrgLineConnector connects nodes to achieve the OrgLayout appearance.
- Designed specifically for connecting nodes in OrgChartLayoutManager.

### Content

**Event arranges all the nodes in parent-child relationship with the new OrgLineConnector that connects the nodes to get the OrgLayout appearance.**  
**The OrgLineConnector is specially designed for connecting the nodes in OrgChartLayoutManager.**

| Property             | Description                                                                 |
|----------------------|-----------------------------------------------------------------------------|
| Model                | Represents the model of the diagram, which is displayed as an OrgLayout.    |
| RotationDirection    | Gets / sets the layout directions. There are four major directions, which are as follows: <br> - BottomToTop <br> - LeftToRight <br> - RightToLeft <br> - TopToBottom |
| HorizontalSpacing    | Holds the value for the horizontal offset between adjacent nodes (float value). |
| VerticalSpacing      | Holds the value for the vertical offset between adjacent nodes (float value).  |

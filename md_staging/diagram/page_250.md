# Essential Diagram for Windows Forms

## Overview

- Displays diagram nodes with different styles, including `GhostCopy`, `FilledRectangle`, `SolidOutline`, and `DashedOutline`.
- Introduces the concept of dynamic properties for nodes and connectors.

## Content

### Illustrated Diagram Nodes

#### Figure 152: Diagram nodes with different styles

The figure illustrates various styles for diagram nodes:

1. **GhostCopy**: Shows ghosted nodes with a highlighted bounding box.
2. **FilledRectangle**: Demonstrates a filled rectangular shape with a highlighted bounding box.
3. **SolidOutline**: Displays a rectangular shape with a solid outline and a highlighted bounding box.
4. **DashedOutline**: Illustrates a rectangular shape with a dashed outline and a highlighted bounding box.

Each node has a green bounding box with a small circular grip at the top center, indicating manipulability and style options.

### 4.6.13 Dynamic Properties

#### Overview

This feature allows users to add additional properties or data to the nodes and connectors. Any type of data can be added as additional data or properties.

#### Implementation Details

The node's `PropertyBag` property, which is a key-value pair, is used to add, edit, and remove the additional properties or data. This information can be serialized when saving the diagram.

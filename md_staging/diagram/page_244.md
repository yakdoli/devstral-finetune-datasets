# Essential Diagram for Windows Forms

## Overview
- Demonstrates how to use palette groups in a Windows Forms application.
- Explains setting up a Diagram object using C# and VB.NET.

## Content

### Setting Up Palette Groups

#### Code Example in C#

```csharp
paletteGroupView1.Diagram = diagram1;
// Palette group bar
paletteGroupBar1.Diagram = diagram1;
```

#### Code Example in VB.NET

```vb
' Palette group view
paletteGroupView1.Diagram = diagram1;
' Palette group bar
paletteGroupBar1.Diagram = diagram1;
```

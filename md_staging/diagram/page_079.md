# Essential Diagram for Windows Forms

## Overview
- Diagrams can be created using the Diagram Builder and used as .edd files for application development.
- The Symbol Designer application allows the creation and modification of symbol palettes (*edp files).

## Content

### 3.4 Symbol Designer

The Symbol Designer application enables the creation of new palettes with symbols and the modification of existing palettes. These palettes are saved with the *.edp extension and can be utilized in both applications and the Diagram Builder.

#### Software Path
`..\Syncfusion\Essential Studio\<Version Number>\utilities\Diagram\Symbol Designer`

#### Creating EDP File

To create custom symbols in the symbol designer, follow these steps:

1. **Open the Symbol Designer Tool** which is available in the above software path.
2. **Symbol Designer Interface** contains the following three sections:
   - **Symbol Palette**  
   - **Object Model**  
   - **Properties Window**  

3. If you want to create a new palette:
   - Select **New** option in the **File** menu.
   - Type a name for the palette (as shown in the below sample).
   - Click **OK**.

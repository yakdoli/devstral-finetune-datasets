# Essential Diagram for Windows Forms

## PaletteGroupBar

### Creating a PaletteGroupBar Control through Designer

This section depicts the step-by-step procedure to create a PaletteGroupBar control through the Visual Studio designer in a .NET Windows Forms application.

#### Steps

To create a PaletteGroupBar control through the designer:

1.  Create a new Windows Forms application.
2.  Open the Designer Form window.
3.  Drag `PaletteGroupBar` from the Toolbox and drop it to the Designer Form window.

**Figure 15: Dragging PaletteGroupBar to the Designer**

#### Explanation

The PaletteGroupBar control will be added to the designer, and its dependent assemblies will be added to the project once you drop it to the Designer Form window.

### C#

```csharp
[STAThread]
static void Main(string[] args)
{
    // Create an arraylist of random MyObjects.
    ArrayList list = new ArrayList();
    Random r = new Random();

    for(int i = 0; i < 10; i++)
    {
        list.Add(new MyObject(r.Next(5)));
        Console.WriteLine(list[i]);
    }

    // Pause
    Console.ReadLine();
}
```

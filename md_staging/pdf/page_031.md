## Creating an ASP.NET Application

To know how to deploy a web application, refer the ASP.NET Behind the scenes section in the Getting Started guide in the documentation.

### Overview
- Describes the steps to create and deploy a web application using ASP.NET in Visual Studio.
- Involves selecting the ASP.NET Web Site template and deploying Essential PDF into the created application.

### Content

#### Creating an ASP.NET Application

1. **Open Microsoft Visual Studio.**  
   Go to the **File** menu and click **New Website**.  
   In the **New Website** dialog, select the **ASP.NET Web Site** template, name the website, and click **OK**.

   ![New Website dialog for ASP.NET Website](Figure 16: New Web Site dialog-ASP.NET Website)

   A web application is created.

2. **Now you need to deploy Essential PDF into this ASP.NET application.**  
   Refer to the **ASP.NET** topic for detailed information.

#### Creating a WPF Application

1. **Open Microsoft Visual Studio.**  
   Go to the **File** menu and click **New Project**.  
   In the **New Project** dialog, select the **WPF Application** template, name the project, and click **OK**.

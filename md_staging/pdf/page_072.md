# Essential Studio Dashboard

The User Interface edition panel is displayed by default.

## Overview
- The User Interface edition panel displays samples for WPF and Silverlight platforms.
- Users can access documentation, add-ons, toolbox configuration, and common utilities.
- The dashboard includes options for Reporting and Business Intelligence.

## Content

### Accessing Reporting Samples
1. **Select Reporting** and click the drop-down button of any platform listed under **Samples** (e.g., Windows Forms).

#### Options Available:
- **Run Locally Installed Samples**: View locally installed XlsIO samples using the sample browser.
- **Run Online Samples**: View online XlsIO samples.
- **Explore Samples**: Locate the XlsIO samples on the disk.

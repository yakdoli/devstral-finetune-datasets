## Content

### Class Reference

#### Online
Here's how to access the online class reference:

1. Visit the following URL:
   [
   `http://help.syncfusion.com/resources`
   ](http://help.syncfusion.com/resources)

2. Navigate to the "Reporting User Guide."
3. Select the "PDF" option.
4. Click the "Class Reference" link located in the upper right section of the page.

#### Installed Documentation
The installed documentation can be accessed via the following path:

- Dashboard -> Documentation -> Installed Documentation.

### Access Paths
- **Online**: Follow the steps mentioned above to access the online class reference.

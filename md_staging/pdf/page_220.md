## Drawing Shapes

### Overview
- Essential PDF provides a comprehensive set of APIs for drawing various shapes, such as rectangles, circles, arcs, and curves.
- Shapes can be filled using different types of brushes.
- Supports transparency settings for shapes.

### Content

#### 4.1.7.2 Drawing Shapes

**Essential PDF has a comprehensive set of APIs that can be used for drawing a variety of shapes such as:**

- Rectangles
- Circles
- Arcs
- Curves and so on

**Shapes are filled by using different types of brushes like gradient brush, Tiling brush, and so on.**  
Essential PDF supports drawing of shapes with different color spaces. Transparency of shapes can also be set.

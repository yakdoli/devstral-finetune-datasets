# Essential PDF

## Overview
- Describes the step-by-step procedure to create a Windows application using Microsoft Visual Studio.
- Focuses on creating a Windows Forms Application with a .NET Framework 3.5 interface.

## Content

### Creating Platform Application

This section illustrates the step-by-step procedure to create the following platform applications.

#### Creating a Windows Application

1. **Open Microsoft Visual Studio.**  
   Go to **File** menu and click **New Project.**  
   In the **New Project** dialog, select **Windows Forms Application** template, name the project, and click **OK.**

   **Figure 15: New Project dialog-Windows Forms Application**

   ![New Project dialog - Windows Forms Application](<Figure 15>)

   - A Windows application is created.

2. **Now you need to deploy Essential PDF into this Windows application.**  
   Refer **Windows** topic for detailed info.

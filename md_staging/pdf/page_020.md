# Essential PDF

## Overview
- A high-performance server-side .NET library that can read, write, and manipulate Adobe PDF files.
- Full-featured object model providing complete control over positioning of elements in a PDF document.
- Does not require any Adobe product and is built entirely in C#.
- Fully integrated into ASP.NET MVC.

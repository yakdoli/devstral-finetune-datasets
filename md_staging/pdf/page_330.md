# Essential PDF

## Overview
- Supports features like line spacing and left, right, and center justification.
- Handles multicolumn text and headers/footers, including images, text, and page number fields.
- Supports bulleted, numbered, and multilevel lists with proper indentation and alignments.
- Handles images with corresponding positions and sizes.

## Content

### Features
- **Line Spacing**: Adjusts spacing between lines of text.
- **Left, Right, and Center Justification**: Allows text alignment.
- **MultiColumn Text**: Word documents containing multicolumn text are supported.
  - **Known Limitations**: Full-justification formatting on columns may lead to differences in output.
- **Headers and Footers**: Supported with options for images, text, and page number fields.
- **Bulleted, Numbered, and MultiLevel Lists**: Proper indentation and alignments are preserved.
  - **Known Limitations**: Image bullets set on the document may be replaced by default symbols.
  - **Image replacement example**:
    ![Image replacement](https://user-images.githubusercontent.com/133403520/265855253-0f010e00-3c72-4e0a-8bb1-f425c1c7c660.png)
    **Figure 65: Image replacement**
  
### Images
- Supported with corresponding positions and sizes.
- **Known Limitations**: Images placed inside a shape are not preserved in the generated PDF document.

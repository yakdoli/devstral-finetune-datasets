# Essential PDF

## Overview
- Explanation of how to use the Windows Forms Sample Browser to display and browse PDF samples.
- Introduction to the WPF platform and instructions on viewing and running samples.

## Content

### Windows Forms

#### Overview
Essential Studio 2012 Volume 3 Windows Forms provides a comprehensive library for producing Adobe PDF files using the .NET framework.

#### Features
Essential PDF is a .NET library capable of producing Adobe PDF files. It features a full-fledged object model for easily creating PDF files from any .NET language. It does not use any external libraries and is built from scratch in C#. It can be used on the server side (ASP.NET or any other environment) or with Windows Forms applications. Essential PDF supports many features for a PDF document, including drawing text, images, shapes, etc. All can easily be done with Essential PDF.

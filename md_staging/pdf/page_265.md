### Overview

- Attachments can be added to a PDF document, as shown in the example where `Manual.txt` is attached to a PDF.
- Dynamic Fields or Automatic Fields are special objects that display dynamically calculated information before the document is saved.

#### Dynamic Fields

Automatic Fields or Dynamic Fields are special objects that display information calculated automatically, just before the document is saved.

The fields display the following:

- Page number
- Count of pages
- Author of the document
- Creation and current date

## Essential Edit for Windows Forms

### Action Commands

| Action       | Shortcut Key       |
|--------------|--------------------|
| Save         | CTRL+S             |
| SaveAs       | CTRL+SHIFT+S       |
| New          | CTRL+N             |
| Open         | CTRL+O             |
| Print        | CTRL+P             |

### Positioning

| Action       | Shortcut Key       |
|--------------|--------------------|
| Go to line   | CTRL+G             |
| Go to start  | CTRL+HOME          |
| Go to end    | CTRL+END           |

### Search and Replace

| Action       | Shortcut Key       |
|--------------|--------------------|
| Find         | CTRL+F             |
| FindNext     | F3                 |
| FindSelected | CTRL+F3            |
| Replace      | CTRL+H             |

### Undo and Redo

| Action       | Shortcut Key       |
|--------------|--------------------|
| Undo         | CTRL+Z             |
| Redo         | CTRL+Y             |

### Bookmark

| Action               | Shortcut Key                     |
|----------------------|-----------------------------------|
| Toggle unnamed bookmark | CTRL+F2, CTRL+K->CTRL+K       |
| Go to next bookmark  | F2, CTRL+K->CTRL+N              |
| Go to previous bookmark | F3, CTRL+K->CTRL+P           |
| Toggle named bookmark | CTRL+[index of bookmark]       |
| Go to named bookmark  | CTRL+SHIFT+[index of bookmark] |

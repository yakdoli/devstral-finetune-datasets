# Essential Chart for Windows Forms

## Overview
- Demonstrates the visual appearance of a chart when the Skins value is set to "Marble" and "Midnight."
- Focuses on how different skin settings affect the chart's visual style.

## Content

### Chart with Skins under "Marble" Skin

The following output is displayed when the Skins value is set to Marble.

![Chart with Skins under "Marble" Skin](#)
*Figure 338: Marble*

- The chart titled "Chart with Skins" is rendered with a light theme.
- Data points are represented as black dots on a white background with a grid overlay.
- The x-axis ranges from 0 to 10, and the y-axis ranges from 0 to 60.
- The series is labeled as "Series 1," and the filling color is black.

### Chart with Skins under "Midnight" Skin

The following output is displayed when the Skins value is set to Midnight.

![Chart with Skins under "Midnight" Skin](#)

- The chart titled "Chart with Skins" is rendered with a dark theme.
- Data points are represented as blue dots on a dark background with a grid overlay.
- The x-axis ranges from 0 to 10, and the y-axis ranges from 0 to 60.
- The series is consistently colored blue, which contrasts against the dark background.

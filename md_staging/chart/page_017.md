# Essential Chart for Windows Forms

To view the samples, follow the steps below:

1. Click Start-->All Programs-->Syncfusion-->Essential Studio <version number> -->Dashboard.

## Content

### User Interface Overview

The Syncfusion Essential Studio Dashboard provides a comprehensive suite of tools and features for developing Windows Forms applications. Below is a snapshot of the dashboard interface:

- **User Interface Section**: 
  - Includes options for various frameworks such as ASP.NET MVC, ASP.NET, WPF, Silverlight, Window Phone, and Mobile MVC.
- **Reporting, Business Intelligence, Add-ons, Utilities, and License Management**: 
  - These sections offer additional functionalities and tools for enhancing applications.

#### User Interface Edition for Windows Forms

- **Integrate User Interface Edition into your applications with ease**.
- **Develop professional desktop applications with the help of 9 comprehensive products** (containing over 100 components) that will ease development and improve productivity.

### Figure 2: Syncfusion Essential Studio Dashboard

![](Figure 2: Syncfusion Essential Studio Dashboard)

In the Dashboard window, click **Run Samples for Windows Forms** under **UI Edition**. The **UI Windows Forms Sample Browser** window is displayed.

#### Note: Sample Viewing Options

You can view the samples in any of the following three ways:

- **Run Samples**: Click to view the locally installed samples.
- **Online Samples**: Click to view online samples.
- **Explore Samples**: Explore BI Web samples on disk.

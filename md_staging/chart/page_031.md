# Essential Chart for Windows Forms

## Overview

- Introduces the `ChartPoint Collection Editor` for configuring and managing chart data points in Syncfusion Winforms.
- Allows the addition and removal of data points with specific \( X \) and \( Y \) values.
- Provides options to modify the properties of each data point.

## Content

### ChartPoint Collection Editor

**Figure 12: ChartPoint Collection Editor**

The interface in Figure 12 displays the `ChartPoint Collection Editor`, a tool for managing data points to be plotted in a chart within Syncfusion Winforms. This editor provides functionalities to:

1. **Members List**: On the left, the `Members` section shows a list of existing data points with their respective \( X \) and \( Y \) coordinates.
2. **Properties Panel**: On the right, the `Properties` section allows editing individual properties of the selected data point, such as \( X \), \( Y \) values, and other miscellaneous settings.

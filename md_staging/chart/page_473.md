## Overview

- This page explains the behavior and customization of Chart Area and Chart Series context menus in Syncfusion Winforms.
- It covers disabling these context menus via specific properties.

## Content

### Chart Area Context Menu

#### Chart Area Context Menu

![Figure 303: Chart Area Context Menu](Illustrates%20chart%20Area%20Context%20Menu)

*Figure 303: Chart Area Context Menu*

This context menu can be disabled by setting the `DisplayChartContextMenu` property to `false`.

#### Chart Series Context Menu

2. **Chart Series context menu** - This will be displayed when the mouse is over a series.

![Figure 304: Chart Series Context Menu](Illustrates%20chart%20Series%20Context%20Menu)

*Figure 304: Chart Series Context Menu*

This context menu can be disabled by setting the `DisplaySeriesContextMenu` property to `false`.

### Disabling Context Menus

- **Chart Area Context Menu:** Use the `DisplayChartContextMenu` property to control visibility.
- **Chart Series Context Menu:** Use the `DisplaySeriesContextMenu` property to control visibility.

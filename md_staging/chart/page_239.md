### GapRatio

#### Description

Gets or sets the gap size between funnel chart or pyramid chart segments. The default value is `0.0`. The maximum gap size is limited by the number of points.

| **Details**          |                                                                 |
|----------------------|-----------------------------------------------------------------|
| Possible Values      | Ranges from `0.0`                                              |
| Default Value        | `0`                                                           |
| 2D / 3D Limitations | No                                                             |

### Changing Legend Properties at Run Time

The Legend's look and feel can also be customized during runtime. Double-clicking the legend's text will pop up the below properties window. Properties set through this dialogue can be applied to the chart.

**Note:** These settings will be lost when the application is closed.

![Legend Properties Dialog Box](https://i.imgur.com/unknown.png)
*Figure 282: Legend Properties Dialog Box*

### Legend Properties

- **Border**
  - Border: `Syncfusion.Windows.Forms.Chart`
  - ShowBorder: `False`

- **Misc**
  - BackColor: `Transparent`
  - ColumnsCount: `1`
  - FloatingAutoSize: `True`
  - ItemsAlignment: `Near`
  - ItemsShadowColor: `Gray`
  - ItemsShadowOffset: `2, 2`
  - ItemsSize: `20, 20`
  - ItemsTextAlignment: `Center`
  - LegendAlignment: `Near`
  - OnlyColumnsForFloating: `True`

# Essential Chart for Windows Forms

## Overview

- The Chart Wizard interface for configuring chart elements.
- Points labels can be customized for style attributes like color, font, angle, and position.
- The final customization option in the Chart Wizard is the ChartControl ToolBar.

## Content

### Chart Wizard Interface

#### Figure 18: Points button selected in Chart Wizard

The screenshot shows the Chart Wizard interface with the Points button selected. The interface includes options for configuring chart elements such as Chart type, Series, Appearance, Axes, Points, ToolBar, and Legend. The Points tab allows customization of points labels, including their color, font, rotation angle, and alignment. A default bar chart is displayed to the right, showing data labels on each bar.

#### 4.1.6 Toolbar

- **The final option in the Chart Wizard is the ChartControl ToolBar. It has two tabs.**
  - **ToolBar**
    Under this tab, the user can customize the toolbar's back color, button style, as well as set the width and height for the buttons through the respective options.

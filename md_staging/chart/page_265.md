## 4.5.1.43 LabelStyle

### Overview
- This section describes the LabelStyle property that governs the positioning and presence of DataPoint labels in Pyramid and Funnel charts.

### Content

#### LabelStyle Property
- **Gets or sets the Pyramid chart or Funnel chart label style.**
- This property dictates the presence and overall positioning of the DataPoint labels.

| **Details** |  |
|---|---|
| **Possible Values** |  |
|  | - **Inside** - DataPoint label is drawn inside the Pyramid segment. |
|  | - **Outside** - DataPoint label is drawn outside and next to the Pyramid segment. |
|  | - **OutsideInColumn** - DataPoint is drawn outside the Pyramid segment with all labels aligned. |

# Essential Chart for Windows Forms

## Overview
- Demonstrates the use of the Interactive Cursor feature in a line chart for product sales per month.
- Explains properties related to cursor orientation and colors for horizontal and vertical cursor lines.
- Provides details on how to interact with the cursor to view sales data points.

## Content

### Chart with Interactive Cursor

![Figure: Interactive Cursor with Horizontal Orientation](#)

*Figure 309: Interactive Cursor with Horizontal Orientation*

The figure above shows a line chart titled "Product Sales per Month," depicting monthly sales data over 18 months. The chart includes an interactive cursor feature, allowing users to view specific sales data points by clicking and dragging on the cursor lines.

#### Chart Details
- **Title:** Product Sales per Month
- **Y-Axis:** Sales values ranging from 200 to 500.
- **X-Axis:** Months from 2 to 18.
- **Data Points:** Individual sales figures are labeled on the chart.
- **Cursor Movement:** Options to move the cursor "Up" or "Down" are provided, facilitating navigation.

### Properties

The following table outlines the properties and their descriptions relevant to the Interactive Cursor feature:

| Property               | Description                                                                                     | Data Type |
|------------------------|-------------------------------------------------------------------------------------------------|-----------|
| CursorOrientation      | Indicates the orientation in which the Interactive Cursor is to be drawn. The options are: <br> • Horizontal <br> • Vertical <br> • Both | Enum        |
| HorizontalCursorColor  | Specifies the color, which is to be used when Horizontal Interactive Cursor is drawn.              | Color     |
| VerticalCursorColor    | Specifies the color, which is to be used when Vertical Interactive Cursor is drawn.               | Color     |

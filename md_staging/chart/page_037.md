# Essential Chart for Windows Forms

## ChartAxisLabel Collection Editor

![ChartAxisLabel Collection Editor](image.png)

**Description:** The ChartAxisLabel Collection Editor allows you to manage and customize the labels displayed on chart axes in Windows Forms.

### Options Available in the Editor

- **Members:**
  - Lists the available nodes in the chart axis labels collection.
  - Allows adding, removing, and reordering nodes using the buttons provided.

- **Properties:**
  - **Color:** Sets the text color to Black.
  - **CustomText:** Allows custom text formatting.
  - **DateTimeFormat:** Not applicable for non-date data.
  - **DoubleValue:** Sets the default value to 0.
  - **Font:** Sets the font to Verdana, 10pt.
  - **Format:** Specifies how the values are formatted.
  - **LogBase:** Sets the base for logarithmic scale to 2.
  - **RoundingPlaces:** Specifies decimal precision to 2 places.
  - **ValueType:** Sets the data type to Double.

## Points

### 4.1.5 Points

This particular section in the wizard provides options for displaying labels for the data points by simply checking the **Show labels** option.

### Options for Data Point Labels

- **Labels Font:** Sets the font style for the labels.
- **Labels Color:** Sets the color for the labels.
- **Labels Rotate Angle:** Sets the angle of rotation for the labels.
- **Label Alignment:** Sets the alignment of the labels on the series points.
- **Apply To:** Specifies the series name to which the above settings should be applied.

### Summary

The **ChartAxisLabel Collection Editor** and the **Points** section of the wizard allow for detailed customization of chart labels in Windows Forms, providing controls for font, color, and placement, ensuring that data visualization is presented clearly and effectively.

## Essential Chart for Windows Forms

### Overview
This section provides guidance on understanding and utilizing the localization features of the Essential Chart control for Windows Forms, showcasing a sample chart interface and instructions to interact with its localized elements.

### Content

#### WinForms Localization Chart
Figure 372 illustrates a localized chart in Windows Forms. The chart demonstrates scaling data in millions by country, with various tools available for modifying and interacting with the chart. Key features include:

- **Localization UI elements**: Options for activating X and Y zooming are visible, showcasing the ability to adjust the view of the chart based on internationalization settings.
- **Zooming tools**: Users can perform zooming actions to focus on specific data sets within the chart.

### Sample Link
To view a sample of the localization feature in Windows Forms:

1. Open the Syncfusion Dashboard.
2. Select User Interface > Windows Forms.
3. Click Run Samples.
4. Navigate to Culture Localization > Localization sample.

# Essential Chart for Windows Forms

## Overview
- This section guides through the configuration of a Data Source Type in Windows Forms applications using the Data Source Configuration Wizard.
- Focuses on selecting how the application will retrieve data from various sources like databases, services, or objects.
- Includes a step to click "New Connection" after selecting the desired Data Source Type.

## Content

### Data Source Configuration Wizard

The wizard provides an interface to configure the data source type for your application. The figure below illustrates the selection process:

#### Figure 30: Selecting the Data Source Type

- **Choose a Data Source Type**
  - The application allows you to select from different types of data sources:
    - **Database**: Connects to a database and enables selection of database objects for your application.
    - **Service**: Uses a service to retrieve data.
    - **Object**: Retrieves data from objects.

#### Instructions:

1. Select the appropriate data source type based on your application's requirements.
2. Proceed by clicking "Next >" to continue the configuration process.

- **Then click New Connection.**

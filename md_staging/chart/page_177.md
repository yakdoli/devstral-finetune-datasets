# Essential Chart for Windows Forms

## Overview
- Describes various chart types supported in Syncfusion WinForms.
- Lists properties relevant to different chart components like Series, Points, and individual chart types.
- Provides details about specific attributes such as *AngleOffset*, which pertains to the rendering of Pie charts.

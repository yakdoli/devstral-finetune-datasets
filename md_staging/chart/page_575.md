# Essential Chart for Windows Forms

## Overview

- This page describes the process of exporting a chart to a DocIO document using the Syncfusion Essential Chart library.
- Outlines the necessary steps to add required assemblies and namespaces for exporting charts.
- Includes example code snippets for C# and VB.NET to facilitate implementation.

## Content

### Exporting Chart to DocIO

The image below the text shows an example of a chart exported to a Microsoft Word document. The chart is a bar chart with a single series labeled "Series 1," and the title of the chart reads "Essential Chart-Chart Exporting."

![](https://i.imgur.com/MissingImage.png)
*Figure 354: Exporting Chart to DocIO*

Given below are the steps that will guide you through this process:

1. Add the `Syncfusion.DocIO.Base` and `Syncfusion.DocIO.Windows` assemblies.
2. Add the namespaces `Syncfusion.DocIO` and `Syncfusion.DocIO.DLS` in your form.

#### Code Examples

##### C#
```csharp
using Syncfusion.DocIO;
using Syncfusion.DocIO.DLS;
```

##### VB.NET
```vb
Imports Syncfusion.DocIO
Imports Syncfusion.DocIO.DLS
```

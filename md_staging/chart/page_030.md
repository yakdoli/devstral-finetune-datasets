# Essential Chart for Windows Forms

## Chart Wizard

The Chart Wizard interface allows for easy configuration of charts within Windows Forms. The figure below shows the Series tab selected in the Chart Wizard.

### Series Tab Interface

![Figure 11: Series button selected in Chart Wizard](https://charts.imagelink)

Below are the three tabs in the Wizard for Series:

1. **Add points to series**
   - Click the **Add** button to add a new series.
   - Select the series to which you need to add points.
2. **Series type and customization**
   - This tab provides options to select any chart type using the **Type** combobox and series name using the **Series Name** textbox.
3. **Edit Points**
   - Clicking the **Edit points** button opens the **ChartPoint Collection Editor**.

### Summary of Key Functions

- **Add points to series**: Allows the addition of new data points to the chart for analysis and visualization.
- **Chart type customization**: Users can choose the type of chart (e.g., Line, Bar, Column, etc.) and set series names for clarity.
- **Point editing**: Advanced editing of chart points is facilitated through the **ChartPoint Collection Editor**.

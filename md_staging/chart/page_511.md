#### 4.10.7 Watermark Support

**Overview**

- Essential Chart supports watermark feature that allows displaying text, images, or both as watermarks inside the chart area.

**Watermark Properties with Descriptions**

| ChartAxis Property | Description |
|--------------------|-------------|
| Text | Sets the watermark text. |
| Image | Used to display an image as the watermark. |
| Opacity | Sets the opacity of the watermark. |

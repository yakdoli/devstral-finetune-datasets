# Essential Chart for Windows Forms

## Chart Wizard

### Overview
The Syncfusion ChartWizard allows users to customize the appearance and behavior of charts, both at design time and runtime. This guide outlines how to access and use the Chart Wizard interface and commands.

### Design Time

To display the chart wizard at design time, follow these steps:

1. Add a ChartControl to your form.
2. Right-click anywhere in the chart to see a context menu.
3. Select the chart wizard item from the context menu.

## Figure 9: Chart Wizard (Windows)

**About the Interface:**
The Chart Wizard interface (Figure 9) provides options to select various chart types such as Line, Spline, etc., from the available options and customize the chart's appearance using features like Appearance, Axes, Points, ToolBar, and Legend.

### Chart Wizard Sidebar Options:
- **Chart Type**: Choose Line, Spline, or other chart types.
- **Series**: Manage data series for the chart.
- **Appearance**: Adjust visual appearance of the chart elements.
- **Axes**: Customize axes for better data representation.
- **Points**: Modify data points' attributes.
- **ToolBar**:Add or customize toolbar functionalities.
- **Legend**: Configure chart legend settings.

### Chart Type Display:
- **Line**: Displays a line chart with examples.
- **Spline**: Displays a spline chart with examples.

### Navigation:
- **Auto Run Wizard**: An option to run the wizard automatically.
- **Apply/Finish/Cancel**: Action buttons for applying, finalizing, or canceling changes.
- **3D Style**: An option to enable 3D style for the chart.

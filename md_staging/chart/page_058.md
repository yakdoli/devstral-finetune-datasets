# Essential Chart for Windows Forms

## Overview
- Configuring data source settings for a Windows Forms application
- Creating a new connection to a database
- Selecting the appropriate data source (MS SQL Server or MS Access)

## Content

### Data Source Configuration Wizard

To set up a database connection in a Windows Forms application using the Data Source Configuration Wizard, follow these steps:

1. **Choose Your Data Connection:**
   - Select the desired connection from the dropdown list or click **New Connection...** to create a new one.

2. **Sensitive Data Handling:**
   - Decide whether to include sensitive data in the connection string or exclude it.
     - **No:** Exclude sensitive data; manage it in the application code.
     - **Yes:** Include sensitive data directly in the connection string.

3. **Connection String:**
   - Enter or manage the connection string details as needed.

4. **Navigation:**
   - Use the **Previous**, **Next**, **Finish**, or **Cancel** buttons to guide the configuration process.

### Steps for Creating a New Connection

5. **Selecting the Data Source:**
   - In the **Choose Data Source** dialog box, select the data source as an **MS SQL Server database** or an **MS Access database**.
   - Click the **Continue** button to proceed with the configuration.

# Essential HTMLUI for Windows Forms

## Overview
- Provides an overview of the Essential HTMLUI for Windows Forms.
- Demonstrates how to run samples for Windows Forms under the UI Edition.
- Explains the methods to view samples using Run Samples, Online Samples, and Explore Samples.

## Content

### Figure 2: Syncfusion Essential Studio Dashboard
A screenshot of the Syncfusion Essential Studio Dashboard for Windows Forms.

The dashboard includes the following:
- **User Interface** section with ASP.NET MVC, ASP.NET, Windows Forms, WPF, Silverlight, Windows Phone, and Mobile MVC options.
- An image showcasing the User Interface Edition for Windows Forms.
- A section labeled **Run Samples** with an arrow icon to download and run samples.
- **User Interface Edition for Windows Forms** details:
  - Integrate User Interface Edition into applications for ease.
  - Develop professional desktop applications with 9 comprehensive products containing over 100 components to ease development and improve productivity.

Steps to access samples:
1. In the Dashboard window, click **Run Samples** for Windows Forms under UI Edition.
2. The UI Windows Form Sample Browser window is displayed.

#### Note: You can view the samples in any of the following three ways:
- **Run Samples**: Click to view the locally installed samples.
- **Online Samples**: Click to view online samples.
- **Explore Samples**: Explore BI Web samples on disk.

# Essential HTMLUI for Windows Forms

## Overview
- Guide on adding a new HTML file in the Solution Explorer.
- Walkthrough of accessing the Solution Explorer.
- Details on interacting with the Add menu and selecting the Add New Item option.
- Instructions for creating an HTML page within a Windows Forms project.

## Content

### Step-by-Step Process

1. **Open the Solution Explorer**: Locate the Solution Explorer panel in your Visual Studio IDE.
2. **Right-click on the C# file name**: Right-click on the file in the Solution Explorer to open a context menu.
3. **Click the Add tab**: From the opened context menu, select the **Add** tab to display the sub-menu.

#### Figure 19: Menu options in adding a new HTML File
![Menu options in adding a new HTML File](image.png)

- The sub-menu will display options such as **Add New Item**, **Add Existing Item**, and others.
- Select **Add New Item** to proceed to the next step.

4. **In the sub-menu, click AddNewItem**: Clicking this option will display a template wizard.
5. **In the wizard, select HTML Page**: In the wizard, choose **HTML Page**. The default name for the page is `HTMLPage1.htm`.
6. **Optional: Change the name**: Use the **Name** tab at the bottom of the wizard to modify the default name of the HTML page as needed.

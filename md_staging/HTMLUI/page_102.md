## Essential HTMLUI for Windows Forms

### HTML Code Sample

```html
<html>
<body>
<form>
    <input type="text"/><br/>
    <input type="button"/><br/>
    <input type="checkbox" checked /><br/>
</form>
</body>
</html>
```

Loading the above HTML document into HTMLUI creates a Form with the three controls specified as shown below.

# Essential HTMLUI for Windows Forms

## Overview

- Explanations of various keyboard shortcut keys supported by HTMLUI.
- Demonstrates printing and print preview features to help users create hard copies and preview printed content.

## Content

### 4.18.1 HTMLUIKeyboard Sample

This sample shows how elements in the document can be focused by using the Keyboard support in HTMLUI.

**Figure 48: HTMLUIKeyboard Sample**

The HTMLUI control supports the use of the following keyboard shortcut keys:

i. **Ctrl+A** - Select all the text available in the document, displayed in the control.

ii. **Ctrl+C** - Copy the text selected in the control to the ClipBoard.

iii. **Ctrl+F** - Open the search dialog box to find the occurrences of a specified text in the document, displayed in the control.

iv. **Tab** - Focus the next control in the document based on its tab index.

v. **Shift+Tab** - Focus the previously focused control in the document based on its tab index.

vi. **Enter** - Tries to open a link if it is currently focused.

vii. **SpaceBar** - Click the button or Check the CheckBox if they are focused.

viii. **ArrowKeys** - Shift the focus between the controls in the document displayed in the HTMLUI control.
